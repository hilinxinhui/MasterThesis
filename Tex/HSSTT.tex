\chapter{基于层次化随机时空 Transformer 的锂离子电池健康状态可信估计方法}\label{chap:HSSTT}

\section{引言}\label{sec:HSSTT_introduction}

锂离子电池（Lithium-ion Batteries, LIBs）凭借其优越的功能特性，在工业领域得到了广泛的应用，尤其是作为电动汽车动力电池\cite{NASAJPOURESFAHANI2024114783, hannan2017review}。然而，严苛的工业环境与复杂的内部电化学反应相叠加，亟待采取更为严密的保障措施，以确保LIBs长期的安全与可靠运行。因此，对LIBs的性能进行有效监测以保障其持续稳定运转显得尤为关键。通常而言，LIBs 的性能由 SOH 表征，该指标被定义为 LIBs 当前最大可用容量与其标称容量之比\cite{WANG2024110325, 10391265}。

深度学习的蓬勃发展极大地推动了数据驱动 SOH 估计模型的涌现。此类模型极大降低了模型开发过程对领域专家知识的依赖，并在捕捉复杂退化机制动态特性方面展现出卓越能力，因而正逐步取代传统的基于物理的模型\cite{mit_dataset_attia2020closed, tri_dataset_severson2019data, tongji_dataset_zhu2022data}。然而，工业环境中不可避免的噪声波动与多变的负载需求，往往导致样本数据呈现出复杂且难以解析的分布特征；在处理此类样本时，数据驱动模型依然极易产生不准确的估计结果\cite{10214203}。此外，受限于其确定性本质，此类模型仅能提供 SOH 的单一点估计，而缺乏对潜在误差的预警能力。
% 
图~\ref{fig:hsstt_challenges} 展示了三种不同类型的模型，分别是确定性模型、具有不确定性量化能力但缺乏校准机制的模型和具有不确定性量化和校准能力的可信模型，该图显示了 SOH 估计结果。图中黑线表示 SOH 真值，红线表示 SOH 估计值，阴影表示不确定性范围。
% 
过度依赖此类确定性模型势必会对后续的决策过程带来风险。例如，若 LIBs 的 SOH 估计值低于实际值，将导致 LIBs 提前退役，进而造成不必要的经济损失；反之，若高估 SOH，则会引发 LIBs 的过度循环使用，从而增加系统热失控风险。

为规避确定性模型引发的潜在问题，并切实保障工业级监测与维护的可靠性，对数据驱动的估计模型输出结果的可信度进行评估显得至关重要。鉴于其与高可信 SOH 估计之间的紧密联系，不确定性的量化被认为居于核心地位\cite{THUY2024330, 10539360}。一方面，不确定性可作为一种量化测度，通过衡量模型表征的概率分布与真实分布之间的偏差，客观地反映出估计结果的可信程度；另一方面，不确定性亦可作为一种以用户为导向的可靠性分析指标，能够自然融入涉及人工干预的实际决策流程之中。综上，实际工业应用对 LIBs 的高可信估计模型提出了迫切需求，而在构建此类模型时，仍面临着以下需要解决的问题：

（1）表征关联学习过程中的不确定性构成了第一项挑战。在关联信息聚合阶段，模型生成权重分数时所固有的不确定性被忽略。某些本不重要且可能具有高不确定性的循环信号片段易被错误地分配较高的权重值 \cite{20222012122312, 20191806851746, 10342686}。如图~\ref{fig:hsstt_challenges}(a) 所示，对这些冗余信号片段的过度聚合，将误导对复杂退化动态特征的提取，最终导致 SOH 估计精度的降低。

（2）对时空不确定性的忽视构成了第二项挑战。针对 LIBs 的循环充放电信号，主导通道的变动会导致跨传感器依赖关系发生波动，进而引发传感器数据融合过程中的不一致性。与此同时，跨时间依赖关系的稀疏分布特征，亦为提取信号片段对之间的交互特征引入了模糊性。如图~\ref{fig:hsstt_challenges}(a) 所示，若未能对此类时空不确定性进行充分且有效的建模，估计结果的精度亦将降低 \cite{wang2023inherent, 9729404, capsnet_rul_li2024sensor}。

（3）不确定性的潜在校准失真构成了第三项挑战。在实际应用场景中，原始数据的真实分布难以解析，导致通过不确定性量化所导出的预测分布可能与真实分布存在严重偏差 \cite{kuleshov2018accurate, 9729404, 10217044}。如图~\ref{fig:hsstt_challenges}(b) 所示，这种分布差异引发了区间估计的过度自信或置信度不足问题。其结果不仅削弱估计结果的可信度，同时损害了基于不确定性分数的分布外检测性能。

为应对上述挑战，本章提出一种层次化随机时空 Transformer（Hierarchical Stochastic Spatial-Temporal Transformer, HSSTT）。在 HSSTT 中，设计一种基于 Gumbel-Softmax 重参数化（Gumbel-Softmax Reparameterization, GSR）的自注意力机制。该机制将传统的确定性自注意力转化为随机自注意力，旨在处理特征提取与特征映射过程中的不确定性。随后，结合具有不确定性量化能力的膨胀时间步卷积与上述随机自注意力机制，以充分提取跨传感器的空间退化动态特性与跨循环的时间退化动态特性。最后，通过偏差-方差分解解释模型过度自信或置信度不足的成因，并据此构建模型无关的校准策略，通过校准后的无偏的不确定性，模型能够准确区分 ID 与 OOD 退化模式。HSSTT 的贡献如下：

（1）提出一种全新的随机自注意力机制，利用 GSR 赋予自注意力以不确定性量化能力，为高可信 SOH 估计提供支撑。

（2）以不确定性为导向进行时空依赖性建模与退化动态提取，消除循环数据中的模糊性，提高模型 SOH 估计性能。

（3）阐明模型校准失真成因，提出一种模型无关的不确定性校准方法以精准拟合实际数据分布，验证将校准得到的无偏不确定性应用在分布外退化模式检测中的潜力。

（4） 在 4 个公开 LIBs 循环老化数据集上开展详尽的综合实验。实验结果充分证实 HSSTT 在可信 SOH 估计任务中的有效性与先进性。

从理论层面看，本章讨论的不确定性涵盖退化特征提取、时空依赖建模与预测输出全过程，可视为一种联合不确定性表征。具体地，在循环信号片段的关联学习过程中，片段重要性的判别本身具有不确定性。当输入样本受到噪声扰动、工况波动或退化模式复杂性的影响时，模型需要刻画不同片段之间关联强度的变化范围及其不确定程度，而非仅给出固定的注意力分配结果。进一步地，这种局部关联层面的不确定性会在空间通道交互与时间退化依赖建模过程中持续传播，并最终体现为电池健康状态估计结果的分布不确定性。因此，本章所涉及的不确定性可理解为退化特征聚合不确定性、时空依赖建模不确定性与预测结果不确定性的统一表征。
% 
相应地，本章强调的无偏性指向不确定性量化结果与真实误差风险之间的一致关系。若模型给出的置信区间与实际预测误差之间存在系统性偏离，则该类不确定性只能反映模型内部的数值波动，难以为可信决策提供有效支撑。基于这一认识，本章将无偏不确定性界定为经校准后能够真实表征预测风险的置信度表达。其量值既反映模型在不同样本上的估计不确定程度，也与实际误差概率保持一致，从而抑制过度自信或置信度不足等问题。基于上述理论认识，后续内容通过随机自注意力、层次化时空建模以及基于保序回归的校准机制，构建面向电池健康状态估计的无偏不确定性量化框架，并进一步利用校准后的不确定性支持分布内与分布外退化模式辨识。

\section{基于层次化随机时空 Transformer 的状态估计方法}\label{sec:HSSTT_methodology}

本节详细阐述所提出的 HSSTT 模型，系统性介绍实现无偏不确定性量化和可信 SOH 估计的方法，同时对该模型在实际工业场景中的部署可行性进行分析。
如图~\ref{fig:hsstt_framework} 所示，该方法自底向上地集成多个模块。
首先，为实现对循环信号片段间关联信息的聚合，构建基于 GSR 的随机自注意力机制。
其次，为捕获 LIBs 的复杂时空退化动态，设计计及不确定性的层次化随机 Transformer，其所提取的特征通过下游回归器映射至 SOH 估计值。
最后，模型的不确定性通过蒙特卡洛采样（Monte Carlo Sampling, MCS）量化，引入偏差-方差分解阐述估计精度和不确定性量化质量的权衡关系，并设计基于保序回归的不确定性校准策略以修正置信度偏差。

\subsection{基于 GSR 的随机自注意力机制}\label{subsec:stochastic_attention}

为解决因忽略模型不确定性导致的循环信号片段聚合模糊性问题，本章提出一种基于 GSR 的自注意力机制。该机制是构建 HSSTT 模型的基础。

基于 Transformer 的模型所具备的上下文建模能力，主要源于多头自注意力（Multi-Head Attention, MHA）机制。以第 $j$ 个注意力头为例，输入序列 $X=[x^{(1)}, x^{(2)}, \dots, x^{(T)}] \in \mathbb{R}^{d_{in} \times T}$ 经由可训练的投影矩阵 $W_Q^{(j)}, W_K^{(j)}, W_V^{(j)} \in \mathbb{R}^{d_{emb} \times d_{in}}$ 执行仿射变换，生成查询矩阵 $Q^{(j)}$、键矩阵 $K^{(j)}$ 和值矩阵 $V^{(j)}$。其中，$T$ 为每个序列样本的总循环数，$d_{in}$ 和 $d_{emb}$ 分别表示输入维度与嵌入维度。

由 Softmax 函数生成的注意力权重 $A^{(j)}$ 具有离散概率分布的形式，表明该权重可视为从类别分布中采样的结果 \cite{20193407356730, 20230613569727}。通过将 Softmax 函数替换为 Gumbel-Softmax 函数，并以概率采样方法取代确定性计算，即可将标准自注意力转换为随机化变体。此项转换使每个循环信号片段的注意力权重转变为具有多个离散结果的类别分布，其形式表述为
% 
\begin{equation}
  \tilde{A}^{(j)} \sim \mathcal{G}((K^{(j)\top}Q^{(j)}) / \tau)
\end{equation}
% 
其中，$\mathcal{G}$ 表示 Gumbel-Softmax 分布，$\tau$ 为温度参数。从而，经由自注意力机制计算得到的权重 $\tilde{A}^{(j)}$，从确定性的归一化对数几率（logits）转变为从特定离散分布中采样的随机变量。通过这种方式，确定性自注意力转化为具有不确定性量化能力的随机形式。

构建随机自注意力的基础在于对离散类别分布进行连续松弛，前述 GSR 技术可有效实现该过程。考虑一个非负且不全为零的离散类别分布 $(\pi_1, \pi_2, \dots, \pi_T)$，其表征经过自注意力机制计算得到的特定循环信号片段的注意力权重。具体地，将矩阵 $K^{(j)}$ 和 $Q^{(j)}$ 按列划分为向量 $k_i$ 与 $q_i$，其内积 $\pi_i = k_i^\top q_i$ 构成该分布的非归一化参数，其中 $i$ 为列索引。为实现从该类别分布中的离散采样，首先引入 Gumbel-Max 方法构造独热向量 $z$，定义为
% 
\begin{equation}
  z = \text{one\_hot}(\text{argmax}(g_i + \log(\pi_i)))
\end{equation}
% 
其中，$T$ 为该离散类别分布中所有潜在结果的总数，对应于输入循环信号片段的数量。$g_i$ 是服从 $\text{Gumbel}(0,1)$ 分布的随机变量，定义为
% 
\begin{equation}
  g_i = -\log(-\log(u_i))
\end{equation}
% 
其中，$u_i$ 在 0 和 1 之间均匀分布，$g_i$ 和 $u_i$ 均为独立同分布的简单随机样本。$g_i$ 亦称为 Gumbel 噪声。由于上述 argmax 操作不可微，采用 GSR 技术将其转化为连续形式，定义为
% 
\begin{equation}
\tilde{A}^{(j)} = \frac{\exp((g_i + \log(\pi_i))/\tau)}{\sum_{k=1}^{T}\exp((g_k + \log(\pi_k))/\tau)}
\end{equation}

该式与前述 Gumbel-Softmax 公式等价。在 MHA 框架下，$\tau$ 可作为原始缩放因子 $1/d_{emb}$ 的替代。从 GSR 的角度看，上式的右侧部分能够紧密逼近离散类别分布 $(\pi_1, \pi_2, \dots, \pi_T)$。当 $\tau$ 趋近于零时，Gumbel-Softmax 分布愈发接近原始的离散类别分布。采用 GSR 进行连续松弛，将不可微的类别分布转化为平滑且可微的 Gumbel-Softmax 分布。这一转换为随机自注意力的训练直接应用反向传播算法提供可能。在训练初期，$\tau$ 设定为相对较大的值，并随着模型收敛而逐步减小，以在训练速度与性能之间取得平衡。

GSR 技术通过将 Gumbel 噪声引入 logits，并对合成后的向量施加 Softmax 函数来生成注意力权重。
% 
为阐述这一机制与 MCD 等变分推断方法的理论一致性，需考察 Gumbel 噪声与高斯噪声之间的解析映射关系 \cite{potapczynski2020invertible}。给定标准高斯随机变量 $\epsilon \sim \mathcal{N}(0, 1)$，其累积分布函数定义为 $\Phi(\epsilon)$。根据概率积分变换定理，变量 $u = \Phi(\epsilon)$ 严格服从标准均匀分布 $\mathcal{U}(0, 1)$。由于 Gumbel 噪声 $g_i$ 经由均匀分布样本的逆变换采样生成，将 $u$ 的表达式代入前述 Gumbel 噪声公式，建立两者之间的严格双射如下
% 
\begin{equation}
  g_i = -\log(-\log(\Phi(\epsilon)))
\end{equation}
% 
MCD 等变分近似方法的核心理论基础在于通过注入高斯扰动或可渐近等效为高斯分布的随机噪声，以逼近神经网络权重的后验分布。从而，基于 GSR 的离散变量连续松弛在概率测度空间上与连续高斯扰动同构。这种同构性质从数学层面论证，采用 GSR 驱动的随机机制与基于 MCD 的连续空间近似方法具有一致的不确定性量化理论基础。
% 
本文在第 \ref{subsec:HSSTT_ablation_study} 小节呈现的实验结果进一步表明，所提方法在建模模型不确定性方面优于基于 MCD 的方法。

将确定性注意力机制随机化，为阐明模型内部的决策过程提供直接可能性。随机化不仅实现不确定性量化，同时提升模型的可解释性。将注意力权重概念化为概率分布后，其在多次随机前向传播中的方差即可作为可量化指标，用于衡量模型对每个输入循环信号片段显著性的置信度。低方差表明模型对特定特征作为稳定退化指标的有效性具有高置信度；反之，高方差则意味着模型在处理含噪或模糊的数据片段时存在不确定性。这种内嵌的可解释性为生成关于输入数据的不确定性图谱提供可能，并为后续可信特征提取过程奠定基础。

\subsection{计及不确定性的层次化随机 Transformer}\label{subsec:hierarchical_spatialtemporal}

受层次化时空建模 \cite{qianhstformerhierarchicalspatialtemporaltransformers} 与概率 Transformer \cite{9879185} 领域并行发展的启发，设计一种层次化随机时空 Transformer，旨在融合这两种策略的建模优势，以应对 LIBs 退化建模中固有的复杂性与可信度双重挑战。
% 
为提取空间信息，引入计及不确定性的膨胀卷积，用以生成感知跨传感器依赖性的嵌入向量。随后，将该嵌入向量送入距离敏感的随机 Transformer 编码器执行时序建模。该编码器内部集成一种聚类机制以处理特征稀疏性问题，进而构成层次化的 Transformer 结构。

具体地，为挖掘跨传感器间依赖关系，引入计及不确定性的时间步卷积方法。该方法作用于第 $t$ 个循环期间从 $S$ 个传感器、长度为 $L$ 的范围内采集的多变量时间序列数据 $r^{(t)} \in \mathbb{R}^{S \times L}$，具备以下关键特性：

（1）采用膨胀卷积扩展感受野，从而识别不同时间步上不同传感器间的潜在关联。

（2）利用随机自注意力，通过经不确定性重加权的注意力机制识别主导传感器通道。

（3）设定大于 1 的卷积核水平分量，以建模跨传感器依赖性。

（4）将卷积生成的特征图展平为列向量，作为样本 $X_i$ 中特定于该时间步的分量。

经卷积处理，$r^{(t)}$ 转换为 $x^{(t)} \in \mathbb{R}^{d_{in} \times 1}$。

在时间维度，退化动态的表征通过自注意力机制建模。鉴于传统 Transformer 所采用的正弦位置编码可能致使模型忽略对 SOH 估计关键的循环间距离信息，受文献 \cite{wu-etal-2021-da} 启发，通过修正随机自注意力权重 $\tilde{A}^{(j)}$ 将真实距离信息融入模型。针对给定时间序列 $X$，任意两个 token 位置 $p$ 和 $q$ 之间的距离定义为 $r_{p,q}=|p-q|$，由此构建距离矩阵 $R=(r_{p,q}) \in \mathbb{R}^{T \times T}$。此外，为 MHA 中的每个注意力头分配可学习参数 $\omega$ 以动态调整距离敏感度。以第 $j$ 个头为例，其特定的距离矩阵 $R^{(j)}=\omega^{(j)}R$ 促使模型区分长程与短程信息，其中正值 $\omega$ 强化长程依赖，负值则强化短程依赖。为将加权距离矩阵 $R^{(j)}$ 与自注意力无缝融合，利用动态 Sigmoid 函数将 $R^{(j)}$ 投影为重缩放系数，投影后的距离矩阵定义为
% 
\begin{equation}
% \hat{R}^{(j)}=f(R^{(j)},V^{(j)})=\frac{1+\exp(V^{(j)})}{1+\exp(V^{(j)}-R^{(j)})}
\hat{R}^{(j)} = \frac{1+\exp(V^{(j)})}{1+\exp(V^{(j)}-R^{(j)})}
\label{eq:distance_matrix_coefficient}
\end{equation}
%
其中，$V^{(j)}$ 为第 $j$ 个头的可训练参数，控制 Sigmoid 函数的上限与上升陡度。随后，采用 $\hat{R}^{(j)}$ 对前述随机注意力权重进行调整，修正后的注意力权重矩阵为
% 
\begin{equation}
% \hat{A}^{(j)} \sim \mathcal{G}\left(\frac{\sigma({K^{(j)}}^\top Q^{(j)}) * \hat{R}^{(j)}}{\tau}\right)
\hat{A}^{(j)} \sim \mathcal{G}((\sigma({K^{(j)}}^\top Q^{(j)}) * \hat{R}^{(j)}) / {\tau})
\label{eq:distance_sensitive_self_attention}
\end{equation}
%
其中，$*$ 为哈达玛积，$\sigma$ 为激活函数，此处选用 ReLU 函数以确保数值非负性。

为使模型在稀疏特征空间内高效提取与性能退化状态相关的健康特征，同时免受冗余或不相关信息干扰，引入一种层次化随机自注意力机制 \cite{clustered_attention_vyas2020fast}。该机制在对注意力权重进行采样前，首先对键（Key）向量头执行聚类。此操作等效于对 Key 注意力头施加正则化，使模型聚焦于预定义的稀疏聚类中心。将 Softmax 替换为 Gumbel-Softmax，聚类前后这一近似的完整性仍得以保持，以保证聚类前后键注意力的差异始终有界。进一步地，摒弃通常的如文献 \cite{clustered_attention_vyas2020fast} 中建议的在 Hamming 空间运用  Lloyd 算法执行 $k$-均值聚类的聚类迭代方法，将质心矩阵设定为可训练矩阵，通过梯度下降求解最优聚类中心并完成聚类操作。该策略有助于模型在收敛过程中高效探寻最优簇划分。此外，层次化机制将自注意力的二次计算复杂度降维至线性 \cite{clustered_attention_vyas2020fast}，从而有效缓解变分推断引入的计算开销，提升模型响应速率。

所提出的层次化随机距离敏感自注意力机制形式化表述为
\begin{align}
\label{eq:attention_kqv} & Q^{(j)}, K^{(j)}, V^{(j)} = W_{Q}^{(j)}X, W_{K}^{(j)}X, W_{V}^{(j)}X \\
\label{eq:hsstt_k} & \hat{A}_{C} \sim \mathcal{G}(({K^{(j)}}^\top C) / \tau_{1}) \\
\label{eq:hsstt_clustered_k} & \hat{K}^{(j)} = \hat{A}_{C}C^\top \\
\label{eq:hsstt_hierarchical_attention} & \hat{A}^{(j)} \sim \mathcal{G}((\sigma(\hat{K}^{(j)\top} Q^{(j)})*\hat{R}^{(j)}) / \tau_{2}) \\
\label{eq:hsstt} & \hat{H}^{({j})}=\hat{A}^{({j})}V^{{(j)}} \\
\label{eq:hsstt_mha} & \widehat{MHA}=concat(\hat{H}^{(1)},\hat{H}^{(2)},\cdots,\hat{H}^{(h)})W_{O}
\end{align}
%
其中，$C \in \mathbb{R}^{d_{emb} \times C}$ 为质心集合，$\hat{A}_C$ 是从 Gumbel-Softmax 分布中采样的质心权重，$\hat{K}^{(j)}$ 为聚类后的 Key 向量头，$\tau_1$ 和 $\tau_2$ 为温度系数，$h$ 为注意力头总数，$W_O \in \mathbb{R}^{h \times d_{emb} \times d_{in}}$ 为可训练的投影矩阵。

获取 LIBs 退化模式表征后，由多层全连接层构成的回归器将上述特征图映射为 SOH 估计值。

不确定性引导 HSSTT 的特征提取过程，促使模型动态地将注意力资源分配给表现出高置信度的特征。该机制在空间与时间两个维度实施，即计及不确定性的卷积与随机 Transformer 均倾向于抑制高方差特征的影响，进而削弱含噪传感器通道与模糊时间片段的影响。此过程产生抵抗不相关信息的鲁棒特征表征。
% 
SOH 估计的全局预测不确定性实质上由上述时空特征提取阶段的局部不确定性聚合构成。借由该聚合机制，模型在特征选择阶段的置信度被传递为估计结果的置信度，进而在内部推理过程与外部输出可信度之间构建逻辑关联。

\subsection{基于偏差-方差分解与保序回归的不确定性校准}\label{subsec:uncertainty_calibration}

为获取准确且无偏的不确定性估计，即确保模型输出的置信度严格反映其实际的预测误差水平，进而提升 HSSTT 模型的可信度，本节系统讨论不确定性的校准问题。首先，基于偏差-方差分解视角，从理论层面剖析模型产生过自信或欠自信结果的内在成因，明确不确定性校准的必要性。随后，提出一种数理严密的模型无关校准策略，旨在对齐 HSSTT 模型的预测不确定性与数据样本导出的经验不确定性。

经验观察表明，现有的计及不确定性方法呈现出一种两难趋势，具备较高估计精度的模型所量化的不确定性存在置信度偏差，而能够提供高质量不确定性量化的模型的估计精度则较低。
% 
采用偏差-方差分解对该现象进行理论分析，这种权衡关系可以形式为
\begin{align}
\label{eq:bias_variance_tradeoff} & E((y-\hat{f}(X))^{2}) = Bias^{2}(\hat{f}(X))+Var(\hat{f}(X))+\sigma^{2} \\
\label{eq:bias} & Bias(\hat{f}(X)) = E(\hat{f}(X))-E_{y \mid X}(f(X)) \\
\label{eq:variance} & Var(\hat{f}(X)) = E((E(\hat{f}(X))-\hat{f}(X))^{2})
\end{align}
%
其中，$(X,y)$ 为测试样本，$f(\cdot)$ 为真实函数，$\hat{f}(\cdot)$ 为预测函数。模型分别采用后验采样的均值 $E(\hat{f}(X))$ 和方差 $\text{Var}(\hat{f}(X))$ 进行 SOH 估计与不确定性量化。因此，如上式所示，这类模型内在地对估计值与不确定性的组合量级施加一个上限，从而在精度与置信度之间形成权衡关系。
% 
给出过自信或欠自信结果的现象也称为模型校准失当 \cite{kuleshov2018accurate}，表现为由预测不确定性表征的分布与实际数据分布之间的不一致。

不同于情感分类等任务 \cite{20230613569727}，在 LIBs 退化建模这类安全关键场景中，开环的不确定性量化并不完备。因此，模型的不确定性校准对于弥合置信度偏差鸿沟、实现可信 SOH 估计至关重要。
% 
为简化表述，下文将 HSSTT 模型简记为 $f$。视数据样本 $(X_i,y_i) \in \mathcal{D}_X \times \mathcal{D}_Y$ 为源于总体 $\mathcal{X}$ 和 $\mathcal{Y}$ 的观测，其分布由 $\mathbb{P}$ 表示。随机编码器输出 $y_i$ 的累积分布函数（Cumulative Distribution Function, CDF）记为 
% 
\begin{equation}
  F_i(\cdot): \mathcal{D}_Y \rightarrow [0,1]
\end{equation}
% 
该函数 $F_i(\cdot)$ 具有单调非递减性质，其逆函数 $F_i^{-1}(\cdot): [0,1] \rightarrow \mathcal{D}_Y$ 被称为分位数函数，定义为
% 
\begin{equation}
  F_i^{-1}(p)=\inf\{y \mid p \leq F_i(y)\}
\end{equation}
% 
其中 $\inf(\cdot)$ 为下确界。

模型 $f$ 被认为是良好校准的，当且仅当
% 
\begin{equation}
\forall p \in [0,1], \sum\nolimits_{i=1}^{N} \mathbb{I}\{y_{i}\leq F_{i}^{-1}(p)\}/N\rightarrow p,N\rightarrow\infty
\label{eq:model_calibration_iff}
\end{equation}
%
其中，$N$ 是样本数量，$\mathbb{I}\{\cdot\}$ 是指示函数。
考虑数据分布 $\mathbb{P}$，式 \eqref{eq:model_calibration_iff} 可以写作
\begin{equation}
\forall p\in[0,1], \mathbb{P}(\mathcal{Y}\leq F_{\mathcal{X}}^{-1}(p))=p
\label{eq:model_calibration_sufficient}
\end{equation}%
其中 $\mathbb{P}$ 为连续分布。式~\eqref{eq:model_calibration_sufficient} 也是模型良好校准的一个充分条件 \cite{kuleshov2018accurate}。

基于式 \eqref{eq:model_calibration_iff} 和式 \eqref{eq:model_calibration_sufficient}，
修正校准失当的模型需要寻找一个映射 $g: [0,1] \rightarrow [0,1]$，使模型的输出分布与实际数据分布对齐。因此，复合函数 $g \circ f$ 将产出所期望的良好校准模型。
% 
为确定映射 $g$，利用数据集 $\mathcal{D}=\{(X_i,y_i)\}_{i=1}^{N}$ 和模型 $f$ 构建校准数据集 $\mathcal{S}=\{(F_i(y), \hat{p}(F_i(y)))\}_{i=1}^{N}$。其中，$F_i(y)$ 表示由 $f$ 生成的预测 CDF，而 $\hat{p}(F_i(y))$ 表示真实的经验 CDF，定义为
% 
\begin{equation}
\begin{aligned}
% \hat{p}(F_{i}(y))&=|\{y_{i}|F_{i}(y)\leq p,i=1,2,\cdots, N\}|/N \\
% &=\sum\nolimits_{i=1}^{N}\mathbb{I}\{F_{i}(y)\leq p\}/N
\hat{p}(F_{i}(y)) = \sum\nolimits_{i=1}^{N}\mathbb{I}\{F_{i}(y)\leq p\}/N
\end{aligned}
\label{eq:true_empirical_cdf}
\end{equation}%

在数据集 $\mathcal{S}$ 上训练的模型 $g$ 即为校准模型。鉴于 CDF 的非递减性质，将 $g$ 指定为保序回归器。

不确定性校准的意义不仅限于调整预测区间。良好校准得到的无偏的不确定性能够准确反映模型的真实置信度，进而提升模型可解释性和可信性。校准机制确保内部的注意力方差忠实地反映数据中固有的证据不确定性，免受模型系统性偏差的影响。由此产生的不确定性图谱是更可靠的特征相关性表征。

\subsection{基于无偏不确定性的可信 SOH 估计}\label{subsec:trustworthy_soh_estimation}

为提升 OOD 退化模式检测性能并实现可信的 SOH 估计，基于 HSSTT 构建完整预测框架。
本小节概述这一基于无偏不确定性量化的估计和检测框架，包括 SOH 估计结果的生成、不确定性的量化与校准机制和应用所得到的不确定性识别分布外退化模式。此外，针对 HSSTT 的复杂度分析进一步论证其在工业场景中部署的适用性。

给定测试样本 $X$ 的循环信号输入，HSSTT 对目标 SOH 值 $y$ 产生的后验分布为
% 
\begin{equation}
p(y \mid X,D)=\int_{\theta}p(y \mid X,\theta^{\prime})p(\theta^{\prime} \mid D)d\theta^{\prime}
\label{eq:posterior_distribution}
\end{equation}
%
其中，$p(\theta \mid D)$ 为给定训练集 $D$ 条件下模型参数 $\theta$ 的后验分布。该计算过程涉及从 $\theta$ 的后验分布中采样 $B$ 个参数观测值，进而实例化 $B$ 个并行网络。此参数观测值集合记作 $\Theta$，对应的 $B$ 次前向传播输出结果集合记作 $Y$。集合 $Y$ 的统计均值提供模型的点估计，而标准差表征模型的不确定性，SOH 估计值及其不确定性定义为
% 
\begin{align}
  \label{eq:predictive_mean} \hat{y} & =\frac{1}{|\Theta|}\sum_{\theta^{(i)} \in \Theta}f_{\theta^{(i)}}(X) \\
  \label{eq:predictive_std} \hat{\sigma} & =\sqrt{\frac{1}{|\Theta|-1}\sum_{\theta^{(i)}\in \Theta}(f_{\theta^{(i)}}(X)-\hat{y})^{2}}
\end{align}

区间 $[\hat{y} - 3\hat{\sigma}, \hat{y} + 3\hat{\sigma}]$ 被定义为 SOH 的区间估计。不确定性 $\hat{\sigma}$ 依据第 \ref{subsec:uncertainty_calibration} 小节所述方法执行校准。实现可信 SOH 估计的完整端到端流程归纳于算法 \ref{alg:hsstt}。
% 
模型不确定性被用于衡量模型的置信度。较低的预测不确定性表征对估计结果的高置信度；反之，较高不确定性反应较低的置信水平。据此，模型不确定性构成区分 ID 退化样本与 OOD 退化样本的潜在判据。若模型针对特定测试样本输出较低的不确定性，表明该样本特征分布与训练数据高度相似，进而将其归类为 ID 样本；反之，显著升高的不确定性则标志其为 OOD 样本。上述基于数据分布特性的认知机制有效增强 SOH 估计后的下游决策过程。

\begin{algorithmic}[1]
    \Require 训练集 $\mathcal{D}=\{(X_{i}, y_{i})\}_{i=1}^{N}$，测试样本 $X$，质心矩阵 $C$，训练轮数 $n_{epoch}$，蒙特卡洛采样次数 $B$
    \Ensure 校准模型 $g \circ f$，估计的 SOH $\hat{y}^{*}$ 及校准不确定性 $\hat{\sigma}^{*}$
    
    \Statex \textbf{阶段 1: 层次化随机模型训练}
    \For{epoch $=1$ \textbf{to} $n_{epoch}$} \Comment{迭代至收敛或达到最大轮数 $n_{epoch}$}
        \ForAll{$(X_{i}, y_{i}) \in \mathcal{D}$}
            \State $Q^{(j)}, K^{(j)}, V^{(j)} \leftarrow \textsc{SpatialEmbedding}(X_{i})$ \Comment{计算计及不确定性的空间嵌入}
            \State $\hat{R}^{(j)} \leftarrow \frac{1+\exp(V^{(j)})}{1+\exp(V^{(j)}-R^{(j)})}$ \Comment{距离敏感矩阵的动态 Sigmoid 投影}
            \State $\hat{A}_C \sim \mathcal{G}\left((K^{(j)\top} C) / \tau_1\right)$ \Comment{质心权重 Gumbel-Softmax 采样}
            \State $\hat{K}^{(j)} \leftarrow \hat{A}_C C^\top$ \Comment{重参数化计算聚类键向量}
            \State $\hat{A}^{(j)} \sim \mathcal{G}\left((\sigma(\hat{K}^{(j)\top} Q^{(j)}) * \hat{R}^{(j)}) / \tau_2\right)$ \Comment{计算距离敏感随机注意力}
            \State $\hat{y}_{i} \leftarrow \textsc{Regressor}\big(\textsc{Concat}_{j}(\hat{A}^{(j)}V^{(j)})W_{O}\big)$ \Comment{融合特征并映射至 SOH}
            \State $\mathcal{L} \leftarrow \lVert y_{i} - \hat{y}_{i} \rVert_2^2$ 
            \State $\theta \leftarrow \theta - \eta \nabla_{\theta}\mathcal{L}$ \Comment{计算 MSE 损失并执行梯度下降}
        \EndFor
    \EndFor
    \State $p(\theta \mid \mathcal{D}) \leftarrow \textsc{Posterior}(\theta)$ \Comment{确定模型参数的后验分布}
    
    \Statex \textbf{阶段 2: 不确定性校准}
    \State $\mathcal{S} \leftarrow \emptyset$
    \ForAll{$(X_{i}, y_{i}) \in \mathcal{D}$}
        \State $\hat{y}_i, \hat{\sigma}_i \leftarrow \textsc{MCEstimate}(X_i, p(\theta \mid \mathcal{D}), B)$ \Comment{采样 B 次获取均值与方差}
        \State $F_i(\cdot) \leftarrow \textsc{CDF}(\hat{y}_i, \hat{\sigma}_i)$ \Comment{生成模型输出的累积分布函数}
        \State $\mathcal{S} \leftarrow \mathcal{S} \cup \{F_i(y_i)\}$
    \EndFor
    \State $\hat{p}(\cdot) \leftarrow \textsc{EmpiricalCDF}(\mathcal{S})$ \Comment{依据所有样本计算真实的经验分布}
    \State $g \leftarrow \textsc{IsotonicRegression}\big(\{(F_i(y_i), \hat{p}(F_i(y_i)))\}_{i=1}^N\big)$ \Comment{训练保序回归器}
    
    \Statex \textbf{阶段 3: 推理与可信估计输出}
    \State $\Theta^{*} \sim p(\theta \mid \mathcal{D})$ \Comment{针对测试集抽取 B 个网络参数观测值}
    \State $Y^{*} \leftarrow \{f_{\theta}(X) \mid \theta \in \Theta^{*}\}$ \Comment{执行多次随机前向传播}
    \State $\hat{y}_{raw}, \hat{\sigma}_{raw} \leftarrow \textsc{Mean}(Y^{*}), \textsc{Std}(Y^{*})$ \Comment{计算点估计与原始不确定性}
    \State $\hat{y}^{*}, \hat{\sigma}^{*} \leftarrow g(\hat{y}_{raw}, \hat{\sigma}_{raw})$ \Comment{通过复合函数输出校准后结果}
    \State \Return $g \circ f$, $\hat{y}^{*}$, $\hat{\sigma}^{*}$
\end{algorithmic}

HSSTT 的计算负载通过集成的聚类注意力机制得到优化，使得所提出的框架在工业级设备上部署具有切实潜力。

通过优先对 Key 向量执行聚类操作，继而逐簇计算注意力矩阵，HSSTT 将原始 Transformer 的二次计算复杂度降低到线性。具体地，给定的聚类中心数 $n_{\text{c}}$，自注意力模块的计算复杂度优化为 $\mathcal{O}(T \cdot n_{\text{c}} \cdot d_{\text{emb}})$，相对输入序列长度 $T$ 具有线性增长关系。

本节构建一种兼具可解释性与可靠性的可信 SOH 估计框架。在特征提取阶段，随机注意力机制于退化特征层面量化不确定性，进而引导 HSSTT 模型聚焦输入数据中最可靠的证据以构建鲁棒退化表征。随后，数理严密的校准方法对模型量化的原始不确定性开展无偏校准，导出具备严格统计意义的无偏置信区间。以校准后的不确定性作为可靠的诊断指标，实现 ID 与 OOD 样本的有效辨识，这在安全关键型应用场景的实际部署中具备重要工程价值。所提出的框架使用通用的数据驱动模型训练策略，即可使 HSSTT 不仅输出高精度的点估计值，同时对其估计结果的内在可信性提供系统评估。

模型采用连续 $T$ 个循环的直接测量数据构造样本输入。每个循环的数据包含由物理传感器采集的电流、电压以及相对采样时间三个通道上的序列。
数据预处理阶段首先对所有采集信号按通道执行最大最小值归一化，随后进行下采样以统一特征维度，最后将重采样后的时间序列水平堆叠，构成模型输入样本 $X$。进而对样本 $X$ 施加膨胀卷积以提取跨传感器依赖关系。卷积生成的特征图进展平为列向量，用以创建进行时间步编码的特征表示。

所有实验均在配备 Intel Core i9-13900KF CPU 与 NVIDIA 4090 GPU 的服务器上开展，软件环境统一配置为 PyTorch 2.1.0 及 CUDA 11.8。
模型评估时严格执行留一交叉验证（Leave-One-Out Cross-Validation, LOOCV）规则。具体地，该规则依次排除单一电池作为独立测试集，并将剩余样本按 90\% 和 10\% 的比例划分为训练集与验证集。交叉验证循环持续至每个电池均完成测试集评估。此设定在确保模型泛化能力得到全面验证的同时，严格保持训练、验证与测试阶段的数据隔离。模型训练阶段采用均方误差（Mean Squared Error, MSE）作为损失函数，并通过 Adam 优化器执行参数更新。
最大训练轮数设定为 100，同时引入基于验证集损失的早停机制以规避过拟合风险。HSSTT 模型的关键超参数依据第 \ref{subsec:HSSTT_sensitivity_analysis} 小节详述的敏感性分析进行系统优化，具体参数取值归纳于表 \ref{tab:HSSTT_hyperparams}。

评估指标方面，采用平均绝对误差（Mean Absolute Error, MAE, $\downarrow$）和均方根误差（Root Mean Square Error, RMSE, $\downarrow$）衡量 SOH 估计精度。
其中，括号内的 $\uparrow$ 与 $\downarrow$ 符号用于标明指标的优劣方向。$\uparrow$ 表示该指标数值越高模型性能越优，$\downarrow$ 表示该指标数值越低模型性能越优。
MAE 反映估计值与真实值之间误差的绝对平均水平，定义为
% 
\begin{equation}
  MAE = \frac{1}{N} \sum_{i=1}^{N} |y_i - \hat{y}_i|
\end{equation}
% 
RMSE 侧重于惩罚较大的估计误差，定义为
% 
\begin{equation}
  RMSE = \sqrt{\frac{1}{N} \sum_{i=1}^{N} (y_i - \hat{y}_i)^2}
\end{equation}
% 
其中，$N$ 表示样本总数，$y_i$ 与 $\hat{y}_i$ 分别表示真实与估计的 SOH 值。
% 
采用负对数似然（Negative Log-Likelihood, NLL, $\downarrow$）评估数据拟合质量以及 HSSTT 模型的预测不确定性。假定模型输出遵循高斯分布，NLL 定义为

\begin{equation}
  NLL = \frac{1}{N} \sum_{i=1}^{N} \left[ \frac{\ln(2\pi\sigma_i^2)}{2} + \frac{(y_i - \hat{y}_i)^2}{2\sigma_i^2} \right]
\end{equation}
% 
其中，$\sigma_i^2$ 为模型输出的预测方差。同时，引入期望校准误差（Expected Calibration Error, ECE, $\downarrow$）评估不确定性量化质量 \cite{kuleshov2018accurate, nemani2023uncertainty}。其通过计算不同置信度区间内的估计准确率与平均置信度的加权偏差量化所输出的不确定性的校准程度，定义为

\begin{equation}
  ECE = \sum_{m=1}^{M} \frac{|B_m|}{N} |acc(B_m) - conf(B_m)|
\end{equation}
% 
其中，$M$ 为划分的区间数量，$B_m$ 为第 $m$ 个区间的样本集合，$acc(\cdot)$ 与 $conf(\cdot)$ 分别表示该区间的准确率与置信度。

除上述定量指标外，SOH 估计、不确定性量化以及模型校准的可视化分析为模型综合评估提供直观的定性补充。

讨论具体的对比方法，各模型在估计精度、不确定性量化质量与计算代价之间呈现出不同特征。
MRVR 与 GP-ICE 虽然在部分数据集上取得较高的点估计精度，但受限于模型时空退化信息表达能力和分布建模能力，其预测结果对噪声敏感，并表现出不稳定性。
同时，这两类方法在无偏量化方面仍存在不足，因而难以满足可信 SOH 估计的要求。
DeNN 在一定程度上改善了上述问题，但其集成式建模策略显著增加了训练与部署阶段的计算负担，从而限制了实际应用的可行性。
% 
BMNN 采用贝叶斯反向传播（Bayes by Backprop, BBB）范式实现参数分布建模，但这种策略同样引入较高的计算开销。
此外，该方法将 SOH 估计与不确定性量化过程紧密耦合，因此容易受到输入特征分布稀疏性的影响，进而制约估计精度。
SNGP 能够通过单次前向传播获得输出后验分布的统计信息，在计算形式上更为高效，然而，当输入信息存在稀疏性时，该方法仍可能产生偏差较大的 SOH 估计结果，并降低不确定性量化的可靠性。
BDAE 基于 MCD 范式实现不确定性量化，但其性能对超参数设置敏感，通常依赖较为复杂的参数搜索过程。
% 
此外，上述方法侧重刻画退化过程中的时间相关性，对传感器信号之间的空间相关性考虑不足。
SACN 通过胶囊网络提取时空特征以弥补这一局限，但在输入特征稀疏条件下，其性能仍受到明显制约，因此难以实现兼具精度与可靠性的 SOH 估计。
对比之下，HSSTT 在模型设计中同时考虑估计精度、不确定性量化可靠性以及时空相关特征建模等关键因素，因此能够获得可信的电池状态预测结果。图~\ref{fig:hsstt_comparison} 给出了相应的定性比较结果，以进一步说明各方法之间的性能差异。

MCDT 在 4 个数据集上的 RMSE 与 NLL 整体偏高，说明基于随机失活的注意力扰动难以稳定刻画 SOH 退化过程的概率分布。

表 \ref{tab:ablation_study_2} 表明，经过校准后，4 个数据集上的平均 ECE 均下降。其中，MIT、TRI、TJU 与 XJTU 数据集的平均 ECE 分别由 26.188、28.567、31.980 和 37.155 降至 8.554、12.020、11.365 和 12.394。若从 4 个数据集的整体平均水平考察，ECE 由 30.973 降至 11.083，相对降幅为 64.2\%。更为重要的是，表中所有单体电池的 ECE 在校准后均出现下降，说明该改进并非来源于少量样本的偶然波动，具有较好的跨样本一致性。
% 图 \ref{fig:hsstt_calibration} 所示校准曲线进一步表明，校准后的置信度曲线整体更接近理想对角线，说明所采用的校准策略能够有效修正预测置信度与真实经验频率之间的偏差，从而缓解过置信与欠置信现象，提升模型输出分布的可信性。
校准曲线是评估模型预测概率分布与真实经验频率之间一致性的分析工具。该曲线以模型的期望置信水平作为横坐标，以在真实数据中统计得到的观测置信水平作为纵坐标绘制而成。在理想的概率预测模型中，输出的预测置信度应严格等于其实际发生的经验频率，反映在二维坐标系中，即校准曲线完全重合于从原点出发的对角线。当曲线轨迹偏离对角线时，其几何距离直接量化了模型的置信度误差。若曲线位于对角线下方，表明模型的期望置信水平普遍高于实际观测水平，存在过置信缺陷；反之，则说明模型存在欠置信现象。图 \ref{fig:hsstt_calibration} 所示校准曲线进一步表明，校准后的置信度曲线整体更接近理想对角线，说明所采用的校准策略能够有效修正期望置信水平与观测置信水平之间的偏差，从而缓解过置信与欠置信现象，显著提升模型输出分布的可信性。

消融实验从策略替换、模块剥离与后验校准三个层面共同验证 HSSTT 的设计合理性。GSR 策略为随机注意力提供更稳定的概率建模基础，层次化时空特征提取模块保证对复杂退化信息的充分表征，而后验校准进一步提升模型输出的置信度一致性。上述结果共同表明，HSSTT 的性能优势来源于各关键组件的协同作用，而非单一结构改动所带来的局部收益。

\subsection{分布内外退化模式辨识实验}\label{subsec:HSSTT_id_ood_identification}

为评估 HSSTT 在 OOD 退化样本检测问题中的表现，评估其所量化的无偏不确定性在分布内外样本辨识问题上的应用潜力，本小节设计两项对比实验，以模拟工业场景中常见的分布漂移问题。

图~\ref{fig:hsstt_sensitivity_ood}(a) 和 (b) 给出了两个数据集上电压信号与电流信号的核密度估计曲线。不同充放电协议下的信号分布存在差异。

任务 I 将训练集与测试集均设定为 MIT 数据集，对应测试样本为 ID 样本。任务 II 则在 TRI 数据集上训练模型，并在 MIT 数据集上进行测试。考虑到前述特征分布差异，任务 II 中的 MIT 测试样本被定义为 OOD 样本。
为定量评估模型在两类样本上的估计行为，采用 NLL 和 锐度（Sharpness）作为不确定性评价指标，其中，
Sharpness 是模型在测试集上输出的预测标准差的平均值，定义为
% 
\begin{equation}
  \text{Sharpness} = \frac{1}{N}\sum_{i=1}^{N}\hat{\sigma}_{i}
\end{equation}
% 
其中 $\hat{\sigma}_{i}$ 为第 $i$ 个样本的预测不确定性。

定量结果如表~\ref{tab:id_ood} 所示。对于 ID 样本，HSSTT 输出较低的不确定性水平，平均 Sharpness 为 0.024，平均 NLL 为 -2.760，说明模型在同分布测试条件下能够给出较为集中的预测分布，并保持较好的概率拟合能力。相比之下，在 OOD 条件下，模型的平均 Sharpness 提升至 0.054，平均 NLL 增加至 7.008，表明当输入样本偏离训练分布时，模型输出的不确定性显著上升。进一步观察各独立电池单元可以发现，这种变化趋势在不同测试对象上均保持一致，说明 HSSTT 对分布漂移具有稳定的响应特性。

\section{本章小结}\label{sec:HSSTT_conclusion}

针对 LIBs 的可信 SOH 估计问题，本章提出 HSSTT 模型。
首先，设计基于 GSR 的随机自注意力机制，以执行计及不确定性的底层退化特征提取。
继而，构建对序列片段距离敏感的层次化随机 Transformer 编码器，并将其与感知跨传感器不确定性的时间步卷积融合。
基于此，HSSTT 模型得以有效捕捉循环数据中的时空演化动态，精准提取与退化模式相关的核心特征，并实现不确定性量化。
进一步地，引入一种数理严密的校准回归器以修正潜在的置信度偏差与校准失当现象，为可信 SOH 估计，尤其是存在 ID 和 OOD 退化样本的 SOH 估计，提供保障。
综合实验结果充分论证，HSSTT 框架在预测精度与不确定性量化质量方面均显著优于现有主流方法，验证其理论设计的优越性。

% 尽管所提出的方法在特定测试条件下展现出卓越性能，现代工业环境对预测系统的泛化能力提出更高要求，即在不依赖海量独立同分布（Independent and Identically Distributed, IID）数据执行校准的前提下，维持高效稳定的可信输出。针对此需求，后续研究沿两个维度继续进行。其一，推进该框架在真实复杂工业场景中的规模化部署与验证，以严格评估其在强噪声干扰下的实际效能与工程鲁棒性。其二，集中研究数据依赖问题，研究基于不确定性量化的无源域自适应（Source-Free Domain Adaptation, SFDA）机制，以期在目标域数据分布发生偏移时实现模型的自主进化与校准。
