\chapter{绪论}\label{chap:Intro}

\section{研究背景及意义}\label{sec:Intro_background_and_significance}

% 【P1】
在全球能源结构加速向清洁低碳转型的战略背景下，交通运输行业作为温室气体排放的主要来源之一，其电气化升级对实现“碳达峰、碳中和”目标具有重要支撑作用。电动汽车作为新能源交通工具的典型代表，正在经历技术迭代与市场扩张并行的产业化发展阶段。据 2025 年国际能源署（International Energy Agency, IEA）统计\cite{iea2025global}，2024 年全球电动汽车年销量逾 1700 万辆，市场渗透率突破 20\%，全球在役车辆存量接近 5800 万辆，电动汽车已在全球范围内形成高保有量的常态化应用格局。
% 
在一系列宏观产业政策的持续驱动下\cite{statecouncil2021carbon, statecouncil2020nev}，我国电动汽车产业展现出强劲增长势头，据统计，“十四五”期间年均复合增长率约 46.15\%。伴随动力电池技术的突破与制造成本的显著下降，我国已在全球电动汽车产业链中占据主导地位。
随着市场保有量的激增与预期服役周期的延长，产业界对电动汽车的关注点逐渐由新车状态下的基础续航能力拓展至其全生命周期内的安全性与运行可靠性\cite{geslin2025dynamic, nguyen2025closing}。电动汽车的电驱动架构依赖复杂的异构动力系统。在长期服役的要求下，动力系统存在渐进式性能退化与突发故障风险，探究动力系统可靠性演化规律并实现主动健康管理是保障电动汽车长期安全服役的基础\cite{zhao2025bridging, he2020electrical}。

% 【P2】
为应对退化与故障风险，预测与健康管理（Prognostics and Health Management, PHM）技术被广泛引入，以保障整车全生命周期的可靠性。随着车载传感器技术与控制器局域网络（Controller Area Network, CAN）等车载总线技术的不断成熟，多源高维传感数据的规模化积累为数据驱动方法奠定了基础。传统依赖物理失效机理的 PHM 模型基于这些观测数据实现预测和诊断，但其信息融合有效性存在不足，此类模型难以精确描述复杂时变工况下动力系统高度非线性的动态演化过程，且在处理异构数据时存在计算瓶颈，面临模型泛化约束。近年来，深度学习技术凭借端到端的高维特征挖掘与表征能力，有效克服了传统方法对专家先验知识与繁杂手工特征工程的依赖，能够在海量数据中自适应捕捉非线性隐式退化规律，显著提升模型在复杂实际场景下的泛化鲁棒性，智能预测与健康管理（Intelligent PHM, IPHM）范式应运而生\cite{chen2025big, lei2025llmphm}。

电动汽车的动力系统主要由储能子系统和传动子系统构成。基于~IPHM~技术体系，本文的研究对象聚焦于支撑整车动力输出的两大核心易损部件。第一类部件是作为整车能量中枢的动力电池，其在长期的充放电循环中发生不可逆的性能退化。第二类部件是作为动力传动中枢的旋转机械，其在持续的高转速与变载荷工况下易产生疲劳并引发结构性故障。在本文的研究设定中，针对动力电池的退化建模被解耦为两个侧重点不同的评估维度。一方面，是健康状态（State of Health, SOH）估计，即估计当前最大可用容量与额定容量之比，以评估电池当前循环性能；另一方面，是剩余使用寿命（Remaining Useful Life, RUL）预测，即预测从当前循环到寿命终止（End of Life, EOL）循环间的循环数，以评估电池未来性能演化趋势。针对旋转机械，本文开展故障诊断研究，即识别系统的异常状态及具体损伤类别。
% 图\ref{fig:intro_ev_PHM}展示了本文所研究的电动汽车动力系统架构及其退化预测与故障诊断联合技术框架。

% 【P4】
动力电池的电化学衰退微观过程难以观测、持续时间长且退化特征隐蔽。动力电池性能表征量，无论是~SOH~或~RUL，都具有连续性，针对性能表征量的估计或预测在~IPHM~框架下属于回归问题。旋转机械的局部损伤通常由宏观瞬态冲击与高频振动引发，演变过程迅速。由于损伤宏观可观测，基于损伤的发生位置和严重程度，容易定义离散标签描述故障内容，基于~IPHM~框架，故障诊断问题具有典型的分类性质。鉴于动力电池与旋转机械在失效机理、演化时间尺度以及管理任务的形式上存在本质差异，本文针对两者的不同特性，分别设计了适配其演化规律和问题形式的研究路线。
尽管如此，针对动力电池和旋转机械的~IPHM~方法在应用时都面临未知工况和分布差异带来的挑战。具体地，动力电池受电极材料、环境温度和充放电协议等多重耦合因素影响，其实际运行数据与循环老化数据反映的退化路径不尽相同。旋转机械受负载情况、转速变化和装配方式等复杂内外部条件影响，其故障特征的类别分布随工况变化，类别边界随工况漂移。基于此，本文所涉及的两类方法均需要克服训练阶段历史数据和实际部署阶段测试数据间的分布不一致性。

% 【P5】
面向电动汽车关键系统的~IPHM~模型需要具备高度可靠性，这在数据分布存在不一致的情况下包括三个方面：可信目标要求模型不仅能给出点估计结果，还能准确量化自身推断的认知边界与不确定性；可泛化目标要求模型在存在测试工况跨越与域偏移时，仍能保持稳健的特征挖掘与推断能力；抗幻觉指在引入大语言模型（Large Language Models, LLMs）等先进方法开展诊断或决策时，能够有效抑制模型在面对非期望输入时生成脱离物理事实的虚假输出。为满足上述可靠性目标，亟需在算法层面建立对未知分布特征认知、量化与利用的闭环机制。为此，本文引入了广义贝叶斯范式。基于该范式，本文分别针对动力电池的 SOH 评估任务、RUL 预测任务，以及旋转机械的故障诊断任务，提出了三个~IPHM~模型，旨在为实际车载工况下的数据分布漂移难题提供新颖的理论框架，对推动下一代高性能新能源汽车的长效安全服役与规模化应用具有重要的工程价值与深远的现实意义。

\section{国内外研究现状}\label{sec:Intro_literature_review}

本节围绕动力系统中的动力电池和旋转机械两个核心部件的智能化退化建模和故障诊断的国内外研究现状进行综述。
% 
如前述，本文重点研究可靠~PHM~问题，并将可靠性分解为可信性、可泛化性和抗幻觉三个方面，并采用分研究路线的组织形式。
% 
相应地，本节根据研究内容重点回顾
\textbf{计及不确定性的电池健康状态估计}、
\textbf{面向分布漂移的电池退化模式}和
\textbf{幻觉鲁棒的旋转机械故障诊断}
三个部分的相关工作。
% 
这三个部分的探讨并不局限于单一研究对象，而致力于提炼相关研究的抽象机制。具体地，本节内容的目的为探究如何在连续退化过程预测中引入不确定性量化，以满足场景对可信性的要求；研究如何消除跨工况或跨设备等导致的数据分布偏移，以实现可泛化退化建模；以及探索如何检测深度模型，尤其是大语言模型，在面对未知或边界样本时产生的盲目自信，从而赋予模型抗幻觉能力。

\subsection{计及不确定性的健康状态估计研究现状}\label{subsec:SOH_estimation}

%     \centering
%     \includegraphics[width=\textwidth]{Img/Intro_figures/Gemini_Generated_Image_jzsifrjzsifrjzsi.png}
%     % \bicaption{确定性与概率性健康状态估计方法对比示意图}{}
%     \bicaption{确定性与概率性健康状态估计方法对比示意图}{Schematic comparison of deterministic and probabilistic state-of-health estimation methods}
%     \label{fig:intro_UQ_framework}
% \end{figure}

复杂工况下，点估计往往难以满足系统对高可靠决策的需求，引入不确定性量化技术以提升健康状态估计的可信性是必要环节。
% 
% 如图~\ref{fig:intro_UQ_framework}~所示，
在贝叶斯方法的视角下，状态估计即学习从输入 $\boldsymbol{x}$ 到目标 $y$ 的条件预测分布 $p_{\boldsymbol{\theta}}(y|\boldsymbol{x})$，其中 $\boldmath{\theta}$ 是可学习的参数。传统确定性模型的预测分布因退化为零方差的狄拉克函数，即 $p_{\boldsymbol{\theta}}(y|\boldsymbol{x}) = \delta(y - f_{\boldsymbol{\theta}}(\boldsymbol{x}))$， 而表现出对预测结果的盲目置信，对未知工况时存在误判风险。对应地，概率性模型直接输出具备方差的连续概率密度函数，例如高斯分布，即 $p_{\boldsymbol{\theta}}(y|\boldsymbol{x}) = \mathcal{N}(\mu_{\boldsymbol{\theta}}(\boldsymbol{x}), \sigma^2_{\boldsymbol{\theta}}(\boldsymbol{x}))$，从而能够以方差 $\sigma^2_{\boldsymbol{\theta}}$ 对认知与偶然误差进行量化，在面对未见退化模式的样本时表达低置信度，赋予估计结果可信性。
%
本节综述现有的两类具有不确定性量化能力的估计模型，包括计及不确定性的物理驱动估计模型和计及不确定性的数据驱动估计模型。

（1）计及不确定性的物理驱动估计模型 % \label{subsec:physics-based_models}

计及不确定性的物理驱动估计模型通过将退化机制与用于状态估计和参数更新的滤波技术结合实现状态估计和对应不确定性的量化。所使用的滤波器，通常是卡尔曼滤波器（Kalman Filter, KF）或粒子滤波器（Particle Filter, PF），天然属于贝叶斯模型。这一类方法通过对系统状态和噪声分布进行概率推理以量化不确定性\cite{chen2003bayesian}，从而实现对估计结果的动态置信区间估计。

% 2016
% Bressel等人\cite{7386653} propose an extended Kalman filter-based algorithm integrated with the inverse first-order reliability method for SOH and RUL prediction, along with the corresponding confidence intervals.
文献\cite{7386653}中，Bressel~等人提出了一种基于观测器的预测方法，所提出的方法通过扩展卡尔曼滤波器（Extended Kalman Filter, EKF）建模设备的健康状态和退化动态，基于逆一阶可靠度方法（Inverse First-Order Reliability Method, IFORM）外推健康状态轨迹直至触发预设的失效阈值，从而实现对变功率负载工况下的质子交换膜燃料电池的剩余使用寿命预测，并给出90\%置信区间作为对预测结果的不确定性的量化。
% 2021
% Kim等人\cite{9032349} propose a prognostic model utilizing a dynamic particle filter which quantifies the uncertainty associated with the battery's end-of-life and adaptively adjusts the parameters of the degradation model.
文献\cite{9032349}中，Kim~等人结合异常检测技术和基于~PF~的预测方法提出了一种平移核粒子滤波（Shifting Kernel Particle Filter, SKPF）算法，解决了特定循环阶段带有异常退化轨迹的电池的突变特征辨识问题，实现了兼具精度和稳健性的剩余使用寿命预测，所提出方法同样给出预测结果的置信区间。
% 2021
% Haque and Choi \cite{9113648} propose a prognostic model utilizing a sparse kernel ridge regression-assisted particle filter and quantifie abrupt changes in degradation trajectories by accounting for noise uncertainty.
晶体管的可靠应用依赖剩余使用寿命预测，晶体管性能退化具有不确定性、突变性和样本贫化性，这与电池性能退化在数据表现上一致，文献\cite{9113648}中，Haque~和~Choi~提出了一种稀疏核岭回归辅助的粒子滤波（Sparse Kernel Ridge Regression Assisted Particle Filter, SKRR-PF）算法，通过在后验概率密度函数估计的重采样环节引入稀疏核岭回归机制，实现了对晶体管器件导通电阻轨迹突变的识别和剩余使用寿命的预测。
% 2022
% Ahwiadi and Wang \cite{AHWIADI2022110817} propose an enhanced particle filter model based on evolutionary fuzzy predictors to mitigate sample degeneracy and impoverishment issues while achieving high-precision remaining useful life interval estimation.
文献\cite{AHWIADI2022110817}中，Ahwiadi~和~Wang~提出了一种增强型粒子滤波（Enhanced Particle Filter, EPF）框架，通过融合增强型粒子机制和演化模糊预测器（Evolving Fuzzy Predictor, EFP），克服基于传统~PF~方法进行电池系统状态估计和寿命预测时由权值退化和粒子多样性丧失导致的建模不确定性，从而实现准确且鲁棒的电池系统健康状态估计和剩余寿命预测。
% 2023
% Rezaei等人\cite{REZAEI2023107883} develope a fuzzy robust two-stage unscented Kalman filter to estimate cycle capacity and quantify uncertainty in a phased manner using the noise covariance matrix.
文献\cite{REZAEI2023107883}中，Rezaei~等人提出了一种模糊鲁棒两阶段无迹卡尔曼滤波器（Fuzzy Robust Two-Stage Unscented Kalman Filter, FRTSUKF），实现对缺乏先验统计特性的电池模型进行不确定性建模，其在第一阶段基于模糊推理系统剔除测量噪声，初步估计电池模型状态变量，在第二阶段进行不确定性量化，并基于量化结果补偿电池性能退化向状态变量引入的不确定性。
% 2024
% Cui等人\cite{10385053} design a prognostic model based on dual unscented particle filters to address uncertainties in dynamic degradation trajectories and achieve robust prediction.
旋转机械设备维护过程同样存在状态估计和寿命预测需求，且设备性能退化行为与电池系统一致。文献\cite{10385053}中，Cui~等人提出了一种双无迹粒子滤波器（Dual Unscented Particle Filter, DUPF），通过集成双流异构模型并设计全局融合策略，实现跨时间尺度轴承退化特征，并通过显式量化动态贝叶斯最大失效概率提升了预测分布的统计可靠性。
% 2024
% Wang等人\cite{10672556} propose a zonotopic Kalman filter-based method with physics constraints to achieve online capacity interval estimation.
文献\cite{10672556}中，为了实现电池在线容量估计，Wang~等人提出了一种物理约束中心对称多胞形卡尔曼滤波器（Physics-Constrained Zonotopic Kalman Filter, PCZKF），其中所提出的滤波器实现电池模型的参数及其区间估计的联合更新，所提出的物理约束则用于应对电池容量再生挑战，所提出的方法相比传统基于~KF~的方法能够输出更加精确容量预测轨迹和更加收敛的容量置信区间。

（2）计及不确定性的数据驱动估计模型 % \label{subsec:data-driven_models}

计及不确定性的数据驱动估计模型可进一步分为基于统计学习的方法\cite{van2021bayesian, gruber2025sourcesuncertaintysupervisedmachine}和基于深度学习的方法\cite{ABDAR2021243}。基于统计学习的方法依赖显式概率模型进行分析或近似推理。与之相对的，基于深度学习的方法通过利用随机化参数空间或输出分布的统计特性实现不确定性量化。

% 
相关向量回归（Relevance Vector Regression, RVR）和高斯过程回归（Gaussian Process Regression, GPR）是两种典型的具有不确定性量化能力的统计学习方法，被广泛应用在状态估计和寿命预测问题中。
% Wang等人\cite{rvr_rul_9552552} introduce a prognostic method based on relevance vector regression (RVR), while Richardson等人\cite{gpr_soh_8263147} employ Gaussian process regression (GPR) for SOH estimation.
文献\cite{rvr_rul_9552552}中，Wang~等人针对传统~RVR~方法局限于一维退化过程和单变量观测数据的问题，提出了一种基于动态多元相关向量回归（Dynamic Multivariate Relevance Vector Regression, DMRVR）的退化轨迹跟踪和剩余寿命预测框架，所提出方法考虑多元物理环境因素，构建多步回归架构以表征退化动力学，并依据RVR中间结果的矩阵分布基于加速梯度算法优化求解过程。
% 
文献\cite{gpr_soh_8263147}中，Richardson~等人面向恒流充放电工况提出了一种基于高斯过程回归的原位容量数据驱动诊断方法（Gaussian Process Regression for \textit{In Situ} Capacity Estimation, GP-ICE），其在特定的电压窗口内，仅需提取短至十秒的连续恒流工况片段即可实现高精度电池健康状态估计。所提出的方法是一种贝叶斯非参数化回归方法，能够输出预测结果的不确定性作为对数据复杂性的表征。
% 
% Zhang等人\cite{bbb_rul_9738999} develop a Bayesian mixture neural network (BMNN) to capture the distribution of prediction results. 
文献\cite{bbb_rul_9738999}中，Zhang~等人提出了一种贝叶斯混合神经网络（Bayesian Mixture Neural Network, BMNN）以应对因内部反应非线性和高混杂性导致的剩余寿命预测不确定性大的问题。所提出网络引入准格雷姆角场（Quasi-Gramian Angular Field, Q-GAF）进行信号预处理以增强先验分布辨识能力，并融合贝叶斯卷积神经网络（Bayesian Convolutional Neural Network, BCNN）和贝叶斯长短期记忆网络（Bayesian Long Short-Term Memory Network, B-LSTM），由变分贝叶斯推断层输出预测结果的后验概率分布。
%
% Li等人\cite{capsnet_rul_li2024sensor} propose a dropout-based remaining useful life (RUL) prediction framework ，命名为SACN.
为了解决考虑不确定性的预测方法存在的模型合理性和可解释性欠缺的问题，文献\cite{capsnet_rul_li2024sensor}中，Li~等人提出了一种考虑传感器的胶囊神经网络（Sensor-Aware Capsule Neural Network, SACN），这一工作提出了基于胶囊神经元激活水平的蒙特卡洛随机失活机制（Capsule-Activation-Based Monte-Carlo Dropout, CA-MCD），并开展了考虑跨传感器相关特征分布的不确定性归因研究。

% Zhang等人\cite{ensemble2_soh_10102497} combine a multi-model fusion method and a deep ensemble neural network to quantify SOH estimation uncertainty，命名为DeNN.

（3）计及不确定性的健康状态估计研究的局限性

纵观现有关于计及不确定性的~SOH~估计研究，尽管前人工作在模型构建与预测精度上取得了显著进展，但在提供高精度且无偏差的预测结果方面，仍存在若干亟待解决的局限性。
首先，现有的基于物理机理的模型主要关注由噪声引起的不确定性，导致对模型置信度的表征能力不足，进而引发预测结果的不一致。相反，数据驱动模型在不确定性量化方面缺乏细粒度特征与结构适应性，从而阻碍了对退化模式的有效提取。
其次，现有~SOH~估计方法多侧重量化模型自身的不确定性，往往忽略了数据中固有的时空不确定性。这容易导致在退化建模过程中引入无关或错误的数据片段，进而降低最终的预测精度。
最后，受限于近似推断的本质，现有的具有不确定性估计能力的模型往往无法准确表征循环数据的真实分布。即使在估计值严重偏离真实值的情况下，这些模型仍可能表现出较低的不确定性，从而无法有效区分分布内（In Distribution, ID）和分布外（Out of Distribution, OOD）样本。
针对上述现有方法的不足，本文提出了一种面向高可信~SOH~估计方法。通过引入不确定性建模并结合置信度校准机制，同时强化了注意力聚合过程与时空退化依赖关系的特征提取能力，从而有效克服了现有模型在不确定性量化偏差与时空特征提取上的局限。

（4）面向分布漂移的退化建模研究的局限性

回顾现有的面向分布漂移的退化建模工作，虽然各类方法在跨域场景下进行了有益探索，但在复杂多变的实际应用中仍面临诸多挑战。
首先，由于严重依赖目标域数据，传统的预训练-微调与域适应范式在面临动态工况和部分可观测场景时，往往存在较大的部署局限性。虽然基于表征学习和元学习的方法能够实现无需目标域数据的退化建模，但此类方法高度依赖退化先验知识，如需要辅助元数据或退化相关代理任务，这限制了模型的扩展能力与泛化性能。
其次，尽管近期的研究表明基于数据调弈的策略展现出了一定潜力，但其在鲁棒的电池退化建模中仍面临诸多未解难题，主要包括时空退化信息匮乏、分布漂移引发的特征空间统计不确定性，以及多源域间存在的长尾退化分布问题。
为应对上述挑战，本文提出了一种面向~OOD~退化模式泛化的新框架。该框架采用启发式特征重构与概率性特征统计量生成策略，在低复杂度的潜在特征空间中生成连续的退化模式，以克服时空信息匮乏与统计不确定性难题。进一步地，本文设计了一种结合统计特征校正的增强策略，用于解决多源域间长尾分布带来的泛化性能下降问题。

\subsection{幻觉鲁棒的故障诊断研究现状}\label{subsec:fault_diagnosis}

目前，传统深度学习模型（Conventional Deep Learning Models, CDLMs）在智能化旋转机械故障诊断占据主流，但其针对大规模数据时模型容量和泛化能力有限，且仅能输出单一故障标签，难以提供归因分析与维修决策。LLMs~具备强泛化能力和可读诊断报告生成能力，在近年来获得广泛关注。
% 图~\ref{fig:intro_LLM_framework}~展示了一个基于大语言模型开展 PHM 的示例\cite{}。
% 
CDLMs 和 LLMs，而尤其是 LLMs，均存在输出事实性错误，即幻觉，的风险，这在高安全要求的工业场景下是不可接受的。为满足可靠~PHM~的抗幻觉需求，一系列幻觉鲁棒的~PHM~方法已被提出。
本节基于幻觉的诱因与模型输出特性的差异，综述三类已有方法，包括旨在缓解统计偏差的 CDLMs 与 LLMs 方法、具备非期望输入拒绝能力的 CDLMs 与 LLMs 和具有序列级幻觉检测能力的 LLMs。

（1）{缓解统计偏差的 CDLMs 与 LLMs 方法}

% In \cite{Peng_Liu_Du_Gao_Wang_2025}, H. Peng等人 design a cross-dataset diagnostic LLM, termed BearLLM.
文献\cite{Peng_Liu_Du_Gao_Wang_2025}中，Peng~等人提出了一种融合多模态大语言模型的轴承健康管理框架，命名为~BearLLM，仅需加载一套预训练权重即可实现跨数据集高精度故障诊断。该框架首先提出了一种先验知识增强的统一振动信号表征机制；进一步地，预训练一个故障分类网络捕获信号深度表征；最后，基于所标注的大规模振动信号-文本描述对，将所提取的时序特征映射对齐到词嵌入空间，通过大语言模型联合推理。

开放集故障诊断方法往往倾向于开展全局的特征对齐而忽略了各个数据域的私有特征表达，且检测未知类别是依赖基于先验知识的判定阈值。针对此，文献\cite{ZHANG2023109518}中，Zhang~等人提出了一种加权域分离网络（Weighted Domain Separation Network, WDSN），引入二元分类器自适应构建已知故障类别和未知故障类别的划分边界以规避人工预设阈值，同时剥离多个源域的私有特征空间，并基于此提取跨域共享的纯净特征表示。
% In \cite{10382502}, B. Lu等人 design a contrastive embedding-based diagnostic strategy that jointly preserves domain-specific and domain-invariant features, termed MDCC.
文献\cite{10382502}中，Lu~等人面向开放集域泛化问题，提出了一种多域对比编码（Multidomain Contrastive Coding, MDCC）框架。该框架通过设计对比编码任务和对应的损失函数驱动模型学习故障信息，实现在有效保留各数据域专属特异性知识的同时同步实现全局跨域知识的高效泛化。
% In \cite{20242516270368}, B. Liu等人 benchmark LLaMA-series LLMs for undesired input detection and design a cosine distance-based detector using isotropic embeddings, termed LLM-OOD.
LLMs~在模型参数规模、预训练目标和推理范式上与传统模型存在显著差异，基于后者开展的~OOD~检测研究的结论适用性有限。文献\cite{20242516270368}中，Liu~等人针对~LLMs~的~OOD~检测能力开展了实证研究，评估了各类主流~OOD~检测器在零梯度和微调场景下的性能表现，并基于~LLMs~嵌入空间的各向同性特质，提出了一种基于余弦距离的~OOD~检测器，命名为~LLM-OOD。

（3）{具有序列级幻觉检测能力的 LLMs}

传统~LLMs~幻觉检测需要获取多轮模型响应或依赖庞大外部数据库而难以部署在实际环境中的问题，文献\cite{20251218102787}中，Sriramanan~等人提出了一种利用~LLMs~隐状态、注意力映射图集输出预测概率分布实现在黑盒和白盒的双重设定下，仅使用单次模型响应即可实现幻觉识别的方法，命名为~LLM-Check。所提出方法可以拓展至具备真实参考文本的特定应用场景，如检索增强生成范式下的幻觉检测问题。
% 
不确定性量化技术是实现~LLMs~幻觉检测的一种具有应用前景的途径，但对~LLMs~响应开展不确定性评估存在精确性不足的问题，这主要来源于语言的冗余性，即自回归文本生成过程中并非所有 token 等价地承载语义信息。针对此，文献\cite{20243917091282}中，Duan~等人提出了一种高相关性注意力转移（Shifting Attention to more Relevant, SAR）机制，通过对 token 级别和句子级别的相关性进行重赋权，该方法实现了更精确的不确定性量化，并将量化结果应用在~LLMs~幻觉检测问题上。
% 
文献\cite{20240715556058}中，Zhang~等人提出了一种无参考的、基于不确定性量化的幻觉检测方法，其核心是一种认知聚焦机制。具体地，这种机制在幻觉检测过程中首先聚焦于包含关键语义信息的核心 token；其次排除历史上下文中潜在的不可靠 token 从而阻断其可能诱发的级联幻觉效应；最后分析 token 类型和 token 频次等固有属性对生成结果可靠性的影响。
% 
事实核查是实现~LLMs~幻觉检测的合理方法，但传统的基于事实核查的方法一方面难以被应用在闭源系统中，另一方面高度依赖外部数据库且通常需要通过结构复杂的独立模块进行交互。为了解决这些问题，文献\cite{selfcheckgpt}中，Manakul~等人提出一种基于简单采样的无资源事实核查机制，命名为~SelfCheckGPT。该方法基于语言模型确切掌握概念内在知识的启发性假设，依据对~LLMs~多次采样结果的一致性评估进行模型幻觉检测。
% 
传统的基于后处理范式的~LLMs~幻觉检测方法因为独立于模型的推理过程，存在计算开销大的问题。为了克服这一局限性，文献\cite{su-etal-2024-unsupervised}中，Su~等人提出了一种无需人工标注且直接利用~LLMs~内部隐状态的实时幻觉检测机制，命名为MIND。这一工作同时构建了名为~HELM~的基准评估体系，不仅包含高度多样化的模型输出文本，同时记录了模型在推理过程时内部隐状态的演化轨迹。
% 
为了实现模型无关幻觉检测，文献\cite{10.1007/978-3-031-86623-4_13}中，Quevedo~等人提出了一种基于监督学习的幻觉检测方法，命名为~SHalluDetect，旨在基于大型~LLMs~的无幻觉或少幻觉推理结果评估小型~LLMs~输出的幻觉水平。该方法通过部署两个浅层分类器，以基于上述两种不同规模~LLMs~推理结果中的 token 分布构造的四个数值特征为输入，实现跨模型架构的高精度幻觉检测。
% 
为了评估~LLMs~在何种内容和何种程度上容易产生幻觉，文献\cite{li-etal-2023-halueval}中，Li~等人构建了一个基准测试，提出了一种基于~ChatGPT~的先采样后过滤的两阶段数据生成框架，通过人工操作对模型响应中潜在的幻觉进行审查和标注，证明通过引入外部事实知识或在模型内部增加显示逻辑推理步骤可以提升模型自身的幻觉识别能力。

（4）幻觉鲁棒的故障诊断研究的局限性

尽管现有方法，尤其是基于~LLMs~的方法，在故障诊断领域展现出了巨大的应用潜力，但现有研究在提升模型应对幻觉的鲁棒性方面仍存在若干悬而未决的共性问题。
首先，现有模型往往忽略了由跨数据集差异与类别不平衡共同引发的复合统计偏差，容易将领域特有特征误判为随机噪声。而传统的基于简单扰动的数据增强方法由于生成了高度相关的样本，无法有效丰富少数类样本的多样性，反而进一步加剧了这一问题。
其次，当面对与训练数据共享高层语义的非预期输入时，现有诊断模型往往会低估其预测不确定性，进而导致诊断精度的显著下降。
最后，由于生成文本中存在的语义冗余以及模型自身的过度自信，现有的幻觉检测方法通常难以有效识别并拦截模型生成的幻觉内容。
为了应对这些挑战，本文提出了一种缓解与检测大语言模型幻觉的新策略。该策略通过高效、多样化地扩充训练数据以消除统计偏差，并量化生成结果的句子级不确定性，克服了诊断模型面对非预期输入时的过度自信问题，提升了故障诊断结果的准确度、幻觉鲁棒性和可靠性。

\section{研究目标与关键研究挑战}\label{sec:Intro_research_objectives_and_challenges}

本文聚焦动力电池与旋转机械两类核心易损部件，围绕健康状态估计、退化建模与故障诊断三项关键任务开展研究，旨在构建适用于复杂实际工况的可靠~IPHM~方法体系。
这里所指的可靠性，不仅是模型在测试集上的预测精度，同时强调模型在分布漂移、信息不完备与未知输入扰动等条件下，仍具备可解释、可置信与可用的推断能力。
% 
基于这一总体目标，本文将相关研究问题归纳为可信、可泛化与抗幻觉三个相互关联的维度，并将无偏不确定性量化确立为贯穿全文的主线。
本文的关键研究挑战如下：

（1）\textbf{锂离子电池健康状态估计中的可信挑战}

现有锂离子电池健康状态估计方法大多侧重提升点估计精度，对不确定性的表征仍较为粗略，量化过程缺乏完整性，模型预测性能与不确定性量化质量之间的内在关系亦未得到充分揭示，因此难以建立预测分布与真实数据分布之间的无偏对应关系。在实际工业场景中，电池循环信号普遍受到噪声扰动、负载波动以及多传感器耦合关系变化的影响。若在相关性表征学习过程中忽视模型自身不确定性，高不确定性但低贡献的局部片段容易被错误聚合，进而削弱退化动态的提取质量。同时，主导空间通道的动态变化与有效时间特征的稀疏分布，进一步加剧时空依赖建模中的表征模糊性。更为关键的是，即使模型能够输出不确定性，若预测分布与真实分布之间存在系统失配，仍会产生过置信或欠置信现象，从而降低评估结果的可信性，并削弱模型区分分布内样本与分布外样本的能力。因此，如何在复杂时空相关建模过程中实现细粒度不确定性量化，并通过校准机制使其无偏反映真实数据分布，构成电池健康状态可信估计的首要挑战。

（2）\textbf{锂离子电池退化建模中的分布外稳健泛化挑战}

与分类任务不同，锂离子电池退化建模面向健康状态与剩余使用寿命等连续退化目标，其退化演化具有内在有序性。当训练域与测试域之间存在材料体系、充放电协议、环境温度或运行载荷差异时，模型学习得到的特征与退化状态之间的映射关系容易失效，进而导致分布外场景下的预测性能显著下降。现有迁移学习与泛化方法通常难以同时应对三个层面的关键困难。其一，受非侵入式测量条件限制，原始循环信号在时间维与传感维均存在明显的信息稀疏性，直接从此类数据中提取退化特征，往往导致特征流形结构复杂、连续性不足，难以满足鲁棒表征对低冗余与连续几何结构的要求。其二，现有研究尚未明确刻画分布外退化模式与特征统计量不确定性之间的关系，使得域偏移难以被有效量化与利用。其三，电池退化样本天然呈现多域长尾分布特征，早期样本占优而后期样本稀缺，模型因此容易形成偏向头部退化阶段的表征偏置，进一步削弱对未见退化轨迹的泛化能力。因此，如何为退化特征统计量的不确定性量化提供可靠的表征基础，并在仅依赖源域数据的条件下实现对未知退化分布的稳健建模，构成可泛化退化建模方向上的核心挑战。

（3）\textbf{大语言模型驱动的旋转机械故障诊断中的抗幻觉挑战}

大语言模型具备较强的跨模态理解与生成能力，为复杂故障诊断提供新的技术路径，但其生成机制同时引入事实性错误风险，因此如何实现面向故障诊断场景的可靠幻觉检测与抑制，成为当前智能运维研究中的重要问题。一方面，训练数据中的类别不平衡与统计偏置容易使模型在推理过程中偏向高频状态，从而诱发对少数类故障的误判与幻觉生成。另一方面，在部署阶段，模型可能面对未见故障模式、跨设备输入或与任务无关的提示信息。若缺乏对自身认知边界的有效感知，模型便可能生成语言形式流畅但脱离事实依据的诊断结论。进一步来看，现有幻觉检测通常依赖 token 级后验概率，而故障诊断中的幻觉更常表现为序列层面的局部事实错误。同时，大量语法正确但语义空泛的响应片段会掩盖真正的幻觉内容，使 token 级不确定性与完整响应真实性之间产生显著的粒度失配。因此，如何揭示统计偏置、未见模式与幻觉生成之间的内在联系，建立从模型置信度到事实真值的一致性表征，并实现消除粒度偏差的序列级无偏不确定性量化，是大语言模型抗幻觉故障诊断的关键挑战。

上述三个研究方向虽然面向不同研究对象与任务形式，但均指向同一根本问题，即在复杂分布漂移条件下，如何对模型输出及其内部表征中的不确定性进行合理量化，并使这种量化结果尽可能摆脱数据偏置、粒度失配与场景变化带来的系统性偏差。基于这一认识，本文围绕上述研究目标与关键挑战进一步设计具体研究内容与技术路线，从而形成面向复杂实际工况的可靠 IPHM 方法框架。

\section{研究内容与技术路线}\label{sec:Intro_research_content_and_approach}

面向电动汽车关键部件在复杂实际工况下面临的性能退化与突发故障风险，本文以无偏不确定性量化为理论核心，围绕关键部件的可靠 PHM 开展系统研究。
动力电池与旋转机械是电动汽车动力系统中两类核心且易损的部件，其运行状态直接影响整车全生命周期的安全性与可靠性。
% 
针对上述对象开展健康状态评估、退化规律建模与故障识别，构成关键部件智能运维的核心研究环节，也对算法在复杂场景下的适用性、稳定性与可靠性提出更高要求。
% 
具体地，可靠的 PHM 需要在三个维度实现协同提升。
其一，为评估结果的可信性，即模型需要准确量化自身认知边界并给出可信预测。
其二，为面向未知工况的泛化能力，即模型在分布外退化模式下仍保持稳定鲁棒。
其三，为复杂诊断任务中的抗幻觉能力，即引入大语言模型后，模型能够有效识别并抑制虚假生成内容。
% 
为实现上述多维可靠性目标，本文从不确定性量化与无偏性约束两个层面构建统一的理论主线，并将其贯穿于各项研究工作。
% 
针对锂离子电池健康状态估计任务，研究重点在于模型预测不确定性的量化，并通过不确定性校准使预测不确定性能够无偏反映数据分布，从而提升健康状态评估的可信性。
% 
针对锂离子电池退化建模任务，研究重点转向退化特征统计量不确定性的量化，通过建模已知退化分布与未见退化分布之间的差异，增强模型在 OOD 场景下的鲁棒性。为保证该类不确定性量化具有可靠的表征基础，进一步设计启发式信号重构方法，使所提取退化特征的流形结构具有潜在且连续的拓扑分布，从而为特征统计量的无偏量化提供支撑。
% 
针对旋转机械故障诊断任务，研究重点在于复杂生成响应的不确定性量化，并提出序列级不确定性表征方法，将 token 级不确定性与完整响应级不确定性进行对齐，以消除由粒度失配引起的偏差，进而实现面向故障诊断大模型的无偏幻觉检测。
% 
三项研究分别面向预测输出、退化表征与生成响应三个层面的不确定性问题展开，在研究对象上相互补充，在方法逻辑上层层递进，共同推动 IPHM 技术向高可靠方向发展。

基于此研究路线，本文的具体研究内容分为以下三个部分：

（1）\textbf{基于层次化随机时空 Transformer 的锂离子电池健康状态可信估计方法}
该方法旨在解决工业应用中锂离子电池健康状态估计面临的诸多挑战，包括因忽略模型不确定性导致的特征聚合偏差，以及预测分布与真实分布错位引发的置信度偏差问题。
首先，设计基于 Gumbel-Softmax 重参数化（Gumbel-Softmax Reparameterization, GSR）的随机自注意力机制，实现对模型认知不确定性的初步量化。
其次，提出层次化随机时空 Transformer 架构，通过融合不确定性感知的逐时间步空洞卷积与聚类随机自注意力机制，有效应对主导空间通道的偏移与有用时间特征的稀疏性。
最后，从理论层面通过偏差与方差分解对置信度偏差问题进行剖析，并设计一种原则性的不确定性校准策略。该策略使模型的预测不确定性能够无偏地反映数据的真实分布，从而显著提升模型区分分布外样本的能力与健康评估的整体可信度。

（2）\textbf{基于拓扑约束下不确定性驱动特征调弈的锂离子电池分布外鲁棒退化建模方法}
该方法旨在克服车载电池管理系统在复杂动态工况与材料差异下进行退化建模时所面临的难题，具体包括时空信息匮乏、域偏移导致的特征统计量不确定，以及多源不平衡退化分布带来的早期偏差。
首先，提出一种基于流形收缩的退化表征提取策略。该策略采用降低内在维度且保持退化连续性的启发式信号重构方法，并结合 Mamba 编码器，使提取的退化特征流形具有潜在且连续的拓扑分布，从而为特征统计量的无偏量化提供坚实的表征基础。
其次，提出不确定性驱动的特征调弈方法，利用多元高斯分布量化特征统计量的不确定性，借此建模已知退化分布与未知退化分布间的差异，进而缓解分布外场景下的域偏移问题。
最后，开发长尾感知的自适应重平衡细化策略，利用高斯核平滑引入邻域退化信息以放大尾部样本表征，有效克服早期退化预测的偏差，实现高鲁棒的剩余使用寿命预测。

（3）\textbf{基于扩散增强可信多模态大语言模型的旋转机械抗幻觉故障诊断方法}
该方法聚焦于大语言模型在旋转机械故障诊断应用中存在的固有缺陷，旨在解决因模型权重统计偏差或非相关提示引导而产生的幻觉现象，以及形式正确但语义空洞的响应对局部幻觉的掩蔽问题。
首先，设计由控制网络引导的跨数据集条件扩散模型，通过注入可控的语义引导并合成多样化样本，从源头上缓解多模态数据的统计偏差。
其次，提出一种序列级不确定性量化机制。该机制通过点双列相关分析与显著性检验定位关键 token，并设计数据驱动的关键 token 不确定性重加权策略，充分对齐并消除 token 级别与完整响应级别不确定性之间的粒度偏差，从而实现无偏的幻觉检测。
最后，构建一个统一的抗幻觉运维框架，将扩散生成与无偏幻觉检测无缝集成，并制定针对域无关提示的拒绝机制，在保障诊断性能的同时确保大语言模型在安全关键场景下的应用可靠性。
