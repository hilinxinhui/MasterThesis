\chapter{基于扩散增强可信多模态大语言模型的旋转机械抗幻觉故障诊断方法}\label{chap:DA-TMLLM}

\section{引言}\label{sec:DA-TMLLM_introduction}

电动汽车动力系统的传动子系统等旋转机械的稳定运行对于保障车辆安全至关重要。滚动轴承作为此类机械的核心部件，承担减小摩擦和承载负荷等多种关键功能，且通常运行于恶劣环境中 \cite{PU, JUST}。因此，对轴承运行数据进行持续监测并快速诊断故障特征，对于保障安全至关重要。得益于传感器技术、信号采集技术和深度学习技术的发展，数据驱动的旋转机械故障诊断方法已成为主流，其能够基于历史数据自动推导出诊断结果 \cite{10577994, HIT}。
近年来，LLMs 凭借其优异的性能，在故障诊断领域逐步替代 CDLMs \cite{Peng_Liu_Du_Gao_Wang_2025}，能够在多种工况下提供准确的诊断结果与维护建议。
尽管取得了上述进展，LLMs 遭遇“幻觉”而失效的情况仍是一个突出的问题。幻觉指 LLMs 生成的长文本回答中包含事实错误的现象，在故障诊断语境下，具体表现为给出不正确的诊断结果或不恰当的维护建议 \cite{li-etal-2023-halueval, azaria-mitchell-2023-internal}。实际部署时，真实的故障标签通常不可获取，因此难以判定 LLMs 生成的回复是否包含事实错误。LLMs 的幻觉导致的错误信息可能导致严重的经济损失，甚至危及人员安全。LLMs 的幻觉检测非常重要，但当前相关工作在消除事实错误并检测 LLMs 生成回答中的幻觉方面，仍面临以下挑战：

（1）统计偏差引发幻觉。工业滚动轴承在服役周期的绝大多数时间内保持健康状态，导致轴承故障样本集合存在固有的类别不平衡，呈现长尾分布特性。不平衡的训练集使得模型学习到的决策边界发生偏移，并在 LLMs 中引入统计偏差，促使其倾向于预测多数类。因此，样本量丰富的健康类别及发生频率较高的故障模式主导了梯度更新过程，成为偏差的主要来源。在推理阶段，若输入样本属于少数类别，也即尾部类别，模型依然倾向于预测更常见的类别，从而导致罕见但关乎安全的故障被漏报。

（2）非预期输入引发幻觉。部署后，LLMs 可能会遭遇非预期输入，通常是来自未见过的旋转机械类型的传感器信号或其他与领域无关的数据源。面对此类输入时，LLMs 通常会产生幻觉，生成看似流畅实则编造的回答，而非拒绝响应查询。究其原因，当前 LLMs 缺乏对其认知边界的明确认知，无法判断何时应保持沉默。此外，若非预期输入与训练数据存在较强的语义重叠，例如来自新型轴承的信号与已知数据高度相似，LLMs 可能会表现出过度的自信，为该输入分配较高的后验概率。

（3）序列级幻觉具有隐蔽性。对于回答序列中的每个词元，其后验概率主要受模型内部先验、语料库词频以及局部句法模式的影响，而非取决于该词元与训练数据的匹配程度。因此，当检测器仅基于词元级别的后验概率进行判断时，幻觉现象有较大概率逃避检测。此外，在评估整个回答序列时，诸多形式正确但缺乏实质语义的短语，如“\textit{从提供的列表中最合适的轴承状态类型是（the most suitable bearing state type from the provided list is）}”。这样的语言冗余的存在将稀释幻觉内容。最终，即使回答序列中存在幻觉，幻觉量化指标仍可能输出正常值。

针对上述问题，本文提出一种扩散增强的可信多模态大型语言模型（Diffusion-Augmented Trustworthy Multimodal Large Language Model, DA-TMLLM）。
首先，设计一个 ControlNet 引导的跨数据集条件扩散模型（ControlNet-Guided Cross-Dataset Conditional Diffusion Model, CGCDM），缓解由统计偏差引发的幻觉。
其次，通过在多个轴承数据集的文本-信号对上对预训练 LLMs 进行微调，开发可信多模态大型语言模型（Trustworthy Multimodal Large Language Model, TMLLM）。基于所构建的 TMLLM，识别出一个关键词元，设计一种关键词元驱动的不确定性重加权策略，以有效检测序列级幻觉。
最后，设计一种用于检测和缓解幻觉的抗幻觉机制，将 CGCDM 和 TMLLM 整合为统一框架，阐述拒绝非预期输入的流程。DA-TMLLM 的主要贡献如下：

（1）设计 ControlNet 引导的跨数据集条件扩散模型。通过注入可控的语义引导，在保持语义一致性的同时，提升生成样本的多样性。

（2）通过锁定关键词元并设计关键词元驱动的不确定性重加权策略，构建可信多模态大型语言模型，提升 LLMs 的幻觉检测性能。

（3）设计抗幻觉机制，将 CGCDM 与 TMLLM 整合入统一的工作流，并结合非预期输入拒绝方法，在进一步提升检测性能的同时，缓解 LLMs 的幻觉。

（4）在 11 个数据集上的评估结果表明，DA-TMLLM 在 8 项指标上的表现始终优于 11 种对比模型。

\subsection{ControlNet 引导的跨数据集条件扩散模型}

为减小数据集间差异，
提出一种跨数据集标准化方法，通过固定时长分段、长度对齐和尺度归一化对旋转机械运行时信号进行预处理。
随后，提出基于条件扩散模型（Conditional Diffusion Model, CDM）的方法，用于合成和补充少数故障模式上的振动信号。
最后，向 CDM 中引入 ControlNet，在反向扩散阶段注入可控的语义引导信息。

（1）跨数据集信号预处理标准化方法

鉴于不同轴承数据集在传感器采样率、采集条件和轴承类型方面存在较大差异，所采集信号在特征故障频率与幅值上表现出明显异质性。若将这些异构信号直接输入扩散模型，模型易将跨数据集差异视为随机噪声，从而削弱对有效联合分布的学习能力。为此，提出一种跨数据集标准化方法。

在传统数据预处理中，原始时域信号通常被划分为包含相同采样点数的固定长度样本。由于振动信号的采样率并不一致，固定长度分段会导致特征故障频率发生偏移，并进一步放大跨数据集差异。为缓解该问题，采用固定时长分段策略。设单段持续时间为 1 s，则第 $m$ 段信号可表示为
\begin{equation}
\bm{x}_o^{(m)}=
\bigl[
x_r[m\nu], x_r[m\nu+1], \dots, x_r[(m+1)\nu-1]
\bigr]^{\top},
\end{equation}
其中，$\nu$ 表示数据集对应的采样率，$\bm{x}_r$ 表示原始信号。

% \begin{equation}
% 	\label{eq:scalenorm}
% 	\operatorname{SNorm}(\bm{x})=g\,\frac{\bm{x}}{\lVert \bm{x}\rVert_{2}},
% \end{equation}
% where $g=\rho\sqrt{L_{f}}$, and $\rho$ is a predefined scaling factor.

在固定时长分段之后，为使不同数据集得到的信号片段能够作为统一的网络输入，还需进一步完成长度与幅值对齐。具体而言，依次施加离散傅里叶余弦变换（Discrete Fourier Cosine Transform, DFCT）与尺度归一化（Scale Normalization, SNorm），这两个操作分别定义为 $\operatorname{DFCT}(\cdot)$ 和 $\operatorname{SNorm}(\cdot)$。$\operatorname{DFCT}(\cdot)$ 将时域信号片段变换到频域，并仅保留前 $L_f$ 个谱系数，其中 $L_f$ 为预设长度。对于谱系数长度不足 $L_f$ 的信号，采用零填充；对于谱系数长度超过 $L_f$ 的信号，则进行截断。预处理过程可表示为
% 
\begin{equation}
	\label{eq:dcn}
	\bm{x}_v =
	\begin{cases}
		\operatorname{SNorm}\bigl(\operatorname{DFCT}(\bm{x}_o)[0:L_{f} - 1]\bigr),      & \ell \ge L_{f},\\[4pt]
		\operatorname{SNorm}\bigl(\operatorname{Concat}(\operatorname{DFCT}(\bm{x}_o),\bm{0}_{\,L_{f}-\ell})\bigr), & \ell < L_{f},
	\end{cases}
\end{equation}
% 
其中，$\ell$ 表示实际谱系数长度。$\operatorname{SNorm}(\cdot)$ 定义为
% 
\begin{equation}
\label{eq:scalenorm}
\operatorname{SNorm}(\bm{x})=
g\frac{\bm{x}}{\lVert \bm{x}\rVert{2} + \varepsilon},
\end{equation}
% 
其中，$g=\rho\sqrt{L_f}$，$\rho$ 表示预设缩放系数， $\varepsilon$ 为足够小的正常数，避免上式分母为 0。

（2）ControlNet 引导的条件扩散生成

为同时提升训练稳定性与样本多样性，采用基于扩散模型的数据增强框架。
该框架由前向扩散阶段与反向扩散阶段组成。
为增强生成过程中语义信息的一致性，设计 ControlNet，在每一个反向扩散步中将高层语义特征作为外部条件注入潜在表示。

在前向扩散阶段，通过长度为 $S$ 的马尔可夫链对预处理后的 $\bm{x}_v$ 逐步添加噪声，具体地，在每一个步骤 $s \in {1,\dots ,S}$ 中均增量式加入各向同性 Gaussian 噪声。第 $s$ 步的加噪过程表示为
% 
\begin{equation}
q(\bm{x}_v^{(s)} \mid \bm{x}_v^{(s-1)}) = \mathcal{N}\!\bigl(\sqrt{\alpha^{(s)}}\,\bm{x}_v^{(s-1)},\, (1-\alpha^{(s)})\bm{I}\bigr),
\end{equation}
% 
其中，$\bm{I}$ 表示单位矩阵，$\alpha^{(s)}$ 表示噪声缩放因子，且 $\alpha^{(s)} = 1 - \beta^{(s)}$，$\beta^{(s)}$ 由预定义的方差调度器 $\zeta$ 给定。
经过 $S$ 个步骤后，$\bm{x}_v^{(S)}$ 逐渐逼近纯 Gaussian 样本分布。
利用闭式累积形式，完整的前向扩散过程可表示为
% 
\begin{equation}
q(\bm{x}_v^{(S)} \mid \bm{x}_v^{(0)}) = \mathcal{N}\!\bigl(\sqrt{\hat{\alpha}^{(S)}}\,\bm{x}_v^{(0)},\,(1-\hat{\alpha}^{(S)})\bm{I}\bigr),
\end{equation}
% 
其中，$\hat{\alpha}^{(S)} = \prod_{s=1}^{S} {\alpha}^{(s)}$。$\bm{x}_v^{(S)}$ 作为后续反向扩散阶段的输入。

在反向扩散阶段，初始隐变量 $\bm{x}_v^{(S)} \sim \mathcal{N}(0,\bm{I})$ 在扩散步骤感知的去噪器 $\epsilon_{\bm{\theta}_\mathrm{d}}(\cdot)$ 的作用下迭代优化。
在每一个扩散步骤 $s$ 中，基于 UNet 的噪声预测网络接收三元组 $(\bar{\bm{x}}_v^{(s)},\,s,\,\bm{c})$，估计所加入的 Gaussian 扰动，并计算后验均值
% 
\begin{equation}
	\mu_{{\theta}_\mathrm{d}} \bigl(\bar{\bm{x}}_v^{(s)},\,s,\,\bm{c}\bigr)
	= \frac{\bar{\bm{x}}_v^{(s)} - \sigma^{(s)} \epsilon_{{\theta}_\mathrm{d}} \bigl(\bar{\bm{x}}_v^{(s)}, s, \bm{c}\bigr)}
	{\sqrt{\bar{\alpha}^{(s)}}},
\end{equation}
% 
其中，$\bar{\alpha}^{(s)} = 1-\bar{\beta}^{(s)}$，$\sigma^{(s)} = \sqrt{1-\bar{\alpha}^{(s)}}$，$\bm{c}$ 表示外部条件向量。
$\bm{c}$ 通过向噪声预测网络注入辅助信息，在整个反向扩散阶段提供可控引导。随后，从分布 $\mathcal{N} \bigl(\mu_{{\theta}_\mathrm{d}} \bigl(\bar{\bm{x}}_v^{(s)},\,s,\,\bm{c}\bigr), \bigl(\sigma^{(s)}\bigr)^{2}\bm{I}\bigr)$ 中采样得到新的隐变量 $\bar{\bm{x}}_v^{(s-1)}$。

对所有 $s = S,\ldots,1$ 重复执行上述随机重构过程，能够逐步抑制所注入的噪声，最终使输出 $\bar{\bm{x}}_v^{(0)}$ 收敛至所学习的数据分布，从而生成真实的合成样本。

为在反向扩散阶段注入可控语义引导，设计 ControlNet，将辅助知识嵌入为低维外部条件向量 $\bm{c}$。
% 
具体地，首先在带标签振动信号上对所设计的 ControlNet 进行预训练，使其能够将每个原始信号映射为具有类别判别性的潜在表示。
% 
为构建对跨数据集差异具有鲁棒性的外部条件向量，采用三元组输入 $\bm{H}_{v} = \bigl[\bm{x}_{v},\;\tilde{\bm{x}}_{v},\;\bm{x}_{vr}\bigr]\in\mathbb{R}^{3 \times L_{f}}$，其中，$\bm{x}_{v}$ 表示生成的查询样本，$\tilde{\bm{x}}{v}$ 表示无故障信号样本，$\bm{x}_{vr}=\bm{x}_{v}-\tilde{\bm{x}}_{v}$ 表示用于放大故障相关偏差的残差信号。
% 
如图~\ref{fig_controlnet} 所示，ControlNet 的编码器由 3 个大核卷积分支构成，其后连接 3 个跨通道注意力模块（Cross-Channel Attention Block, CCAB），用于实现跨尺度特征融合。
随后，将编码器输出输入两层全连接分类器，并采用类别交叉熵损失对 ControlNet 进行训练。在扩散模型训练阶段，ControlNet 的编码器权重和分类器权重分别记为 $\bm{\theta}_\mathrm{enc}$ 和 $\bm{\theta}_\mathrm{cla}$，并保持冻结。

在第 $s$ 个反向扩散步骤中，查询向量 $\bm{Q}^{(s)}$、键向量 $\bm{K}^{(s)}$ 和值向量 $\bm{V}^{(s)}$ 的计算方式为

\begin{equation}
	\bm{Q}^{(s)} = \bm{W}_{Q}\bar{\bm{x}}_{v}^{(s)},
	\bm{K}^{(s)} = \bm{W}_{K}{\tau}{(\bm{c})},
	\bm{V}^{(s)} = \bm{W}_{V}{\tau}{(\bm{c})},
\end{equation}
% 
其中，$\bm{W}_{Q}$、$\bm{W}_{K}$ 和 $\bm{W}_{V}$ 表示可学习的线性投影矩阵，${\tau}{(\cdot)}$ 表示一个编码器，用于将 $\bm{c}$ 和 $\bar{\bm{x}}_{v}^{(s)}$ 投影到相同维度的特征空间中。
随后，注意力输出计算为

\begin{equation}

\end{equation}

其中，$d_k$ 表示键向量的维度。
对于 MHA，注意力计算过程在多个注意力头上独立执行。
随后，将各个注意力头的输出进行拼接，并通过线性变换投影得到最终输出，其形式表示为
\begin{equation}
	\operatorname{MHA}\bigl(\bar{\bm{x}}_{v}^{(s)},\bm{c}\bigr)=
	\bm{W}_{o}\bigl[\bm{A}^{(s)}_{1};\dots;\bm{A}^{(s)}_{N_h}\bigr],
\end{equation}
% 
其中，$\bm{W}_{o}$ 表示输出投影矩阵，$N_h$ 表示注意力头的数量。通过上述方式引入外部条件向量 $\bm{c}$，能够在反向扩散过程的每一个步骤中注入语义上下文信息。

所设计的 ControlNet 引导跨数据集条件扩散模型的损失函数可表述为
% 
\begin{equation}
	L^{(s)}=\mathbb{E}_{\substack{s\sim[1,S],\bm{x}_v^{(s)},\epsilon^{(s)}}}
	\Bigl[\operatorname{Huber}\bigl(\lVert r^{(s)} \bigr)\rVert\bigr)\Bigr],
\end{equation}
where $r^{(s)} = \bigl\lVert\epsilon^{(s)} - \epsilon_{{\theta}_\mathrm{d}}\!\bigl(\bar{\bm{x}}_{v}^{(s)},s,\bm{c}\bigr)\bigr\rVert$, and $\operatorname{Huber}(\cdot)$ is Huber loss.
% 
其中，$r^{(s)} = \bigl\lVert\epsilon^{(s)} - \epsilon_{{\theta}_\mathrm{d}}\!\bigl(\bar{\bm{x}}_{v}^{(s)},s,\bm{c}\bigr)\bigr\rVert$，$\operatorname{Huber}(\cdot)$ 表示 Huber 损失。
Huber 损失能够减弱较大逐点残差的影响，因此对于信号样本之间可能存在的相位偏移具有更强鲁棒性，其形式为
% 
\begin{equation}
	\operatorname{Huber}\bigl(r^{(s)}\bigr)=
	\begin{cases}
		\dfrac{1}{2}\, \bigl(r^{(s)}\bigr)^{2},          & r^{(s)} < \lambda ,\\[4pt]
		\lambda\,\bigl(r^{(s)}-\dfrac{\lambda}{2}\bigr),   & \text{otherwise},
	\end{cases}
	\label{eq:huber}
\end{equation}
其中，$\lambda$ 表示 $\operatorname{Huber}(\cdot)$ 函数的转折阈值。

\subsection{可信多模态大语言模型}\label{subsec:DA-TMLLM_TMLLM}

%符号后续还需要统一细化
TMLLM 旨在跨多个数据集提供可信的诊断结果与分析。在 TMLLM 中，通过在文本-信号对上对预训练大语言模型进行微调，构建多模态大语言模型。为获得可信回答，通过计算点二列相关系数并进行显著性检验，识别能量出现异常下降的关键 token。进一步地，设计关键 token 驱动的不确定性重加权策略，以量化大语言模型回答中的不确定性。

（1）基于文本-信号对的大语言模型微调
Since recent pre-trained LLMs are optimized for discrete textual tokens and cannot natively process vibration signals, we introduce a dual-branch tokenizer-encoder block, which converts vibration signal samples and corresponding textual descriptions into signal-specific and token-specific features.
The modality-specific features are subsequently aligned by a text-signal alignment block, and the whole model is fine-tuned to maximize cross-modal alignment.

In the developed dual-branch tokenizer-encoder block, the encoder shares the weight $\bm{\theta}_\mathrm{enc}$ with ControlNet.
Let $\bm{x}_{g}$ be a synthesized sample, the input of the encoder branch can be formulated as
$\bm{H}_{g} = \bigl[\bm{x}_{g},\;\tilde{\bm{x}}_{g},\;\bm{x}_{gr}\bigr] \in \mathbb{R}^{3 \times L_{f}}$, where $\bm{x}_{g}$ is a query sample, $\tilde{\bm{x}}_{g}$ is a fault-free signal sample, and $\bm{x}_{gr}=\bm{x}_{g}-\tilde{\bm{x}}_{g}$ is a residual sample that amplifies fault-related deviations.
This encoder produces signal-specific features for each input.
Let $\bm{D}=\{k_{1},\dots,k_{N_c}\}$ be a collection of short textual fault descriptions, one description per fault category, where $N_c$ denotes the number of categories.
Furthermore, the multimodal input to the TMLLM is obtained by fusing $\bm{H}_{g}$ and $\bm{T}$, and is denoted as $\bm{M}_{g}=\{\bm{H}_{g},\bm{T}\}$.
In the tokenizer branch, we feed $\bm{D}$ through the LLM's native tokenizer $\operatorname{Tok}(\cdot)$ and embedding layer $\operatorname{Emb}(\cdot)$ to obtain a sequence of token-specific features, denoted as $\bm{E}=\operatorname{Emb}(\operatorname{Tok}(\bm{D})) \in \mathbb{R}^{\gamma\times N_c\times h}$, where $h$ is the hidden size of $\operatorname{Emb}(\cdot)$, $\gamma$ is the maximum length of token-specific features.
As illustrated in Fig. \ref{fig_framework}, the weights of the dual-branch tokenizer-encoder block are kept frozen throughout LLM fine-tuning.

To bridge the gap between the signal-specific/token-specific features and LLMs, we develop a trainable text-signal alignment block that fuses signal-specific and token-specific features into language tokens suitable for input to LLMs.
The alignment block is implemented as a three-layer linear multilayer perceptron.
We initialize this alignment module using a combination of the ControlNet's prior weights and token-specific features.
In particular, the weights of the first two linear layers, denoted $\bm{\theta}_\mathrm{l_{1}}$ and $\bm{\theta}_\mathrm{l_{2}}$, are initialized with the weights $\bm{\theta}_\mathrm{cla}$ of the ControlNet's pre-trained classifier.
Furthermore, we flatten the last two dimensions of $\bm{E}$ and use the resulting matrix as the weight of $l_{3}$, formulated
$\boldsymbol{\theta}_{l_{3}} =\operatorname{reshape}\bigl(\bm{E},(\gamma,N_c h)\bigr)
\in\mathbb{R}^{\gamma\times N_c h}$.
In this way, the $i$-th row of $\boldsymbol{\theta}_{l_{3}}$ encodes the token-level embedding of the $i$-th fault description, providing a semantic projection from the signal-specific features to the language tokens.
Finally, the text-signal alignment block embeds the language tokens into a predefined prompt template and outputs $\bm{x}_{pt}$ as the LLM's input.
The weight of text-signal alignment block is denoted as $\boldsymbol{\theta}_\mathrm{ts}=\{\bm{\theta}_\mathrm{l_{1}}, \bm{\theta}_\mathrm{l_{2}}, \bm{\theta}_\mathrm{l_{3}}\}$, and the weight of the pre-trained LLM is denoted as $\boldsymbol{\theta}_\mathrm{llm}$.
To fine-tune the weights of text-signal alignment block and LLM on downstream tasks, we adopt a parameter-efficient approach using low-rank adaptation (LoRA) \cite{20231213775448}.

（2）关键词元定位

To solve the issue of hiddenness of sequence-level hallucinations, we jointly consider the autoregressive generation rule and diagnosis-specific characteristics.
% We hypothesize that key tokens exist that act as sentinel positions that expose imminent uncertainty.
We hypothesize that certain key tokens act as sentinels, signaling imminent uncertainty.
Then, we carry out a point-biserial correlation computation on a held-out corpus to evaluate the hypothesis, comparing the energy of tokens that precede hallucinated answers against matched tokens in factual answers.
Finally, we perform a significance test to show that the decrease in energy is caused by the undesired input rather than by random fluctuation.
The significance test returns $p=3.2779\times10^{-28}$, rejecting the null that low energy tokens occur at random.

When generating diagnostic answers, LLMs autoregressively generate answers by predicting one token at a time and produce a probability distribution over the vocabulary for the next token, which can be formulated as
\begin{equation}
	P\bigl(\mathit{Token}_{k} = w \;\big|\; \mathit{Token}_{<k}\bigr)
	\;=\;
	\frac{\exp\bigl(\operatorname{Proj}(\bm{a}_{k})_{\,w}\bigr)}
	{\displaystyle \sum_{w' \in \mathcal{V}} 
		\exp\bigl(\operatorname{Proj}(\bm{a}_{k})_{\,w'}\bigr)},
\end{equation}
where $\mathit{Token}_{k}$ is the output token at generation step $k$, $\mathit{Token}_{<k}$ is the entire diagnostic answer of tokens generated before $k$, $\mathcal{V}$ is the LLM's native vocabulary set, $w$ is a single token drawn from $\mathcal{V}$, $\bm{a}_{k}$ is the hidden state vector at $k$, $\operatorname{Proj}(\cdot)$ is a learnable linear transformation function, and $\sum_{w\in \mathcal{V}} P\bigl(\mathit{Token}_{k} = w \mid \mathit{Token}_{<k}\bigr) = 1$.
When many tokens have lower probabilities, it indicates higher uncertainty in the next-token prediction.
Furthermore, the negative log-likelihood (NLL) of a token can be interpreted as the  token-level energy, which is formulated as
\begin{equation}
	\operatorname{NLL}{(\mathit{Token}_{k}=w)}
	=
	-\,\log
	\frac{\exp\bigl(\operatorname{Proj}(\bm{a}_{k})_{\,w}\bigr)}
	{\displaystyle\sum_{w' \in \mathcal{V}}
		\exp\bigl(\operatorname{Proj}(\bm{a}_{k})_{\,w'}\bigr)},
\end{equation}
where $\operatorname{NLL}{(\cdot)}$ is the token energy function.
A low NLL means the model assigns a high probability to that token, whereas a high NLL means that the LLM is ambiguous when generating this token.

Based on this analysis, our hypothesis is that when LLM outputs an incorrect fault classification result for the first time, the token energy changes abnormally around the incorrect classification tokens.
This hypothesis is based on two facts.
First, when the model outputs an incorrect fault category in its answer, it is a serious hallucination.
Second, the above analysis about the relationship between hallucination and energy shows that local hallucinations are often accompanied by an unusual change in token energy near the error.
%加一句话阐述什么时候升高什么时候降低

To validate our hypothesis, we first construct a held-out corpus. We use samples from the MBHM dataset \cite{Peng_Liu_Du_Gao_Wang_2025} as normal inputs, use samples from the MaFaulDa dataset \cite{MaFaulDa_dataset} as undesired inputs, and use the Qwen-2 1.5B model \cite{QWEN} as the LLM.
% For normal inputs and undesired input, we generate $N_\mathrm{nor}$ and $N_\mathrm{un}$ answers, respectively.
We generate $N_\mathrm{nor}$ answers for normal inputs and $N_\mathrm{un}$ for undesired inputs.
Moreover, token energies are calculated within a $\pm L_{\mathrm{win}}$ token window centered on the token at which the first misclassification occurs, denoted as $\mathit{Token}_{{\delta}}$.
Then, the point-biserial correlation coefficient, denoted $r_\mathrm{pb}$, is calculated to examine how the token energy relates to the corresponding input.
The point-biserial correlation is the Pearson correlation that fits the special case, quantifying the correlation between binary input type and the continuous token energy.
$r_\mathrm{pb}$ is formulated as
\begin{equation}
	r_\mathrm{{pb}}=
	\frac{\mu_\mathrm{nor} - \mu_\mathrm{un}}{\sigma_\mathrm{total}}
	\sqrt{\frac{N_\mathrm{nor}\,N_\mathrm{un}}{(N_\mathrm{nor}+N_\mathrm{un})^2}},
\end{equation}
where $\mu_\mathrm{nor}$ is the mean token energy for the answers of the normal input, and $\sigma_\mathrm{total}$ is the standard deviation of token energy across all answers.
Finally, we conduct a significance test to evaluate the results of $r_\mathrm{pb}$ to show that the rise in energy is caused by the undesired input rather than by random fluctuation.
The coefficient $r_{pb}$ is converted to a $t$-statistic, which is formulated as
\begin{equation}
	t=r_\mathrm{pb}\sqrt{\frac{(N_\mathrm{nor}+N_\mathrm{un}) - 2}{1 - r_\mathrm{pb}^2}},\,\mathrm{df} = (N_\mathrm{nor}+N_\mathrm{un}) - 2, 
\end{equation}
where $\mathrm{df}$ is the degrees of freedom.
Furthermore, the two-sided $p$-value, denoted as $p_{\text{two-sided}}$, is then obtained from the cumulative distribution function (CDF) of the student-$t$ distribution, which is formulated as
\begin{equation}
	p_{\text{two-sided}} = 2\bigl[1-F_{t_{\mathrm{df}}}(|t|)\bigr]
	= 2P\bigl(T_{\mathrm{df}}\ge |t|\bigr),
	\label{eq:p_value}
\end{equation}
where $F_{t_{\mathrm{df}}}(\cdot)$ is the CDF of the Student-$t$ distribution with $\mathrm{df}$, and $T_{\mathrm{df}}$ is a random variable following this distribution.


As illustrated in Fig. \ref{fig_correlation}, we compute $r_{\mathrm{pb}}$ and $p_{\text{two-sided}}$ between the binary input type and the token-energy sequence within $\pm L_{\mathrm{win}}=3$ centered on $\mathit{Token}_{\delta}$.
At the token $\mathit{Token}_{\delta-1}$, a large effect size is observed, where $|r_{\mathrm{pb}}| > 0.5$ and $r_{\mathrm{pb}} < -0.5$.
The observation indicates that an anomalous decrease in token energy is highly predictive of an undesired input.
Additionally, the significance test returns $p=3\times10^{-28}$, rejecting the null that low energy tokens occur at random.
From a theoretical standpoint, undesired inputs tend to induce localized energy distortions around $\mathit{Token}_{\delta}$.
Because ${\delta}-1$ lies immediately adjacent to ${\delta}$, it is intrinsically more responsive to the earliest signs of undesired inputs, aligning with the statistical evidence.
Accordingly, we adopt $\mathit{Token}_{\delta-1}$ as our key token for undesired input detection, while omitting $\mathit{Token}_{\delta+2}$ despite its higher correlation.

（3）关键词元驱动的不确定性重加权策略

To quantify the uncertainty in LLM outputs and to construct a TMLLM, we propose a key token-driven uncertainty reweighting strategy (KTURS).
The proposed strategy employs token-level energy as an uncertainty quantification metric and assigns greater weights to tokens that exhibit stronger correlation with the designated key token.


For an LLM output $\bm{\hat y}_{g}$ regarding a multimodal input $\bm{x}_{pt}$, denoted as $\bm{\hat y}_{g}=\{\mathit{Token}_{1},\mathit{Token}_{2},\dots,\mathit{Token}_{N_t}\}$, where $N_t$ is the length of output.
We first compute the normalized energy score for each token $\mathit{Token}_{k} (1\leq k\leq N_t)$, which is formulated as
\begin{equation}
	\label{equa:normalized_energy_score}
	\begin{split}
		{\operatorname{\tilde{NLL}}}(\mathit{Token}_k)
		= \,\frac{{\operatorname{NLL}}(\mathit{Token}_k)}
		{\sum_{j=1}^{N_t}{{\operatorname{NLL}}(\mathit{Token}_j)}\,}\,.
	\end{split}
\end{equation}
Then, we derive the token-level uncertainty by enlarging the weight of the key token according to $r_\mathrm{pb}$, denoted as
\begin{equation}
	\label{equa:token_level_uncertainty}
	\begin{split}
		&\operatorname{UT}(\mathit{Token}_k;\omega,r_{\mathrm{pb}})\\
		&= \bigl[1 + (\omega r_{\mathrm{pb}} - 1)\bm{1}_{\{k=\delta-1\}}\bigr]
		\tilde{\operatorname{NLL}}(\mathit{Token}_k),
	\end{split}
\end{equation}
where $\bm{1}_{\{k=\delta-1\}}$ equals 1 when the token index $k=\delta-1$ and 0 otherwise, and $\omega$ is a coefficient to calibrate the degree of token-level uncertainty.
Finally, the uncertainty for the LLM output can be formulated as
\begin{equation}
	\label{equa:overall_uncertainty}
	\operatorname{UA}(\bm{\hat y}_{g})
	= \sum_{k=1}^{N_t}
	\bigl[\,1 + (\omega\,r_{\mathrm{pb}} - 1)\,\bm{1}_{\{k=\delta-1\}}\bigr]\;\tilde{\operatorname{NLL}}(\mathit{Token}_k)\,.
\end{equation}


As shown in Fig. \ref{fig_strategy}, the raw energy scores are first normalized into the uncertainty distribution
$\tilde{\operatorname{NLL}}(\mathit{Token}_k)$, then the uncertainty of the key token $\mathit{Token}_{\delta}$ is amplified by the factor $\omega$, and finally the sequence-level uncertainty $\operatorname{UT}(\mathit{Token}_k;\omega,r_{\mathrm{pb}})$ is obtained by summing the token-level uncertainties over all $N_t$ tokens.

	\caption{}{Illustration of the key token-driven uncertainty reweighting strategy.}

\subsection{基于 DA-TMLLM 的抗幻觉方案}
\label{sec:scheme}

To alleviate and detect LLM hallucinations, the proposed hallucination-resilient scheme integrates the diffusion-based augmentation and the trustworthy multimodal LLM into a unified pipeline.

To alleviate statistical bias-induced hallucinations, the ControlNet-guided cross-dataset conditional diffusion model is designed to produce balanced and semantically aligned synthetic data.
The original and synthetic samples are then fused and used to fine-tune a trustworthy multimodal large language model.  
The pipeline in the training stage is summarized in Algorithm~\ref{alg:training}.
Before any test sample is input, we evaluate the mean and variance of the uncertainty scores on the validation set comprising only normal samples, denoted as $\mu_\mathrm{val}$ and $\sigma_\mathrm{val}$. 
This allows us to define the lower and upper bounds of our uncertainty threshold, $\psi_{lower}=\mu_\mathrm{val} - \sigma_\mathrm{val}$ and $\psi_{upper}=\mu_\mathrm{val} + \sigma_\mathrm{val}$, which together form the decision interval $[\psi_{lower}, \psi_{upper}]$.
Furthermore, we target the key token and design a key token-driven uncertainty reweighting strategy.
The pipeline in the inference stage is summarized in Algorithm~\ref{alg:inference}.

\begin{algorithm}[t]
	\caption{Training Pipeline of Diffusion-Augmented Trustworthy Multimodal LLM}
	\label{alg:training}
	\centering
	\begin{algorithmic}[1]
		\Require Pre-processed dataset $\mathcal{D}_\text{real}=\{(\mathbf{x}_{v},y)\}$, weights $\bm{\theta}_\mathrm{enc}$, ${\theta}_\mathrm{d}$, $\bm{\theta}_\mathrm{cla}$, $\bm{\theta}_\mathrm{ts}$, and $\bm{\theta}_\mathrm{llm}$, hyperparameters $S$, $N_\mathrm{dm}$, $N_\mathrm{llm}$, and $N_{\text{aug}}$, response text $\bm{y}_{g}$
		\Ensure Trained CGCDM $\bm{\theta}_\mathrm{d}^*$, fine-tuned TMLLM $\bm{\theta}_\mathrm{ts}^*$, $\bm{\theta}_\mathrm{llm}^*$
		\Statex \textbf{Stage 1: Train ControlNet-Guided CDM}
		\For{epoch $=1$ \textbf{to} $N_\mathrm{dm}$} \Comment{Training epochs $N_\mathrm{dm}$}
		\ForAll{$(\bm{x}_{v},y)\!\sim\!\mathcal{D}_\text{real}$}
		\State $\bm{x}_{v}^{(S)}\leftarrow\textsc{AddNoise}(\bm{x}_{v},S)$
		\State $L^{(s)}\!\leftarrow\!\lVert\epsilon-\epsilon_{\bm{\theta}_\mathrm{d}}(\bm{x}_{v}^{(s)},y,s)\rVert_2^2$
		\State $\bm{\theta}_\mathrm{d}^*\!\leftarrow\!\bm{\theta}_\mathrm{d}-\eta_{\bm{\theta}_\mathrm{d}}\nabla_{\bm{\theta}_\mathrm{d}}L^{(s)}$
		\EndFor
		\EndFor
		\Statex \textbf{Stage 2: Generate Augmented Samples}
		\For{$n=1$ \textbf{to} $N_{\text{aug}}$} \Comment{Augmented samples $N_{\text{aug}}$}
		\State $\bm x_{v}^{(S)}\sim\mathcal N(\bm 0,\bm I)$ 
		\For{$s=S$ \textbf{downto} $1$}
		\State $\bm c\leftarrow\textsc{ControlNet}(\bm{\theta}_\mathrm{enc}, \bm{\theta}_\mathrm{cla})$
		\State $\bm\mu\leftarrow\textsc{PosteriorMean}(\bm x^{(s)},s,\bm c,\bm{\theta}_\mathrm{d}^*)$
		\State $\bm\bar{\bm x}_{v}^{(s-1)}\sim\mathcal N\!\bigl(\bm\mu,(\sigma^{(s)})^{2}\bm I\bigr)$
		\EndFor
		\State $\mathcal D_{\text{syn}}\gets\mathcal D_{\text{syn}}\cup\{\bar{\bm{x}}_v^{(0)}\}$
		\EndFor
		\State $\mathcal{D}_\text{aug}\!\leftarrow\!\mathcal{D}_\text{real}\cup\mathcal{D}_\text{syn}$
		\Statex \textbf{Stage 3: Fine-tune Trustworthy Multimodal LLM}
		\State Initialize ${\bm{x}}_g\!\leftarrow\!\bar{\bm{x}}_v$
		\For{epoch $=1$ \textbf{to} $N_\mathrm{llm}$} \Comment{Training epochs $N_\mathrm{llm}$}
		\ForAll{$(\bm{x}_{g},\bm{y}_{g})\!\sim\!\mathcal{D}_\text{aug}$}
		\State $\bm{x}_{pt}\leftarrow\textsc{AlignBlock}(\bm{x}_{g})$
		\State $\bm{\hat y}_{g}\leftarrow\textsc{TMLLM}(\bm{x}_{pt})$
		\State $\bm{\theta}_\mathrm{ts}^*, \bm{\theta}_\mathrm{llm}^*\leftarrow \operatorname{LoRA}(\bm{\hat y}_{g}, \bm{y}_{g})$
		\EndFor
		\EndFor
		\State \Return $\bm{\theta}_\mathrm{d}^*$,  $\bm{\theta}_\mathrm{ts}^*$, $\bm{\theta}_\mathrm{llm}^*$
	\end{algorithmic}
\end{algorithm}

\begin{algorithm}[t]
	\caption{Inference Pipeline Based on Key Token-Driven Uncertainty Reweighting Strategy}
	\label{alg:inference}
	\begin{algorithmic}[1]
		\Require Test set $\mathcal{D}_\text{test}$, window length $L_{\mathrm{win}}$, weights of TMLLM $\bm{\theta}_\mathrm{ts}^*$ , $\bm{\theta}_\mathrm{llm}^*$, calibration coefficient $\omega$, key-token index $\delta-1$, uncertainty threshold $\psi$
		\ForAll{$\bm{x}_\mathrm{test}\!\sim\!\mathcal{D}_\mathrm{test}$}
		\State $\hat{\bm{y}}_\mathrm{test} \gets \mathrm{TMLLM}(\bm{\theta}_\mathrm{ts}^*$, $\bm{\theta}_\mathrm{llm}^*;\bm{x}_\mathrm{test})$
		\State $\hat{\bm{y}}_\mathrm{test}=\{\mathit{Token}_1,\dots,\mathit{Token}_{N_t}\}$
		\Statex \textbf{Stage 1: Token-Level Normalization}
		\For{$k = 1$ \textbf{to} $N_t$}
		\State $\operatorname{NLL}_k \gets -\!\log P\!\bigl(\mathit{Token}_k \mid \mathit{Token}_{<k}\bigr)$
		\EndFor
		\State $\displaystyle
		\tilde{\operatorname{NLL}}_k \gets 
		\operatorname{NLL}_k \Big/ \sum_{j=1}^{N_t}\operatorname{NLL}_j$
		\Statex \textbf{Stage 2: Key-Token Reweighting}
		\For{$k = 1$ \textbf{to} $N_t$}
		\State $\displaystyle
		\operatorname{UT}_k \gets
		\Bigl[1 + \bigl(\omega r_{\mathrm{pb}} - 1\bigr)\,
		\mathbf{1}_{\{k=\delta-1\}}\Bigr]\!
		\tilde{\operatorname{NLL}}_k$
		\EndFor
		\Statex \textbf{Stage 3: Sequence-Level Aggregation}
		\State $\displaystyle
		\operatorname{UA}\!\bigl(\hat{\bm{y}}_\mathrm{test}\bigr) \gets
		\sum_{k=1}^{N_t}\operatorname{UT}_k$
		\Statex \textbf{Stage 4: Decision}
		\If{$\operatorname{UA}\!\bigl(\hat{\mathbf{y}}_{\text{test}}\bigr)\not\in[\psi_{lower}, \psi_{upper}]$}
		\State \textbf{Reject} $\bm{x}_\mathrm{test}$ \Comment{hallucination detected}
		\Else
		\State \textbf{Output} $\hat{\bm{y}}_\mathrm{test}$
		\EndIf
		\EndFor
	\end{algorithmic}
\end{algorithm}

为全面评估 DA-TMLLM 及对比模型，实验采用 8 项评估指标。
其中 $\uparrow$ 表示数值越大性能越优，$\downarrow$ 表示数值越小性能越优。
设领域特定测试子集为 $\mathcal{D}_{\mathrm{ds}}=\{(x_i,y_i)\}_{i=1}^{N_{\mathrm{ds}}}$，非预期测试子集为 $\mathcal{D}_{\mathrm{un}}=\{\tilde{x}_j\}_{j=1}^{N_{\mathrm{un}}}$，复合测试集为 $\mathcal{D}_{\mathrm{mix}}=\mathcal{D}_{\mathrm{ds}}\cup\mathcal{D}_{\mathrm{un}}$。
$x_i$ 表示领域特定样本，$y_i\in\{1,\dots,N_c\}$ 表示其真实故障类别，$\hat{z}(\cdot)$ 表示模型在复合测试集上的最终判定结果，其取值为具体故障类别或非预期类标记 $\varnothing$。同时，定义 $c(\cdot)\in\mathbb{R}$ 为领域特定置信分数，其数值越大表示样本越可能属于领域特定分布。

基于上述说明和定义，所使用的 8 项评估指标包括：

（1）领域特定准确率（Domain-Specific Accuracy, DS-ACC, $\uparrow$），用于衡量模型在领域特定测试子集上的故障诊断能力，其定义为
\begin{equation}
\mathrm{DS\mbox{-}ACC}
=
\frac{1}{N_{\mathrm{ds}}}
\sum_{i=1}^{N_{\mathrm{ds}}}
\mathbb{I}\bigl(\hat{z}(x_i)=y_i\bigr),
\end{equation}
其中，$\mathbb{I}(\cdot)$ 为示性函数。

（2）总体准确率（General Accuracy, G-ACC, $\uparrow$）用于衡量模型在复合测试集上的整体判别性能，其定义为
\begin{equation}
\mathrm{G\mbox{-}ACC}
=
\frac{1}{N_{\mathrm{ds}}+N_{\mathrm{un}}}
\left[
\sum_{i=1}^{N_{\mathrm{ds}}}
\mathbb{I}\bigl(\hat{z}(x_i)=y_i\bigr)
+
\sum_{j=1}^{N_{\mathrm{un}}}
\mathbb{I}\bigl(\hat{z}(\tilde{x}_j)=\varnothing\bigr)
\right].
\end{equation}

（3）受试者工作特征曲线下面积（Area under Receiver Operating Characteristic Curve, AUROC, $\uparrow$）用于衡量领域特定样本与非预期样本之间的可分性。对于任意阈值 $\tau$，定义真阳性率与假阳性率分别为
\begin{equation}
\mathrm{TPR}(\tau)
=
\frac{1}{N_{\mathrm{ds}}}
\sum_{i=1}^{N_{\mathrm{ds}}}
\mathbb{I}\bigl(c(x_i)\ge \tau\bigr),
\end{equation}
\begin{equation}
\mathrm{FPR}(\tau)
=
\frac{1}{N_{\mathrm{un}}}
\sum_{j=1}^{N_{\mathrm{un}}}
\mathbb{I}\bigl(c(\tilde{x}_j)\ge \tau\bigr).
\end{equation}
AUROC 定义为受试者工作特征曲线在 $\mathrm{FPR}$ 与 $\mathrm{TPR}$ 平面下的面积，即
\begin{equation}
\mathrm{AUROC}
=
\int_{0}^{1}
\mathrm{TPR}\bigl(\mathrm{FPR}^{-1}(u)\bigr)\, \mathrm{d}u.
\end{equation}

（4）95\% 召回率下的误报率（False Alarm Rate at 95\% Recall, FAR@95, $\downarrow$）用于衡量在达到既定非预期样本召回水平时，领域特定样本被误判为非预期的比例。设阈值 $\tau_{95}$ 使得非预期样本召回率达到或最接近 $95\%$，即
\begin{equation}
\frac{1}{N_{\mathrm{un}}}
\sum_{j=1}^{N_{\mathrm{un}}}
\mathbb{I}\bigl(c(\tilde{x}_j)<\tau_{95}\bigr)
\approx 0.95.
\end{equation}
则 FAR@95 定义为
\begin{equation}
\mathrm{FAR@95}
=
\frac{1}{N_{\mathrm{ds}}}
\sum_{i=1}^{N_{\mathrm{ds}}}
\mathbb{I}\bigl(c(x_i)<\tau_{95}\bigr).
\end{equation}

（5）精确率-召回率曲线下面积（Area under the Precision-Recall Curve, AUPR, $\uparrow$）用于衡量模型在保留领域特定样本与排除非预期样本之间的综合表现。将领域特定样本视为正类时，对于任意阈值 $\tau$，精确率与召回率分别定义为
\begin{equation}
\mathrm{Precision}(\tau)
=
\frac{
\sum_{i=1}^{N_{\mathrm{ds}}}
\mathbb{I}\bigl(c(x_i)\ge \tau\bigr)
}{
\sum_{i=1}^{N_{\mathrm{ds}}}
\mathbb{I}\bigl(c(x_i)\ge \tau\bigr)
+
\sum_{j=1}^{N_{\mathrm{un}}}
\mathbb{I}\bigl(c(\tilde{x}_j)\ge \tau\bigr)
},
\end{equation}
\begin{equation}
\mathrm{Recall}(\tau)=\mathrm{TPR}(\tau).
\end{equation}
AUPR 定义为精确率-召回率曲线下的面积，即
\begin{equation}
\mathrm{AUPR}
=
\int_{0}^{1}
\mathrm{Precision}\bigl(\mathrm{Recall}^{-1}(u)\bigr)\, \mathrm{d}u.
\end{equation}

对于生成质量评估，设真实故障信号与生成故障信号分别为 $\mathbf{s}=(s_t)_{t=1}^{L_f}$ 与 $\hat{\mathbf{s}}=(\hat{s}_t)_{t=1}^{L_f}$。

（6）RMSE （$\downarrow$）用于衡量生成信号与真实信号在时域上的点对点偏差，其定义已在前文给出，从略。

（7）峰值信噪比（Peak Signal-to-Noise Ratio, PSNR, $\uparrow$）用于衡量生成信号的重建质量，其定义为
\begin{equation}
\mathrm{PSNR}
=
10\log_{10}
\left(
\frac{s_{\max}^{2}}{\mathrm{MSE}}
\right),
\end{equation}
其中，
\begin{equation}
s_{\max}=\max_{1\le t\le L_f}|s_t|,
\qquad
\mathrm{MSE}
=
\frac{1}{L_f}
\sum_{t=1}^{L_f}
(\hat{s}_t-s_t)^2.
\end{equation}

（8）频谱余弦相似度（Frequency Spectrum Cosine Similarity, FSCS, $\uparrow$）用于衡量生成信号与真实信号在频域结构上的一致性。记 $\mathcal{F}(\cdot)$ 表示离散傅里叶变换，$\mathbf{a}=|\mathcal{F}(\mathbf{s})|$ 与 $\hat{\mathbf{a}}=|\mathcal{F}(\hat{\mathbf{s}})|$ 分别表示真实信号与生成信号的幅值谱，则 FSCS 定义为
\begin{equation}
\mathrm{FSCS}
=
\frac{
\mathbf{a}^{\top}\hat{\mathbf{a}}
}{
\|\mathbf{a}\|_2\, \|\hat{\mathbf{a}}\|_2
}.
\end{equation}

在场景 1 中，DA-TMLLM 的 DS-ACC 和 G-ACC 分别较次优模型提升 4.05\% 和 5.51\%，同时达到理想的检测性能。在更复杂的混合领域场景 3 中，DA-TMLLM 仍在全部指标上取得最优结果。场景 2 是唯一的例外，在该场景下，DA-TMLLM 在 AUPR 指标上略低于 WDSN，但其 G-ACC 仍保持最高，且 FAR@95 相较于最接近的基线模型降低 98.95\%。综合 3 个场景的结果，DA-TMLLM 相较于最强竞争模型的 G-ACC 平均提升 6.85\%，FAR@95 平均降低 99.65\%。上述结果表明，DA-TMLLM 能够同时实现高精度故障诊断与稳健的幻觉检测。

相比之下，各基线模型表现出不同程度的局限性。VQVAE、DCGAN、QSCGAN 和 BearLLM 由于缺乏显式幻觉检测能力，难以可靠地区分领域特定输入与非预期输入，其性能退化表现为 DS-ACC 与 G-ACC 之间存在明显差距，且 AUROC 与 AUPR 得分较低。在具备检测能力的基线模型中，可以观察到较为明显的性能权衡。属于 CDLMs 范畴的 MLMM、WDSN 和 MDCC 通常在幻觉检测方面表现更优；相对而言，基于 LLM 的方法，包括 LLMCheck、MIND、SHalluDetect 和 LLM-OOD，在分布内分类任务中表现更好。然而，这两类方法均倾向于将有效的领域特定样本误判为非预期输入，并在处理被判定为幻觉的输入时表现出较强保守性。

\subsection{消融实验}\label{subsec:DA-TMLLM_ablation_study}

为量化 DA-TMLLM 各主要组成部分的贡献，开展消融实验。表 \ref{tab:ablation_study} 给出 3 个场景下基于 DS-ACC、G-ACC、AUROC、FAR@95 和 AUPR 的消融实验结果。

（1）w/o ControlNet

以宽核深度卷积神经网络替代 ControlNet 模块，从而移除条件引导机制，并使模型退化为常规去噪扩散概率模型（Denoising Diffusion Probabilistic Model, DDPM）。该变体在 3 个场景中的性能均出现下降，说明 ControlNet 对于在不同故障类型和工况条件下从振动信号中提取判别性特征具有关键作用。

（2）w/o CGCDM

移除所提出的 CGCDM，仅基于原始的类别不平衡故障信号训练诊断模型。该设置导致诊断准确率下降。该结果说明，所提出的基于扩散模型的故障模式增强方法能够有效缓解统计偏置，并修正由数据不平衡引起的决策边界偏移。

（3）w/o KTURS

以式~\eqref{equa:normalized_energy_score} 所定义的归一化能量分数替代式~\eqref{equa:token_level_uncertainty} 所定义的 token 级不确定性，从而移除所提出的 KTURS。结果显示，以 AUROC、FAR@95 和 AUPR 衡量的非预期输入识别性能出现下降。这表明，所设计的 KTURS 对序列级幻觉检测具有实际作用。

（4）w/o TMLLM

移除 TMLLM 后，框架的幻觉检测能力随之消失，非预期输入识别性能因此下降。该结果说明，所提出的方法能够有效量化不确定性，并提供可信的诊断结果。

\subsection{超参数敏感性分析}\label{subsec:DA-TMLLM_sensitivity_analysis}

为评估 DA-TMLLM 对超参数设置的敏感性，在 MBHM 数据集、MaFaulDa 数据集和 UP 数据集上开展敏感性分析。实验考察扩散模型噪声调度器 $\zeta$、诊断信号长度 $L_{f}$、LoRA 微调秩 $r_{LoRA}$ 以及关键 token 校准系数 $\omega$ 的影响。上述参数的影响分别通过生成信号与真实信号之间的 RMSE、ControlNet 在已知类别上的诊断准确率、最终微调轮次的二元交叉熵损失（Binary Cross-Entropy, BCE）以及非预期输入检测率进行评估。

\begin{figure}[!htbp]
    \centering
    \includegraphics[width=0.6\textwidth]{Img/DA-TMLLM_figures/trans/sa1.eps}
    \bicaption{DA-TMLLM~架构图}{Diagram of DA-TMLLM framework}
    \label{fig:sensitivity_sa1}
\end{figure}

（1）$\zeta$ 的影响

对线性、余弦和 Sigmoid 3 种噪声调度器进行比较。结果如图 \ref{fig:sensitivity_sa1} 所示，线性调度器在多数故障类型上均取得更优的信号重建效果。其原因在于，线性噪声注入方式与故障信号重建任务的归纳偏置更为一致，而非线性调度器可能削弱生成过程的稳定性。

\begin{figure}[!htbp]
    \centering
    \includegraphics[width=0.6\textwidth]{Img/DA-TMLLM_figures/trans/sa2.eps}
    \bicaption{DA-TMLLM~架构图}{Diagram of DA-TMLLM framework}
    \label{fig:sensitivity_sa2}
\end{figure}

（2）$L_{f}$ 的影响

通过测试 \{960, 1200, 1600, 2400, 4800\} 不同取值，分析诊断信号长度对 DA-TMLLM 性能的影响。图 \ref{fig:sensitivity_sa2} 表明，诊断准确率随信号长度增加而提高。较短信号难以提供充分的判别信息，而过长信号会在扩散过程中带来 GPU 显存瓶颈。因此，选取 $L_{f}=2400$，以兼顾诊断性能与计算开销。

\begin{figure}[!htbp]
    \centering
    \includegraphics[width=0.6\textwidth]{Img/DA-TMLLM_figures/trans/sa3.eps}
    \bicaption{DA-TMLLM~架构图}{Diagram of DA-TMLLM framework}
    \label{fig:sensitivity_sa3}
\end{figure}

（3）$r_{LoRA}$ 的影响

考虑到 LoRA 设置会影响微调阶段文本-信号对齐效果，实验分别以值投影层、查询投影层和全部线性层作为微调目标模块，对应设置分别记为 $v\_proj$、$q\_proj$ 和 $all\_linear$。上述 3 类模块的可训练参数量分别占模型总参数量的 0.0130\%、0.0223\% 和 0.300\%。同时，令 $r_{LoRA}$ 取值于 \{1, 2, 4, 8, 16\}。图 \ref{fig:sensitivity_sa3} 显示，随着可训练参数占比和 $r_{LoRA}$ 的增加，文本-信号对齐性能持续改善。当微调目标设置为全部线性层且 $r_{LoRA}=4$ 时，BCE 损失最低，为 0.0240。这一结果与表 \ref{tab:Hyperparameter} 中的最优超参数设置一致。

\begin{figure}[!htbp]
    \centering
    \includegraphics[width=0.6\textwidth]{Img/DA-TMLLM_figures/trans/sa4.eps}
    \bicaption{DA-TMLLM~架构图}{Diagram of DA-TMLLM framework}
    \label{fig:sensitivity_sa4}
\end{figure}

（4）$\omega$ 的影响

在 \{1, 2, $\cdots$, 50\} 取值范围内，评估不确定性校准系数 $\omega$ 对非预期输入检测性能的影响。图 \ref{fig:sensitivity_sa4} 表明，较大的系数能够提升 DA-TMLLM 对非预期 token 的识别能力。这是因为较大的 $\omega$ 能够更准确地刻画无偏的 token 级不确定性，从而放大领域特定 token 与非预期 token 之间的分布差异。

\subsection{故障信号生成质量评估}\label{subsec:DA-TMLLM_signal_generation_quality_evaluation}

为评估 DA-TMLLM 在领域特定数据集少数类别上的合成振动信号生成质量，将其与 3 种基于数据增强的基线方法进行比较，包括 VQVAE、DCGAN 和 QSCGAN。表 \ref{tab:comparison_result2_refined} 汇总了相关结果。可以看出，DA-TMLLM 在 RMSE 指标上取得最低值，在 PSNR 和 FSCS 指标上取得最高值。与各项结果中的次优模型相比，DA-TMLLM 的平均 RMSE 降低 86.9\%，平均 PSNR 和 FSCS 分别提升 97.1\% 和 67.2\%。上述结果表明，DA-TMLLM 在数据增强任务中具有更优性能。
% 
该优势主要归因于所采用的条件扩散模型。该模型通过引导式渐进去噪过程，能够更有效地刻画复杂的非平稳信号特征。相比之下，DCGAN 和 QSCGAN 易受训练不稳定性的影响，因而难以完整表征真实数据分布；VQVAE 则倾向于生成过于平滑的信号，难以保留关键高频细节。

\section{本章小结}\label{sec:DA-TMLLM_conclusion}

本文提出 DA-TMLLM，一种用于检测并缓解诊断大语言模型幻觉的新方法。
% 
首先，构建 ControlNet 引导的跨数据集条件扩散模型，通过注入可控语义引导并合成多样化样本，以缓解统计偏置。
% 
其次，设计 TMLLM，通过点二列相关分析与显著性检验定位关键 token，并进一步构建关键 token 驱动的不确定性重加权策略，以提升幻觉检测性能。
% 
再次，提出一种抗幻觉方案，将基于扩散的数据增强机制与 TMLLM 统一集成于同一流程中，实现幻觉的联合检测与缓解。
% 
基于 11 个故障诊断数据集的评估结果表明，DA-TMLLM 在 8 项指标上均持续优于 11 种对比模型。
