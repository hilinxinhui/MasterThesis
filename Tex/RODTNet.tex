\chapter{基于拓扑约束下不确定性驱动特征调弈的锂离子电池分布外鲁棒退化建模方法}\label{chap:RODTNet}

\section{引言}\label{sec:RODTNet_introduction}

在电动汽车作为可持续交通体系载体的关键性越发显现的背景下，其动力总成中的储能部件，LIBs，的重要性不言而喻\cite{10766661, 9849057, 10897846, WASEEM2025236414}。
LIBs 不断进行充放电循环并在这一过程中发生性能退化。
导致可用续航里程削减，热失控风险增加，退役后梯次利用的经济价值降低\cite{LI2024155588}。
从而，对 LIBs 的性能退化建模重要且迫切。通常使用一对各自有所侧重的指标系统评估 LIBs 的性能退化程度，分别是 RUL 和 SOH。
具体地，前者通过量化容量衰减至寿命终止阈值前的剩余循环次数来提供前瞻性预测，后者则通过当前容量与标称容量的比值来反映当前状态。

% para 2
鉴于车载应用对非侵入式测量的限制，LIBs 的 RUL 和 SOH 只能依据车载电池管理系统（Vehiclar Battery Management Systems, VBMS）采集的监测数据，通过构建退化模型进行间接预测或估计。相比于难以应对复杂多物理场耦合效应的传统机理方法，数据驱动方法，尤其是基于深度学习的方法，在建立循环信号与退化状态之间的非线性映射方面展现出显著优势 \cite{XIAO2023107347, XIE2024630}。然而，此类方法的预测过估计性能高度依赖训练数据与测试数据同分布的前置假设，这种假设在复杂多变的实际驾驶工况下往往难以成立。此外，获取覆盖全运行工况的高质量带标签数据伴随着高昂的成本，这进一步加剧了数据驱动模型在实际应用中的泛化难度。

% para 3
现有研究广泛探讨了数据驱动 LIBs 退化建模问题中的训练与测试数据分布漂移问题，并主要借助预训练微调范式或域自适应范式构建模型 \cite{10177255}。然而，此类方法高度依赖目标域数据的可访问性。在已部署的车载系统中，由于通信带宽瓶颈、隐私保护法规以及实时计算资源的约束，获取充足的目标域数据存在很大现实阻碍 \cite{10886998, 10663939, 9849057}。进一步地，LIBs 退化过程对应连续的目标空间且具有内在的序数演化特征，这使其数据分布从根本上区别于异常检测与故障诊断等依赖离散标签的任务的数据分布 \cite{10214203, 10577994}。这种独有的序数特征导致在故障诊断等领域中已被充分研究和验证的泛化失效，仅利用源域数据实现 OOD 退化模式泛化的难度很大。尽管部分融合表征学习或元学习的策略在一定程度上实现了无需目标域数据的 LIBs 退化建模 \cite{TAN2024103725, CHEN2024234696}，但其建模过程仍依赖于预设的退化先验知识过预设的退化相关任务。与之相对地，基于数据调弈（Data Manipulation, DM）的域泛化范式 \cite{9782500} 凭借其简洁性和与底层数据驱动模型解耦的灵活性展现出优势。然而，在建立从原始传感输入到最终退化指标的多阶段复杂信息映射时，基于数据调弈的泛化策略仍需以下三个挑战：

（1）第一个挑战来自于循环数据时空信息的稀缺性，这来源于车载应用对非侵入式测量的限制。
在时间维度上，等间隔采样机制导致 VBMs 在动态复杂且对退化高度敏感的工况区间内数据采集过于稀疏 \cite{9648002, BOSE2023107543, zhu2022data, GUAN2018204, XIONG2020110048}，却在信息量较低的平稳运行阶段产生冗余记录，进而造成对关键时间退化特征的表征不足。
在空间维度上，车载循环数据仅包含有限的传感器模态，所能获取的多物理场信息受限，难以有效捕捉热力学应力 \cite{XIONG2020110048} 与机械变形 \cite{zeng2023nonintrusive} 等 LIBs 内部的关键退化机制。
如图\ref{fig:rodtnet_challenges}(a) 所示，若直接基于这样具有时空稀疏性的原始循环信号建模，将使得映射出退化特征表现复杂的高维流形结构，导致提取的退化特征缺乏必要的拓扑连续性，后续 DM 操作易使模型过拟合局部噪声，并降低其在训练分布之外的泛化性能。

% para 5
（2）第二个挑战在于未考虑域漂移在特征空间表征中引入的统计不确定性。LIBs 性能退化过程的内在复杂性及其对工况的高度敏感性 \cite{LI2024155588}，导致退化模式在多变的运行条件下不可避免地呈现出分化。这种分化直接引发了数据分布的漂移现象，使得基于源域数据训练的模型在面对分布不同的 OOD 数据时，产生统计不确定性 \cite{9577663, pmlr-v202-dai23d, liuncertainty}。如图~\ref{fig:rodtnet_challenges}(b) 所示，域漂移主要表现为由充电协议、电极材料及放电深度等差异引起的协变量漂移，以及由不同退化轨迹演变导致的概念漂移。由域漂移引起的统计不确定性破坏了模型习得的特征与退化状态之间映射关系的一致性，限制了模型向未见目标域的泛化能力，最终阻碍数据驱动方法的实际应用。

% para 6
（3）第三个挑战源于多域长尾退化分布，这一挑战本质上是由 LIBs 内部复杂的热电耦合反应导致的，在宏观上表现为高度多样化的非线性退化轨迹。
具体地，LIBs 性能退化轨迹包括不止一个阶段，其多阶段动态演化过程通常伴随明显的容量拐点，标志着从早期和中期缓慢衰减向后期加速退化的转变。
这种多阶段非线性的退化过程导致循环退化数据相对集中于 LIBs 寿命的早期和中期阶段，对应地，后期加速退化阶段的样本分布则较为稀疏。
如图~\ref{fig:rodtnet_challenges}(c) 中的快速充电工况实例所示，非线性退化特性产生了不对称的长尾样本分布。未能显式建模此分布特征的模型在面对低频次尾部退化样本时表现出严重的性能下降。
长尾分布使得直接进行 DM 操作将使得模型向数据量庞大的早中期寿命阶段倾斜，产生破坏模型跨域泛化能力的偏态特征表征。此外，长尾分布在多个不同源域之间存在异质性，这进一步增加了退化特征空间的不连续程度，阻碍了面向实际应用场景的鲁棒退化建模的实现。

% para 7
\sloppy
为应对上述挑战，本章提出一种鲁棒分布外退化模式跟踪网络（Robust Out-of-Distribution Degradation  Tracking Network, RODTNet），及其长尾优化变体（Robust Out-of-Distribution Degradation Tracking Network with Long-Tail Optimization, RODTNet-LT）。
% 
首先，为克服时空信息稀缺问题，提出基于流形收缩的鲁棒退化表征提取策略（Manifold Contraction-based Strategy for Robust Degradation Representation, MCRDR）。通过解析退化特征流形的拓扑结构，基于具备良泛化能力表征的几何约束，融合启发式信号重构策略与 Mamba 架构实现有效的可泛化特征映射。
% 
其次，为有效应对域分布漂移，提出不确定性驱动的特征合成策略（Uncertainty-Driven Feature Synthesis, UDFS）。利用多元 Gaussian 分布对目标域的固有统计规律进行量化与采样，向外拓展已学习的退化特征流形的分布边界，增强数据驱动的退化建模模型的跨域泛化的能力。
% 
最后，针对多域长尾分布，提出感知长尾分布的自适应重平衡微调策略（Long-Tail-Aware Adaptive Rebalancing Refinement, LARR）。利用 Gaussian 核平滑机制引导特征合成过程向低密度退化模式空间探索，进而在标签空间层面重塑并实现了更加均衡的特征表征。RODTNet 和 RODTNet-LT 贡献如下：

（1）提出基于流形收缩的鲁棒退化表征提取策略，约束特征流形复杂度和连续性，生成 OOD 鲁棒退化表征。

（2）提出不确定性驱动的特征合成策略，通过统计不确定性量化域变异性，指导域不变特征提取和 OOD 退化模式合成。

（3）提出感知长尾分布的自适应重平衡策略，利用特征空间与标签空间间相似性及分布平滑技术，缓解特征合成的头部偏见，增强针对不平衡退化轨迹的泛化能力。

（4）在 4 个 LIBs 老化数据集上通过 11 个 OOD 泛化任务系统验证 RODTNet 和 RODTNet-LT 的有效性，证明其相较于 14 种基线方法及其 20 种变体的先进性，阐述将其应用在可靠 VBMS 中的潜力。

\section{基于拓扑约束下不确定性驱动特征调弈的退化建模方法}\label{sec:RODTNet_methodology}

\begin{figure}[!htbp]
    \centering
    \includegraphics[width=\textwidth]{Img/RODTNet_figures/trans/Fig2_RODTNet_framework.eps}
    \bicaption{RODTNet(-LT) 架构图}{Diagram of the RODTNet-(LT) framework}
    \label{fig:rodtnet_framework}
\end{figure}

为提升智能 VBMS 中预测算法的可靠性，并增强系统面对未见退化模式的鲁棒性，本章提出鲁棒分布外退化跟踪网络 RODTNet 及其针对长尾分布优化的变体 RODTNet-LT。如图 \ref{fig:rodtnet_framework} 所示，RODTNet 包含两个基础组件，即基于流形收缩的鲁棒退化表征提取策略 MCRDR 和不确定性驱动的特征合成方法 UDFS。在此基础上，RODTNet-LT 进一步引入感知长尾分布的自适应重平衡微调机制 LARR，以应对不平衡退化数据分布引发的泛化挑战。
% 
具体地，提取策略 MCRDR 首先利用启发式信号重构处理原始车载传感器数据，并借助基于 Mamba 架构的特征提取器将其映射为具备低拓扑复杂度和高拓扑连续性的潜在流形表征，从而克服时空信息稀疏的局限性。
% 
其次，特征合成策略 UDFS 对已知源域退化模式的特征分布进行统计建模，定量评估由域漂移引发的统计不确定性，并以此为引导合成逼近目标域真实物理分布的模拟 OOD 特征，从而增强模型的 OOD 泛化能力。
% 
最后，微调策略 LARR 通过自适应地校正并重平衡特征空间，进一步优化前序特征的分布结构。该机制提升网络对高频常规模式与稀有尾部退化轨迹的综合表征能力，有效解决多域长尾分布问题。

针对分布漂移条件下的 LIBs 退化建模问题，本节给出 OOD 退化建模的形式化定义。令 $\boldsymbol{x}_{t} \in \mathbb{R}^{L \times C}$ 表示 LIBs 在第 $t$ 次循环时采集的多传感器监测信号序列，其中 $L$ 为序列长度，$C$ 为传感器通道数；同时令 $y_{t} \in \mathbb{R}$ 表示对应的剩余使用寿命或健康状态等退化表征量。成对观测值 $(\boldsymbol{x}_{t}, y_{t})$ 构成定义在样本空间 $\mathcal{X} \times \mathcal{Y}$ 上的随机变量 $(X, Y)$ 的实现样本，该随机变量服从边缘分布为 $P_{X}$ 与 $P_{Y}$ 的联合概率分布 $P_{XY}$。实际应用中，汽车驾驶工况与环境温度等外部物理条件，以及 LIBs 内部电化学状态的动态演变，共同构成复杂多样的运行域。假设系统总计包含 $K = M + N$ 个域，记为 $D^{i} = \{ (\boldsymbol{x}_{t}, y_{t}) \} \sim P^{i}_{XY}$，其中 $i=1,\dots,K$。该问题的挑战在于不同域之间存在显著的分布漂移现象，即对于任意 $i \neq j$ 均满足 $P^{i}_{XY} \neq P^{j}_{XY}$。鲁棒 OOD 退化建模的根本目标在于，仅利用采集自 $M$ 个源域 $\mathcal{S} = \{ D^{i} \}_{i=1}^{M}$ 的数据，训练一个由参数 $\theta$ 决定的映射模型 $f_{\theta}: \mathcal{S} \to \mathcal{T}$。该模型需具备在训练阶段完全不接触目标域数据的前提下，泛化至 $N$ 个未见目标域 $\mathcal{T} = \{ D^{j} \}_{j=M+1}^{M+N}$ 的能力。在数学表达上，该目标等效于最小化模型在未见目标域上的期望预测损失 $\ell$，即求解最优化问题 $\mathop{\min}_{\theta}\mathbb{E}_{(\boldsymbol{x}, y) \in \mathcal{T}} [\ell(f_{\theta}(\boldsymbol{x}), y)]$，其中期望值严格基于目标域的真实数据分布计算。

\subsection{基于流形收缩的鲁棒退化表征提取策略}

为应对时空信息稀疏的挑战，本节提出 MCRDR 策略。
该策略强化关键时间片段表征并实现空间维度扩展，驱动后续特征合成步骤提升退化特征在调整空间中的分布密度。
% 
具体地，
首先，从流形拓扑的视角深入剖析退化特征泛化能力与其固有几何属性间的映射关系。
其次，提出一种启发式循环信号重构方法，这种重构可以得到有利于退化模式泛化几何性质的退化特征。
最后，构建基于 Mamba 的特征编码器，实现重构输入向高维特征空间的映射。

（1）拓扑视角下的退化表征分析

受相关领域，如故障诊断领域，的特征泛化研究的启发，本节系统阐述退化模型中具备强泛化能力的特征的几何属性。
简单起见，暂略去循环与域索引。
显式定义对应于输入 $\boldsymbol{x}$ 的潜在变量 $\boldsymbol{z}$。据此，将数据域重新定义为 $\mathcal{D} = \{( \boldsymbol{x}, \boldsymbol{z}, y)\}$，相应地更新对源域 $\mathcal{S}$ 和目标域 $\mathcal{T}$ 的定义。令 $Z$ 为取值于 $z \in \mathcal{Z}$ 的随机变量，其概率分布记为 $P_Z$。
假设提取的潜在表征均严格分布于嵌入在特征空间中的一个流形 $\mathcal{M}_Z$ 上。

基于信息瓶颈（Information Bottleneck, IB）理论 \cite{alemi2017deep} 评估所得到的退化表征质量，并以本征维度（Intrinsic Dimension, IntDim）\cite{10.5555/3692070.3694510} 量化退化表征的几何复杂程度。

IB 理论的优化准则要求在最大化潜在表征 $Z$ 与退化标签 $Y$ 之间互信息 $\mathcal{I}(Z;Y)$ 的同时，约束该表征所保留的关于原始输入 $X$ 的冗余互信息 $\mathcal{I}(Z;X)$。
即求解下述极小化目标函数
% 
\begin{equation}
    \min_{Z} \mathcal{IB} = \min_{Z} [- \mathcal{I}(Z;Y) + \beta \mathcal{I}(Z;X)], 
    \label{IB_mutual_info}
\end{equation}
其中 $\mathcal{I}(\cdot;\cdot)$ is mutual information, and $\beta > 0$ is a Lagrangian multiplier. 
其中 $\mathcal{I}(\cdot;\cdot)$ 表示互信息度量，$\beta > 0$ 为控制压缩强度的拉格朗日乘子。
不失严谨性地，IntDim 定义为实现流形等距嵌入所需的最小维数。

鉴于 OOD 退化建模任务中跨域时产生的协变量漂移，修改 IB 目标函数的结构，使其聚焦于表征 $Z$ 与标签 $Y$ 之间的条件依赖关系，最小化 IB 目标函数等价于求加下式
% 
\begin{equation}
    \min_{Z} \mathcal{IB} = \min_{Z} [(1-\beta) \mathcal{H}(Y \mid Z) + \beta \mathcal{H}(Z \mid Y)], 
    \label{IB_cond_entropy}
\end{equation}
% 
其中 $\mathcal{H}(\cdot \mid \cdot)$ 为条件熵函数，权重因子满足 $\beta \in (0,1)$。
目标函数中的 $\mathcal{H}(Y \mid Z)$ 项旨在提升标签的条件可预测性， $\mathcal{H}(Z \mid Y)$ 项不仅规避潜在的过拟合风险，亦建立退化表征和退化特征流形拓扑结构的联系。

预测模型在源域与目标域之间的泛化误差以包含互信息 $\mathcal{H}(Z \mid Y)$ 的函数为上限，即有
% 
\begin{align}
    & \mathbb{E}_{\{\boldsymbol{x},\boldsymbol{z},y\}\thicksim \mathcal{S}}[||f(\boldsymbol{z})-{y}||_2] -\mathbb{E}_{\{\boldsymbol{x},\boldsymbol{z},y\}\thicksim \mathcal{T}}(||f(\boldsymbol{z})-{y}||_2) \notag \\
    & \leq 2KQ(\mathcal{H}(Z \mid Y)) \label{generalization_error}
\end{align}
% $||\cdot||_2$ is the $\ell_{2}$ norm

其中 $K$ 为下游任务回归器的 Lipschitz 常数，而泛函 $Q(\cdot)$ 依赖于条件概率分布 $P(Z \mid Y)$，控制该泛化误差上界的紧致度。进一步地，条件熵 $\mathcal{H}(Z \mid Y)$ 以条件流形 $\mathcal{M}_i \subset \mathcal{M}$ 的平均 IntDim 为上界，该流形空间对应于给定退化标签 $y = y_i$ 条件下的观测表征 $\boldsymbol{z}$，其约束关系为

\begin{equation}
    \mathcal{H}(Z \mid Y) \leq \mathbb{E}_{{y}_{i} \sim \mathcal{Y}}[-\log(\epsilon)IntDim(\mathcal{M}_i)+\log\frac{K^{\prime}}{C(\epsilon)}]
    \label{intrinsic_dimension} 
\end{equation}
其中 $\epsilon \to 0^+$ 为定义局部流形结构的极小邻域半径，$K^{\prime}$ 为常数，$C(\epsilon)$ 为仅依赖于 $\epsilon$ 的特定连续函数。本节所提模型满足上述误差边界成立的各项前提假设 \cite{Ghosh_Motani_2023, 10.5555/3692070.3694510}。
% $i$ indexes samples
% $ID(\cdot)$ is the intrinsic dimension of the manifold
% Conditions, proof methodologies and other details for these bounds \cite{Ghosh_Motani_2023, 10.5555/3692070.3694510} are satisfied by our model, and details are omitted for brevity.

基于式 \eqref{generalization_error} 和式 \eqref{intrinsic_dimension}，结合 IB 原则在信息压缩率和建模准确性间的平衡关系，理想的退化表征应当具有以下两条几何性质：

（1）退化特征流形 IntDim 低，即映射出表征流形具有高度紧凑且相对简单的几何结构，允许在更低维的子空间内保持退化信息。

（2）退化特征流形拓扑连续，在特征空间中表现为不同老化阶段的表征分布沿着退化轨迹呈现出有序且可分离的演化结构。

这两条几何性质对跨域泛化退化建模重要。
一方面，低退化特征流形 IntDim 意味着退化特征分布结构简单，将 ID 分布建模和 OOD 退化模式泛化的分布外推限制在规整的表征子空间内，简化了特征生成环节。
% 
另一方面，退化状态间的连续性与有序演进规律促进域不变表征的学习，通过减少相邻老化状态间的特征干扰并实现可靠的密度估计，为特征空间操作提供基础。

IntDim 需要迭代估计，计算成本高昂，直接在原始高维信号空间对退化模型施加包含 IntDim 项的正则化约束以获取拓扑最优的退化表征难度很大。同时，原始信号空间稀疏覆盖使得特征流形在初始训练时即面临很大的不连续性，进一步影响退化模型收敛。

受到基于 PINN \cite{wang2024physics, 10251604} 的 LIBs 退化建模的相关工作的启发，本节提出一种循环信号重构机制。该机制通过提取原始传感序列 $\boldsymbol{x}_{t} \in \mathbb{R} ^ {L \times C}$ 中各传感器通道的统计演化特征实现维度重构。提取的特征包含时间差分以及通道维度的统计量，如均值与标准差。拼接这些统计特征，原始信号被重构为具有潜在低拓扑复杂度的输入信号 $\boldsymbol{\widetilde{x}}_{t} \in \mathbb{R} ^ {D \times 1}$，其中，重构特征的总维度 $D$ 满足 $C < D \ll L$。

信号构建的具体方法详见第 \ref{subsec:experimental_setup} 小节。第 \ref{subsec:comparative_experiments} 小节的实验验证表明，该重构机制减少高频时间噪声干扰并在空间维度上扩展特征，缓解时空退化特征稀疏性导致的表征泛化能力不足，进而生成具备更低 IntDim 与更高拓扑连续性的退化表征，为训练高泛化性退化模型提供几何基础。

基于泛化几何属性的分析和启发式信号重构机制是实现可泛化退化建模的无偏基础。
% 
从特征生成的视角，原始高维观测信号 $\boldsymbol{x}_{t}$ 由退化过程固有变量 $\boldsymbol{x}_{inv}$ 和特定工况下的虚假相关噪声变量 $\boldsymbol{x}_{spu}$ 耦合。直接在原始数据空间进行特征映射导致模型拟合源域特定工况噪声，使提取的潜在表征 $Z$ 与域变量 $D$ 形成伪相关。这种统计依赖导致给定退化标签 $Y$ 时，表征 $Z$ 的条件概率分布跨域不一致，其分布漂移现象形式化表述为
% 
\begin{equation}
    P(Z \mid Y, D) \neq P(Z \mid Y)
\end{equation}
% 
该漂移过程向退化特征空间引入域分布偏置。为消除此偏置，设计的信号重构机制等效于施加物理先验约束的变换算子 $g(\cdot)$。该算子通过约束特征空间与噪声变量之间的互信息 $\mathcal{I}(g(\boldsymbol{x}_{t}); \boldsymbol{x}_{spurious})$ 消除工况干扰，使重构信号逼近退化退化固有变量 $\boldsymbol{x}_{inv}$。由此生成的流形结构 $\mathcal{M}_Z$ 具备低 IntDim 与拓扑连续性，在几何层面上阻止局部域偏置的特征传递，从而提取的退化表征满足以下条件独立性约束
% 
\begin{equation}
    Z \perp \!\!\! \perp D \mid Y
\end{equation}
% 
所提出的启发式循环信号重构使得特征演化轨迹由退化物理规律主导，避免对特定源域分布的过拟合，从而在特征提取阶段确立无偏的泛化退化建模范式。

（2）结合流形收缩的 Mamba 模型

将重构信号映射至特征空间的建模过程需兼顾退化演化的时间序列依赖性、LIBs 系统向线性时不变（Linear Time-Invariant, LTI）系统的可近似性，以及车载边缘计算平台的计算资源约束。Mamba 架构具备长序列建模能力、与 LTI 系统同构的状态空间机制、计算复杂度的线性扩展性以及高效的并行推理特性 \cite{gu2024mamba}。该架构的数学本质与工程优势契合 LIBs 退化建模的物理与计算约束，因此本小节基于 Mamba 架构设计特征映射编码器。

沿时间维度堆叠 $E$ 个连续充放电循环的重构信号，构造二维输入张量 $\widetilde{X}_{Recon} \in \mathbb{R}^{D \times E}$。
该张量首先通过通道独立的线性投影转换为嵌入特征 $\widetilde{X}_{Emb} \in \mathbb{R}^{H \times E}$，其中 $H$ 为隐状态特征维度。
随后，上述嵌入特征经由三分支并联残差结构映射至高维退化特征空间，其前向传播过程形式化表述如下
% 
\begin{align}
& Z^{B}_{MRB} = SiLU(Linear(\widetilde{X}_{Emb})), \\
& Z^{M}_{MRB} = SSM(SiLU(Conv(Linear(\widetilde{X}_{Emb})))), \\
& Z_{MRB} = Linear(Z^{M}_{MRB} \odot Z^{B}_{MRB}) + \widetilde{X}_{Emb},
\end{align}
% 
其中 $Z^{M}_{MRB}$、$Z^{B}_{MRB}$ 和 $Z_{MRB}$ 分别为残差块的主路特征流、旁路特征流和聚合输出。映射过程涉及的基础算子包含逐元素哈达玛乘积 $\odot$、全连接线性映射 $Linear(\cdot)$、一维因果卷积 $Conv(\cdot)$、自门控激活函数 $SiLU(\cdot)$ 和状态空间变换 $SSM(\cdot)$。
% 
状态空间变换 $SSM(\cdot)$ 实现为选择性结构化状态空间序列（Selective Structured State Space Sequence, S6）模型，其内部状态的递推演化关系与参数离散化过程定义为
% 
\begin{align}
& h_{t} = \overline{A}h_{t-1} + \overline{B}\widetilde{x}_{t}, \\
& \overline{A} = DR_{A}(\Delta, A) = exp(\Delta A), \\
& \overline{B} = DR_{B}(\Delta, A, B) = (\Delta A)^{-1}(exp(\Delta A)-I)\cdot\Delta B,
\end{align}
% 
其中 $A$ 和 $B$ 为可训练矩阵。
采用零阶保持（Zero-Order Hold, ZOH）$DR$ 实现参数离散化映射 \cite{gu2024mamba}，
对矩阵 $A$ 实施 HiPPO 初始化 \cite{20212610553629}，
控制矩阵 $B$ 与离散化步长因子 $\Delta$ 为随输入信号 $\widetilde{x}_{t}$ 动态变化的参数化函数。

\subsection{不确定性驱动的退化特征合成策略}\label{subsec:UDFS}

为应对域漂移导致的统计量不确定性，并提升模型泛化能力，本节提出 UDFS 策略。
该策略基于收缩的退化特征执行分布边界的特征外推。具体地，
% 
首先利用多元正太分布（Multivariate Normal Distribution, MND）针对特征统计量构建概率模型。
其次基于该统计量方差的非参数化估计显式量化跨域不确定性。
最后依据量化结果进行特征采样，合成表征潜在分布外 OOD 退化模式的辅助特征。
UDFS 策略嵌入融合流形收缩的 Mamba 架构，从特征层面增强模型对 OOD 实例的鲁棒性。

（1）面向域漂移的多元 Gaussian 建模

特定域的退化模式反映在 Mamba 编码器映射出的退化特征的统计量分布上。域漂移使得不同运行工况下特征统计量产生不一致性。
本文认为这种不一致性是可量化的。
% 
通过特征统计量固有的不确定性显式建模这种统计量不一致性，能够基于源域数据合成代表未见目标域的特征，进而提升退化模型的跨域鲁棒性。为量化这种跨域变异性，需要辨识支配 ID 特征统计量的潜在分布，并结合其采样方差对其分布的参数进行估计。

鲁棒 OOD 退化建模中的统计量不确定性量化
本质上是估计退化特征统计量分布的低阶矩，这一过程的实施要求退化特征统计量的分布具备特定的数学性质。
理想情况下，分布需满足 Carleman 条件 \cite{akhiezer2020classical}，以确保其可由各阶矩唯一确定，且其低阶矩需包含充足的统计信息。
MND 严格满足该条件，且可以完全由均值向量和协方差矩阵刻画，契合上述理论约束。
鉴于 MND 的 Carleman 条件符合性及其在 LIBs 退化建模问题中的成熟应用，采用 MND 显式建模 ID 退化特征统计量的分布。

具体地，对于批量大小为 $B$ 的样本集映射生成的特征张量 $Z_{MRB} \in \mathbb{R}^{B \times H \times E}$，本节采用批次均值 $\mu_{Z}$ 和批次标准差 $\sigma_{Z}$ 构建其统计表征。基于前述 MND，假设特征统计量服从正态分布，即 $\mu_{Z} \sim \mathcal{N}(\mu, \Sigma_{\mu}^{2})$ 和 $\sigma_{Z} \sim \mathcal{N}(\sigma, \Sigma_{\sigma}^{2})$，其中 $\mu$ 与 $\sigma$ 分别为 $\mu_{Z}$ 和 $\sigma_{Z}$ 的数学期望。退化特征统计量的分布的参数可利用特征张量的经验均值与标准差直接估计，定义为
% 
\begin{align}
    & \mu=\frac{1}{HE}\sum_{H}\sum_{E}Z_{MRB}, \\
    & \sigma = \sqrt{\frac{1}{HE}\sum_{H}\sum_{E}(Z_{MRB} - \mu)^{2}}.
\end{align}
% 
其中 $\Sigma_{\mu}^{2}$ 与 $\Sigma_{\sigma}^{2}$ 分别为特征统计量 $\mu_{Z}$ 与 $\sigma_{Z}$ 的方差，这些参数同时用于显式量化由域漂移导致的统计量不确定性。

基于观测的特征统计量批次内变异性，该方差的非参数化估计过程可以形式化为
% 
\begin{align}
    & \Sigma_{\mu}^{2}=\frac{1}{B}\sum_{b\in B}(\mu-\mathbb{E}_{b}(\mu))^{2}, \\
    & \Sigma_{\sigma}^{2} = \frac{1}{B}\sum_{b\in B}(\sigma-\mathbb{E}_{b}(\sigma))^{2},
\end{align}
% 
其中 $ \mathbb{E}{b}(\mu) $ 和 $ \mathbb{E}_{b}(\sigma) $ 分别是批量内第 $b$ 个样本的均值和标准差的期望值。

（2）不确定性驱动的特征合成

基于原始的退化特征统计量服从参数分别为 $\mu, \Sigma_{\mu}$ 与 $\sigma, \Sigma_{\sigma}$ 的 Gaussian 分布前提，退化特征统计量的可微生成过程采用重参数化方法，写作
% 
\begin{align}
    & \mu_{Z}^{\prime}=\epsilon_{\mu}\Sigma_{\mu}+\mu,  \\
    & \sigma_{Z}^{\prime}=\epsilon_{\sigma}\Sigma_{\sigma}+\sigma, 
\end{align}
% 
其中 $\mu_{Z}^{\prime}$ 与 $\sigma_{Z}^{\prime}$ 分别为生成的特征均值与标准差变量，$\epsilon_{\mu}$ 与 $\epsilon_{\sigma}$ 为自标准正态分布 $\mathcal{N}(0,1)$ 采样的独立随机样本。
% 
UDFS 策略以均值统计量 $\mu$ 为基准点，利用标准差 $\Sigma_{\mu}$ 与 $\Sigma_{\sigma}$ 对分布漂移量实施概率参数化估计，通过其可变的方向与幅度实现对随机域漂移的模拟。通过在重参数化方法生成新统计量的过程引入受控的随机性，进而在基准点统计量邻域内扩张特征流形。
% 
通过流形扩张提升模型对潜在未见退化模式的覆盖率，增强模型对未见退化模式的鲁棒性，实现对 OOD 退化模式的有效泛化。
% 
功能上，
UDFS 策略嵌入融合流形收缩的 Mamba 主干网络，部署在退化特征编码器与下游任务回归器之间，执行特征合成，其前向传播过程形式化如下
% 
\begin{equation}
    G(Z_{MRB}) = \sigma_{Z}^{\prime}\left(\frac{Z_{MRB} - \mu}{\sigma}\right) + \mu_{Z}^{\prime}
    \label{UDFS}
\end{equation}

UDFS 策略的集成只在训练阶段引入常数级计算开销，其时间复杂度增量在工程实现中几乎可忽略不计。为保证模型对真实源域观测数据的充分拟合，引入基于 Bernoulli 采样的动态激活机制以调节特征合成发生的频率。
具体地，
每次前向传播迭代均独立采样随机概率值 $p'$，网络仅在满足 $p' < p_0$ 的前提下触发 UDFS 策略实施特征生成，其中 $p_0$ 为预设的激活概率阈值。

\subsection{感知长尾分布的自适应重平衡微调策略}\label{susbec:LARR}

为应对多源域长尾退化分布带来的建模瓶颈，本节提出 LARR 策略。
其通过特征分布校正实现 LIBs 跨域退化轨迹的精准跟踪和准确 OOD SOH 估计。
% 
首先，引入特征分布校正，约束退化特征相似度与标签邻近度之间的映射一致性。
% 
其次，将调整分布校正过程策略融合到特征生成环节中构建 LARR 策略，分离长尾分布包含的特定域属性，提取退化表征的跨域不变性。
% 
最后，分析有 LARR 策略参与的网络训练演化动态，并据此环节退化轨迹跟踪问题中存在的头部偏置，进一步提升退化模式的泛化性能。

（1）基于特征核的特征统计校正

采用均匀分箱策略将有界的 SOH 标签空间离散化到 $B$ 个区间，其索引记为 $b=1,2,\dots,B$。LIBs 退化数据呈现长尾分布特性，不同分箱 $b$ 内的样本数量存在显著差异，这种差异通常是指数级的。
% 
与离散的分类任务不同，连续退化标签的物理属性要求特征空间中的表征相似度与标签距离成反比。循环老化数据的固有的长尾分布特性破坏这种距离度量准则，尾部分箱的样本稀疏性使得模型生成向头部主体分箱偏移的劣质表征。
% 
为抑制这种长尾效应，引入对称核函数传递相邻分箱的先验信息，从而校正每个分箱 $b$ 内的局部特征。
% 
校正后的特征与编码器及经过 UDFS 策略处理输出的原始观测特征相融合，经由非线性映射重构出分布均衡的退化表征。
各分箱内部特征统计量的计算的具体实现与第 \ref{subsec:UDFS} 小节所述的批次统计特征提取机制一致。
针对分箱 $b$ 内的局部特征 $Z_{MRB}$，其均值 $\mu_{b}$ 与标准差 $\sigma_{b}$ 定义为
% 
\begin{align}
    & \mu_{\textit{b}} = \frac{1}{HE}\sum_{H}\sum_{E}Z_{MRB} \\
    & \sigma_{\textit{b}} = \sqrt{\frac{1}{HE}\sum_{H}\sum_{E}(Z_{MRB} - \mu)^{2}}
\end{align}
% 
为校正特征分布，基于对应 SOH 标签的物理邻近度在各分箱间引入核平滑机制。不失一般性地，选取 Gaussian 核函数执行该平滑操作。由此导出的校正均值 $\tilde{\mu}_b$ 与校正标准差 $\tilde{\sigma}_b$ 定义为
% \begin{align}
%     \widetilde{\mu}_{\textit{b}} & = \sum_{1 \leq \textit{b}, \textit{b}^{\prime} \leq \mathcal{B}}Gau(y_{\textit{b}}, y_{\textit{b}^{\prime}})\mu_{\textit{b}^{\prime}} \notag \\
%     & = \sum_{1 \leq \textit{b}, \textit{b}^{\prime} \leq \mathcal{B}}exp(-\gamma\|y_{\textit{b}} - y_{\textit{b}^{\prime}}\|_{2})\mu_{\textit{b}^{\prime}},\label{mu_calibration} \\
%     \widetilde{\sigma}_{\textit{b}} & = \sum_{1 \leq \textit{b}, \textit{b}^{\prime} \leq \mathcal{B}}Gau(y_{\textit{b}}, y_{\textit{b}^{\prime}})\sigma_{\textit{b}^{\prime}} \notag \\
%     & = \sum_{1 \leq \textit{b}, \textit{b}^{\prime} \leq \mathcal{B}}exp(-\gamma\|y_{\textit{b}} - y_{\textit{b}^{\prime}}\|_{2})\sigma_{\textit{b}^{\prime}},\label{sigma_calibration}
% \end{align}
\begin{align}
    \widetilde{\mu}_{\textit{b}} = \sum_{1 \leq \textit{b}, \textit{b}^{\prime} \leq \mathcal{B}}exp(-\gamma\|y_{\textit{b}} - y_{\textit{b}^{\prime}}\|_{2})\mu_{\textit{b}^{\prime}},\label{mu_calibration} \\
    \widetilde{\sigma}_{\textit{b}} = \sum_{1 \leq \textit{b}, \textit{b}^{\prime} \leq \mathcal{B}}exp(-\gamma\|y_{\textit{b}} - y_{\textit{b}^{\prime}}\|_{2})\sigma_{\textit{b}^{\prime}},\label{sigma_calibration}
\end{align}
其中 $Gau(\cdot, \cdot)$ 表示系数为 $\gamma$ 的 Gaussian 核函数， $y_{\textit{b}}$ 和 $y_{\textit{b}^{\prime}}$ 分别是对应于第 $\textit{b}$ 和第 $\textit{b}^{\prime}$ 个分箱的 SOH 标签值。

（2）消除不平衡偏差的退化泛化

特征合成操作 $G(Z_{MRB})$ 前后均存在长尾分布，部署 LARR 旨在执行头部偏置特征校正。
LARR 策略利用特定于分箱的观测统计量 $\{\mu_b, \sigma_b\}$ 和校准统计量 $\{\tilde{\mu}_b, \tilde{\sigma}_b\}$ 两组参数执行特征分布校正，其张量变换定义为
% 
\begin{equation}
    \widetilde{G}(Z_{MRB})=\widetilde{\sigma}_{\textit{b}}\left(\frac{G(Z_{MRB}) - \mu_{\textit{b}}}{\sigma_{\textit{b}}}\right)+\widetilde{\mu}_{\textit{b}}
\end{equation}

在计算图结构上，LARR 策略与式 \eqref{UDFS} 中 UDFS 策略同构，亦仅在网络训练阶段激活。
为说明有 LARR 策略参与的模型训练演化动态，将特定迭代轮次 $i$ 下分箱 $b$ 的观测统计量与校准统计量分别记为 $\{\mu_b^{(i)}, \sigma_b^{(i)}\}$ 和 $\{\tilde{\mu}_b^{(i)}, \tilde{\sigma}_b^{(i)}\}$。
每次前向迭代周期内，首先对退化表征执行分布校正，然后在计算图结束时同步更新对应的统计参数。
为维持训练演化过程的稳定性，观测统计量通过基于动量的滑动平均机制执行更新。其融合历史迭代的隐状态和当前批次的静态观测值，其递推演化关系定义为
% 
\begin{align}
    & \mu_{\textit{b}}^{(i)} \leftarrow \alpha\mu_{\textit{b}}^{(i-1)}+(1-\alpha)\mu_{\textit{b}}, \\
    & {\sigma_{\textit{b}}}^{(i)} \leftarrow \alpha{\sigma_{\textit{b}}}^{(i-1)}+(1-\alpha)\sigma_{\textit{b}},
\end{align}
% 
其中 $\alpha$ 为动量衰减系数。相应的校准统计量依据前述式 \eqref{mu_calibration} 与 \eqref{sigma_calibration} 的核平滑机制进行同步更新。

原始的 LIBs 循环信号依次经过启发式信号重构策略、基于 Mamba 的退化模式编码器、UDFS 策略、LARR 策略和下游任务回归器
被映射到退化状态表征量。
基于 RODTNet 或 RODTNet-LT 模型的 OOD 鲁棒退化建模的完整过程如算法 \ref{alg:udfs_larr_training} 所示。

\begin{algorithm}[t]
    \caption{基于 RODTNet(-LT) 的 OOD 鲁棒退化建模方法}
    \label{alg:udfs_larr_training}
    \centering
    \begin{algorithmic}[1]
        \Require 源域观测数据集 $\mathcal{D}_{src}=\{(\widetilde{X}_{Recon}, Y)\}$，模型参数 $\bm{\theta}_{enc}$ 与 $\bm{\theta}_{reg}$，超参数 $N_{iter}$、$B$、$p_0$ 与 $\alpha$
        \Ensure 优化后的参数 $\bm{\theta}_{enc}^*$ 与 $\bm{\theta}_{reg}^*$，校准统计量 $\{\tilde{\mu}_{b}, \tilde{\sigma}_{b}\}_{b=1}^{B}$
        \Statex \textbf{特征提取和基于 UDFS 策略的流形扩张}
        \For{epoch $= 1$ \textbf{to} $N_{iter}$} \Comment{训练迭代轮次 $N_{iter}$}
            \ForAll{$(\widetilde{X}_{Recon}, Y) \sim \mathcal{D}_{src}$}
                \State $Z_{MRB} \leftarrow \textsc{MambaEncoder}(\widetilde{X}_{Recon}, \bm{\theta}_{enc})$
                \State $\mu, \sigma, \Sigma_{\mu}, \Sigma_{\sigma} \leftarrow \textsc{BatchStat}(Z_{MRB})$
                \State $p' \sim \mathcal{U}(0,1)$
                \If{$p' < p_0$}
                    \State $\epsilon_{\mu}, \epsilon_{\sigma} \sim \mathcal{N}(0,1)$
                    \State $\mu_{Z}^{\prime} \leftarrow \epsilon_{\mu}\Sigma_{\mu}+\mu$
                    \State $\sigma_{Z}^{\prime} \leftarrow \epsilon_{\sigma}\Sigma_{\sigma}+\sigma$
                    \State $G(Z_{MRB}) \leftarrow \textsc{Reparam}(\mu_{Z}^{\prime}, \sigma_{Z}^{\prime})$
                \Else
                    \State $G(Z_{MRB}) \leftarrow Z_{MRB}$
                \EndIf
        \Statex \textbf{基于 LARR 策略的长尾分布校准}
                \State $b \leftarrow \textsc{BinIndex}(Y, B)$
                \State $\mu_{b}^{(i)} \leftarrow \alpha\mu_{b}^{(i-1)}+(1-\alpha)\mu_{b}$
                \State $\sigma_{b}^{(i)} \leftarrow \alpha\sigma_{b}^{(i-1)}+(1-\alpha)\sigma_{b}$
                \State $\tilde{\mu}_{b}, \tilde{\sigma}_{b} \leftarrow \textsc{KernelSmooth}(\{\mu_{k}^{(i)}, \sigma_{k}^{(i)}\}_{k=1}^B, Y)$
                \State $\widetilde{G}(Z_{MRB}) \leftarrow \tilde{\sigma}_{b} \odot \frac{G(Z_{MRB}) - \mu_{b}^{(i)}}{\sigma_{b}^{(i)}} + \tilde{\mu}_{b}$
        \Statex \textbf{退化建模和参数优化}
                \State $\hat{Y} \leftarrow \textsc{Regressor}(\widetilde{G}(Z_{MRB}), \bm{\theta}_{reg})$
                \State $\mathcal{L} \leftarrow \textsc{Loss}(\hat{Y}, Y)$
                \State $\bm{\theta}_{enc}^*, \bm{\theta}_{reg}^* \leftarrow \textsc{Optimize}(\mathcal{L}, \bm{\theta}_{enc}, \bm{\theta}_{reg})$
            \EndFor
        \EndFor
        \State \Return $\bm{\theta}_{enc}^*$, $\bm{\theta}_{reg}^*$, $\{\tilde{\mu}_{b}, \tilde{\sigma}_{b}\}_{b=1}^{B}$
    \end{algorithmic}
\end{algorithm}

\section{实验评估}\label{sec:RODTNet_experiment}

本节基于来源于 4 个数据集的 11 项泛化任务，结合 14 种基线方法及其 20 种变体，对 RODTNet 与RODTNet-LT 的性能进行系统评估。
% 
首先，对比分析 RODTNet、RODTNet-LT 及各类基线方法在不同数据集和不同泛化任务中的表现。
% 
其次，通过消融实验验证 RODTNet 各关键组成部分和所提出的策略的有效性，并进一步评估 LARR 策略在RODTNet-LT 中的具体作用。
% 
最后，考察所提出模型对超参数变化的敏感性，以分析其性能稳定性与参数鲁棒性。

\subsection{实验设置}\label{subsec:RODTNet_experimental_setup}

（1）数据集和 OOD 泛化任务设定

所提出方法在 MATR数据集 \cite{severson2019data, attia2020closed}、Tongji数据集 \cite{zhu2022data}、HUST数据集 \cite{D2EE01676A} 和 XJTU数据集 \cite{wang2024physics} 共 4 个大规模 LIBs 循环老化数据集上，通过设计的 11 个 OOD 泛化任务开展全面评估。
表~\ref{tab:dataset} 总结各数据集在材料组成、电气规格和统计特征等方面的信息。
图~\ref{fig:degradation_trajectory} 展示不同数据集跨域 LIBs 的退化轨迹及其分布密度。
完整的域划分方案见表~\ref{tab:tasks}，域划分标准覆盖充放电协议、电极材料与放电深度等关键因素。
LIBs 的性能退化程度通过 RUL 和 SOH 量化，前者刻画 LIBs 当前可用寿命水平，后者表征其全生命周期退化演变过程。

具体地，任务 1 至 5 \cite{TAN2024103725,10522981} 用于检验模型在快速充电协议变化条件下的鲁棒性。
任务 6 沿用文献 \cite{TAN2024103725} 的设置，评估模型对罕见退化模式的适应能力，其域间共享循环比例仅为 0.9\%。
任务 7 在此基础上进一步扩展，用于评估模型在 OOD 条件下对不同电极材料的泛化能力。
任务 8 \cite{D2EE01676A} 用于检验模型对多阶段放电协议变化的鲁棒性。
参考 MATR 数据集上的设置，任务 9 至 11 进一步分析充电协议、放电协议与放电深度交互耦合所引发的域漂移现象。

具体地，对MATR数据集中的样本，分别使用电压范围在 [2.7, 3.3] V 内微分容量曲线的平方拟合模型的二次系数和偏置，最大值，最小值，方差，循环平均温度，内阻和充电时间得到重构的 8 维输入向量。
%

有 6 个泛化任务使用这样的重构方式。

对比实验覆盖 5 类基线方法。
% 
第一类为先进的 LIBs 退化建模方法，包括采用 3 种聚合策略的 DeNN \cite{10102497}、PINN \cite{wang2024physics} 与 MambaLithium \cite{shi2024mambalithiumselectivestatespace}。
% 
第二类为基于预训练微调范式的退化建模方法，包括 CRNN \cite{D2EE01676A} 与 PINN+ \cite{wang2024physics}。
% 
第三类为基于域自适应范式的退化建模方法，包括 ConvMMD \cite{9788003}、LSTM-DA \cite{HAN2022230823}、DTKL \cite{10029904}、DDAN \cite{9560040} 与 DR-Net \cite{WANG2023108897}。
% 
第四类为基于域泛化范式的退化建模方法，包括 MAGNet \cite{TAN2024103725}、iTransformer \cite{liu2024itransformer}、PatchTST \cite{nie2023a} 与 TSMixer \cite{chen2023tsmixer}。
% 
第五类为在目标域上直接训练的所提出的模型，记为 RODTNet+ 与 RODTNet-LT+。
此外，通用序列建模方法的微调变体 iTransformer+、PatchTST+ 与 TSMixer+ 也一并纳入比较。

所有实验均在配备 Intel Core i9-13900KF CPU 与 NVIDIA 4090 GPU 的服务器上完成，软件环境为基于 CUDA 11.8 的 PyTorch 2.3.1。全部循环数据均经过最小—最大归一化，并缩放至 [0, 1] 区间。SOH 标签遵循各数据集既有缩放协议，其中 MATR 数据集按照其定义缩放至 [0.8, 1] 区间，其余数据集采用最小—最大归一化缩放至 [0, 1] 区间。RUL 标签在训练阶段进行归一化处理，在评估阶段进行反归一化恢复。

基于域泛化的方法遵循严格的验证协议 \cite{9847099}，其验证集完全由源域电池构成。
对于 MATR 数据集的批次 1 至 3 以及 Tongji 和 XJTU 数据集，训练集与验证集比例设定为 3:1；对于 MATR 数据集的批次 9，该比例设定为 4:1；
对于 HUST 数据集，该比例设定为 6:1。
所有目标域电池均完整保留并用于测试。
为支持域自适应方法中的域对齐过程，目标域进一步划分为测试子集与训练子集，其中前者包含指定用于退化建模的电池，后者由其余电池构成。两个子集均保证目标域退化样本的标签在训练阶段完全不可见。对于遵循预训练与微调范式的方法，其预训练阶段采用与域泛化方法一致的协议，测试集与域自适应方法保持一致，其余目标域电池再按照 9:1 的比例划分为训练集与验证集，用于执行微调。样本构建方式、模型架构与超参数设置的具体细节见表~\ref{tab:hyperparameters_setting}。
% 
模型参数优化采用 MSE 损失函数与 Adam 优化器。
训练过程最多迭代 100 轮，并引入早停机制，基于验证集上的建模误差以抑制过拟合。

模型性能采用 MAE、均方误差（Mean Squared Error, MSE）、RMSE 与决定系数（Coefficient of Determination, $R^2$） 4 项指标进行评估，用以量化模型对 OOD 退化模式的泛化能力。MSE 定义为
% 
\begin{equation}
MSE = \frac{1}{n} \sum_{i=1}^{n} \left( y_i - \hat{y}_i \right)^2
\end{equation}
% 
$R^2$ 定义为
% 
\begin{equation}
R^2 = 1 - \frac{\sum_{i=1}^{n} \left( y_i - \hat{y}_i \right)^2}{\sum_{i=1}^{n} \left( y_i - \bar{y} \right)^2}
\end{equation}
其中，$n$ 表示评估样本总数，$y_i$ 与 $\hat{y}_i$ 分别表示第 $i$ 个样本的真实标签与预测值，$\bar{y}$ 表示全部真实标签的算术平均值。
% 
MAE、MSE 与 RMSE 的取值越低，说明模型在未知分布上的预测误差越小，泛化性能越强；$R^2$ 越接近 1，说明模型对真实退化趋势的拟合程度越高，整体泛化能力越强。

为进一步量化泛化性能增益，采用误差下降百分比进行统计分析。具体地，当 RODTNet 或 RODTNet-LT 在某一任务中取得最优结果时，计算其相对于次优基线模型的误差降幅；当其未取得最优结果时，则计算其相对于最优基线模型的性能差值，其中负值表示该模型劣于最优结果。
随后，对 11 项泛化任务上的相对差异取平均值，以量化整体性能增益。
基于该统计准则，在 OOD 的 RUL 预测任务中，RODTNet 的 MAE 与 MSE 分别下降 40.76\% 和 28.65\%；在 OOD 的 SOH 估计任务中，RODTNet-LT 的 MAE 与 MSE 分别下降 5.03\% 和 22.22\%。

上述 11 项泛化任务覆盖多种由异质退化因素引发的域偏移情形，实验结果表明，所提方法对复杂域偏移具有稳定的鲁棒性。相比之下，现有基线模型难以从多个源域中充分提取域不变退化模式，因此即使在允许访问部分目标域数据的条件下实施微调或域适应，其建模精度仍然受限。进一步观察可知，实施域泛化的 RODTNet 与 RODTNet-LT 均优于各自在目标域上直接训练的参考变体。该结果表明，多源域数据的联合利用有助于扩展样本覆盖范围，增强退化模式表征的完整性，并提高模型训练的充分性，从而提升 OOD 泛化能力。

\subsection{消融实验}\label{subsec:RODTNet_ablation_study}

为验证 RODTNet 与 RODTNet-LT 各组成模块的作用，开展消融实验，分别评估 RODTNet 内部关键组件及长尾感知自适应重平衡策略的有效性。
表~\ref{tab:rodtnet_ablation_1_part1} 和表~\ref{tab:rodtnet_ablation_1_part2} 汇报 RODTNet 及其不同变体在 4 个数据集上的 RUL 预测结果，评价指标为 MAE 和 MSE。
表~\ref{tab:rodtnet_ablation_2} 给出 RODTNet 与 RODTNet-LT 在相同数据集上的 SOH 估计结果，评价指标同样为 MAE 与 MSE。

（1）基线模型

以降采样后的原始循环信号作为输入，并采用全线性骨干网络构建基线模型。该模型作为后续各变体的统一比较基础。需要说明的，在 RUL 预测任务上的消融实验和 SOH 估计任务上的消融实验军涉及基线模型，这两个模型结构相同，但使用不同数据训练，任务结果上有所差异。

（2）DGBL

在基线模型基础上引入 UDFS 策略，得到 DGBL。实验结果表明，DGBL 的泛化性能优于基线模型，说明所提出的泛化策略能够有效拓展特征分布边界，并增强模型对分布外退化模式的拟合能力。

（3）LMCRDR

将基线模型的原始循环信号输入替换为基于启发式策略重构后的输入，得到 LMCRDR。该模型的泛化性能较基线模型显著提升，表明所提出的流形收缩型特征表征方式能够与骨干网络形成良好适配，并在退化模式泛化方面表现出明显优势。

（4）MCRDR

在 LMCRDR 的基础上，进一步以 Mamba 编码器替代线性骨干网络，得到 MCRDR。实验结果显示，该替换能够进一步提升模型的泛化性能，表明 Mamba 编码器在提取域不变退化特征方面具有更强能力。

（5）LRODTNet

将 LMCRDR 与 UDFS 策略结合，构建线性骨干版本的 RODTNet，记为 LRODTNet。结果表明，在低本征维度退化表征的基础上引入基于特征调弈的策略后，模型的泛化性能进一步提升，说明该策略能够有效扩展退化表征边界。

（6）RODTNet-LT

针对 SOH 估计任务中存在的长尾分布问题，在 RODTNet 的基础上引入 LARR 策略，得到 RODTNet-LT。实验结果表明，RODTNet-LT 相较于 RODTNet 具有更低的泛化误差，说明所提出的 LARR 策略能够有效缓解头部偏置，并提升模型对长尾退化样本的建模能力。

\subsection{超参数敏感性分析}\label{subsec:RODTNet_sensitivity_analysis}

\begin{figure}[!htbp]
    \centering
    \includegraphics[width=\textwidth]{Img/RODTNet_figures/trans/Fig7_sensitivity_analysis_combined.eps}
    \bicaption{}{}
    \label{fig:rodtnet_sensitivity}
\end{figure}

为分析 RODTNet 对关键超参数变化的敏感性，本节围绕批量大小、编码器隐状态维度、编码器层数与 UDFS 激活阈值开展敏感性分析。实验在 4 个数据集上进行，并以模型在目标域上的均方根误差（Root Mean Squared Error, RMSE）作为评价指标。

（1）批量大小 $n_{bs}$ 的影响

由于 RODTNet 的不确定性量化过程依赖批量统计特征，批量大小可能对模型的泛化性能产生影响。实验将批量大小 $n_{bs}$ 设置为 $\{32, 64, 128, 256, 512\}$ 以评估其作用。如图~xx 所示，当批量大小取 32 时，模型在超过半数任务上取得最优泛化性能。对于基于 MATR 数据集构建的泛化任务，模型误差随批量大小增加而减小，说明在退化样本较为充足的条件下，批量统计差异能够更有效地刻画域漂移。对于其余泛化任务，较小的批量大小更有利于潜在退化模式的识别与捕获。

（2）隐状态维度 $H$ 的影响

隐状态维度 $H$ 通过改变模型容量影响其泛化能力。实验将隐状态维度 $H$ 设置为 $\{16, 32, 64, 128, 256\}$ 以评估其影响。如图~xx 所示，在基于 Tongji 与 HUST 数据集构建的泛化任务中，模型性能随隐状态维度增加而提升，这在一定程度上反映出 RODTNet 具备良好的可扩展性。相对地，在基于 MATR 与 XJTU 数据集构建的泛化任务中，过大的隐状态维度反而导致性能下降，说明过高的模型容量可能引发过拟合。

（3）编码器层数 $n_{env}$ 的影响

编码器层数 $n_{env}$ 同样通过改变模型容量影响其泛化能力。实验将编码器层数 $n_{enc}$ 设置为 $\{1, 2, 3, 4, 5\}$ 以评估其作用。如图~xx 所示，编码器层数变化所呈现的总体规律与隐状态维度基本一致，但层数增加更易引入过拟合瓶颈，说明模型深度的扩展需要与任务复杂度及样本规模保持匹配。

（4）UDFS 激活阈值 $p_{0}$ 的影响

UDFS 激活阈值 $p_{0}$ 通过调节训练过程中真实样本与生成样本的相对比例影响模型的泛化性能。实验将该阈值 $p_{0}$ 设置为 $\{0.0008, 0.004, 0.02, 0.1, 0.5\}$ 以评估其影响。如图~xx 所示，随着激活阈值增大，RODTNet 在 4 个数据集上的泛化性能整体呈上升趋势，表明源域真实样本在 OOD 退化模式建模过程中仍然发挥关键作用。

综合来看，RODTNet 在大多数泛化任务上对超参数变化表现出较强稳健性。需要指出的是，基于 XJTU 数据集构建的泛化任务涉及多种退化影响因素的耦合，其性能对超参数设定更为敏感。为保证对比实验设置的一致性，所有泛化任务统一采用相同的超参数配置。
具体地，批量大小设置为 32，原因在于超过半数任务在该设定下取得最小泛化误差。
隐状态维度设置为 16，编码器层数设置为 1，尽管更大的模型规模在部分任务中有助于提高泛化能力，但综合考虑存储开销与计算成本后，上述配置更具实际可行性。
需要强调的是，即便在并非全局最优的参数组合下，RODTNet 在对比实验中仍然表现出显著优势。
UDFS 激活阈值设置为 0.5，该取值不仅对应较低的泛化误差，同时能够在候选范围内最大限度降低训练开销。

\section{本章小结}\label{sec:RODTNet_conclusion}

面向 VBMS 中 LIBs 退化建模的 OOD 泛化问题，构建鲁棒建模方法 RODTNet 及其长尾优化变体RODTNet-LT。
% 
首先，设计基于流形收缩的鲁棒退化表示提取策略。
该策略融合启发式信号重构信息与基于 Mamba 机制提取的潜在动态特征，在保持退化模式连续性的同时压缩特征流形，从而增强退化表示的紧致性与鲁棒性。
% 
其次，开发不确定性驱动的特征合成方法，以应对多样化车辆运行条件导致的域漂移。
该方法通过概率建模建立特征统计不确定性与域变化之间的关联关系，进而引导域不变特征学习，并生成具有多样性的 OOD 样本。
% 
最后，设计基于特征分布平滑的长尾感知自适应重平衡优化技术。
该技术利用特征与标签之间的相关性，缓解合成特征中的头部偏置，提升模型对不平衡退化轨迹的建模能力。
% 
基于 4 个 LIBs 循环退化数据集构建的 11 项 OOD 泛化任务的实验结果表明，RODTNet 和 RODTNet-LT 相较于 14 种基线方法及其 20 种变体均表现出性能优势，从而为可靠 VBMS 的实现提供方法支持。

Exploring temporal domain adaptation thus presents a promising future direction, aiming to equip models with dynamic online update capabilities for sustained prognostic performance over time.
