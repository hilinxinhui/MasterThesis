# 修改说明

## 目录一级标题格式不一致

问题：目录中“独创性声明”“摘要”“Abstract”属于一级标题，但加粗效果和上下间距与“第一章 绪论”等正文一级标题不一致。

原因：目录样式 `\titlecontents{chapter}` 原本在声明和摘要之后才定义，导致这些前置标题写入 `.toc` 时没有使用统一的一级目录样式；同时原样式中的 `\vspace{-1.2ex}` 会压缩一级目录项的上下间距。

修改：在 `Thesis.tex` 中，将 `\titlecontents{chapter}` 提前到 `\frontmatter` 之前，并将一级目录样式调整为：

```tex
\titlecontents{chapter}
    [1.55cm]
    {\bfseries\large}
    {\contentslabel{3em}}
    {\hspace*{-3em}}
    {\mytitlerule\chapterPageNumber\contentspage}
```

效果：“独创性声明”“摘要”“Abstract”和正文各章使用同一套一级目录格式，保持加粗和上下间距一致。

## 删除插图索引和表格索引

问题：提交版本不需要插图索引和表格索引。

修改：在 `Thesis.tex` 中删除以下内容：

```tex
\intotoc{\listfigurename}
\listoffigures
\intotoc{\listtablename}
\listoftables
```

效果：目录后不再生成“插图索引”和“表格索引”页面，目录中也不再出现对应条目。

## 声明页后空白页计入页码

问题：“独创性声明”为 I，其后一页为空白页；该空白页不应显示页码，也不应计入罗马页码。摘要应从 II 开始，目录中的摘要页码也应为 II。

原因：模板在声明页和摘要页之间通过 `\cleardoublepage` 插入空白页，该空白页被计入页码，导致摘要页码显示为 III。

修改：在 `Thesis.tex` 中新增 `\inputabstractwithoutblankpagecount`，仅在输入摘要时局部重定义 `\cleardoublepage`：

```tex
\inputabstractwithoutblankpagecount
```

效果：声明页后的空白页保留为空白页，但使用 `empty` 样式且不占页码；摘要从 II 开始，目录页码同步更新。

## 英文摘要标题字体不正确

问题：英文摘要页标题 `Abstract` 字体不符合要求，应使用新罗马字体。

原因：模板的章节标题格式包含 `\sffamily`，会使英文摘要标题使用非罗马字体。

修改：在 `Tex/Abstract.tex` 中将英文摘要标题改为：

```tex
\chapter[Abstract]{{\rmfamily Abstract}}\chaptermark{Abstract}
```

效果：正文中的 `Abstract` 标题使用罗马字体族，目录项仍保持普通文本 `Abstract`。
